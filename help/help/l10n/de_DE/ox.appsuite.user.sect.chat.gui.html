Chat

10.1. Das Chat-Fenster

Dieses Fenster wird für alle Chat-Funktionen verwendet:

  • Suchen und Anzeigen von Kontakten, Gruppenkonversationen und öffentlichen Kanälen

  • Anlegen und Bearbeiten von Gruppenkonversationen und von öffentlichen Kanälen

  • Direktnachrichten austauschen mit einem einzelnen Kontakt, Nachrichten austauschen in einer Gruppe oder in einem öffentlichen Kanal

Um dieses Fenster zu öffnen, klicken Sie in der Menüleiste auf das Symbol Chat .

Inhalt

  • In der Titelleiste: Symbol Stick to the right side . Verankert das Chat-Fenster am rechten Rand.

  • Ihr Profilfoto, Ihr Anzeigename

  • Symbol Start new conversation . Öffnet ein Menü mit diesen Funktionen:

    • Eine private Konversation mit einem einzelnen Kontakt anlegen

    • Eine Gruppenkonversation mit mehreren Teilnehmern anlegen

    • Einen öffentlichen Kanal anlegen, dem andere Kontakte beitreten können

  • Eingabefeld Search or start a new chat. Eintippen eines Suchbegriffs zeigt in der Liste Kontakte, Konversationen und öffentliche Kanäle, die zum Suchbegriff passen.

  • Die Liste der Konversationen mit Datum oder Uhrzeit und dem Anfang der letzten Nachricht

    • Ein Symbol zeigt den Lesestatus der Nachricht:

      • Ein grauer Haken: Die Nachricht ist auf dem Server angekommen.

      • Zwei graue Haken: Die Nachricht wurde vom Server an alle Empfänger übertragen.

      • Zwei blaue Haken: Die Nachricht wurde von allen Empfängern gelesen.

    • Recent conversations. Zeigt im Konversationsfeld die verborgenen Konversationen.

    • All channels. Zeigt im Konversationsfeld alle öffentlichen Kanäle. Durch Anklicken von Join treten Sie einem Kanal bei.

    • All files. Zeigt im Konversationsfeld eine Liste der Dateien, die in den Konversationen verwendet werden.

  • Der Titel der Konversation, die in der Liste gewählt ist.

    • Name des Kontakts oder Name der Gruppe, Mitglieder der Gruppe

    • Symbol More actions . Öffnet ein Menü mit diesen Funktionen:

      • Hide chat. Verbirgt die aktuelle Konversation und die zugehörigen Nachrichten. Um die Konversation wieder anzuzeigen, klicken Sie in der Liste auf Recent conversations.

      • Bei einer Gruppenkonversation oder einem öffentlichen Kanal:

        Edit chat öffnet ein Fenster zum Bearbeiten der Konversation.

        Durch Anklicken von Leave chat verlassen Sie die Konversation.

    • Das aktuelle Datum

    • Bei einer Gruppenkonversation: Die Namen der Mitglieder

  • Das Konversationsfeld

    • Enthält die gesendeten und empfangenen Nachrichten der Konversation, die in der Liste gewählt ist.

    • Zeigt Systemmeldungen an, wenn Gruppenkonversationen oder öffentliche Kanäle angelegt oder bearbeitet werden.

  • Das Nachrichtenfeld zum Eingeben einer Nachricht, die Sie in dieser Konversation senden wollen

  • Das Symbol Anhang zum Hinzufügen von Dateien zu dieser Konversation

Verwandte Themen:

  • Konversationen anlegen

    Konversationen bearbeiten

    Einer Konversation beitreten

    Eine Konversation verbergen

    Eine Konversation verlassen

    Nachrichten senden oder lesen
