12.2.1. Visualización de listas de contactos

Para iniciar una conversación puede usar las funciones de la lista de contactos seleccionada. La lista de contactos se mantiene abierta hasta que la vuelva a cerrar. Nota: La lista de contactos y el área de notificación se solapan entre sí.

Cómo abrir la lista de contactos:

  • Pulse el icono Lista de contactos de la parte derecha de la barra de menús.

    Para cerrar la lista de contactos, pulse de nuevo el icono de la barra de menús.
