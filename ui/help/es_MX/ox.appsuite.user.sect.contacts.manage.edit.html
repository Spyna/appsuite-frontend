7.6.3. Edición de contactos

Los datos de contacto se pueden editar posteriormente. La ventana de edición muestra los datos usados más frecuentemente. Se pueden visualizar otros datos.

Cómo editar un contacto:

Requisito: Debe tener los permisos adecuados para editar objetos en la carpeta que aloja al contacto.

  1. Seleccione un contacto en la barra lateral.

  2. Presione en Editar en el área de visualización. Los datos del contacto se mostrarán.

  3. Edite los datos.

  4. Presione en Guardar.
