8.4.3. Aggiungere partecipanti a un'attività

Come aggiungere partecipanti nella pagina Crea attività:

  • Digitare gli indirizzi di posta elettronica dei partecipanti, il nome di un gruppo o il nome di una lista di distribuzione nel campo di inserimento sotto Partecipanti. Suggerimenti:

    • Mentre si digita un indirizzo di posta, sono visualizzati diversi suggerimenti corrispondenti. Per accettare un suggerimento, utilizzare uno dei seguenti metodi:

      • Utilizzare la barra di scorrimento per scorrere l'elenco. Fare clic su un suggerimento.

      • Utilizzare le frecce direzionali per selezionare un suggerimento. Premere Invio.

    • Per selezionare contatti da un elenco, aprire la finestra Seleziona contatti. Per fare ciò, fare clic sull'icona Rubrica icon a destra del campo di inserimento.

    Per rimuovere un partecipante, fare clic sull'icona accanto al nome.
