7.6.4. Creating a distribution list from the list of participants

You can create a distribution list from an appointment's list of participants.

You can create a distribution list from an appointment's list of participants.

  1. Select an appointment from the list or in a calendar view.

  2. Select Save as distribution list from the toolbar.

  3. Complete the details for creating the distribution list.
