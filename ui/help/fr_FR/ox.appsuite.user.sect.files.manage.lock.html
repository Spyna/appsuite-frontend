9.5.10. Verrouiller et déverrouiller des fichiers

Quand vous modifiez un fichier, vous pouvez le verrouiller. Le verrou sert à :

  • Informer les autres utilisateurs que le fichier est en train d'être modifié et qu'il pourrait ne plus être à jour.

  • Empêcher ces utilisateurs de modifier le fichier, dans le cas où il se trouve dans un dossier que vous partagez en ayant accordé les droits de modification.

Vous pouvez verrouiller et déverrouiller un fichier isolé ou plusieurs fichiers à la fois.

Pour verrouiller un fichier :

  • Cliquez sur un fichier dans la zone d'affichage. Dans la fenêtre qui s'ouvre, cliquez sur Plus. Choisissez Verrouiller dans le menu.

Pour déverrouiller un fichier :

  • Cliquez sur un fichier verrouillé dans la zone d'affichage. Dans la fenêtre qui s'ouvre, cliquez sur Plus. Choisissez Déverrouiller dans le menu.
