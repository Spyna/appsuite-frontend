Permissions for already existing folders

You have the following permissions for already existing folders:

  • You are the owner of your personal folders.

  • Each user is a guest of the folder Public Folders.

  • Each user is a guest of the Global Address Book folder.

  • Each user is a guest of the InfoStore and UserStore folder.

  • Each user is a guest of the Public InfoStore folder.
