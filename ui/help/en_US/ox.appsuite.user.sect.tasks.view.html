11.2. Displaying Tasks

How to display a task:

  1. Open a tasks folder in the folder tree.

  2. To quickly find a particular task, use one of the following methods:

    • To find tasks with a particular subject, use the search function.

    • To sort the tasks list, click the Sort icon below the list. Select a sort criteria from the menu.

    • To only display due tasks in the list, click the Sort icon below the list. Disable Show done tasks in the menu.

  3. Click on a task in the sidebar. The task's data are displayed in the display area.

  4. To display another task, use one of the following methods:

    • Click on another task in the sidebar.

    • Use the cursor keys to browse the list.
