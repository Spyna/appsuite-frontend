6.8.1. Persönliches Adressbuch hinzufügen

Sie können unterhalb von Meine Adressbücher weitere persönliche Adressbücher anlegen.

So legen Sie ein neues persönliche Adressbuch an:

  1. Klicken Sie innerhalb der Ordneransicht auf Neues Adressbuch hinzufügen. Ein Fenster öffnet sich.

  2. Geben Sie einen Namen ein. Wenn das neue Adressbuch ein öffentliches Adressbuch sein soll, aktivieren Sie Als öffentlichen Ordner hinzufügen. Klicken Sie auf Hinzufügen.

Verwandte Themen:

  • Externe Adressbücher abonnieren

    Kontakte verschieben oder kopieren
