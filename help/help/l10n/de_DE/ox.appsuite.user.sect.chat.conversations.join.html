10.2.3. Einer Konversation beitreten

Eine Gruppenkonversation ist nur für die Mitglieder der Gruppe sichtbar. Sie können einer Gruppenkonversation nicht von sich aus beitreten. Im Gegensatz dazu sind öffentliche Kanäle für alle Benutzer sichtbar. Sie können einem öffentlichen Kanal einfach beitreten.

So treten Sie einer Gruppenkonversation bei:

  • Bitten Sie ein Mitglied der Gruppe, Sie zu der Gruppenkonversation hinzuzufügen.

So treten Sie einem öffentlichen Kanal bei:

  1. Klicken Sie in der Liste unten auf All channels. Die öffentlichen Kanäle werden im Konversationsfeld angezeigt.

  2. Klicken Sie neben einem Kanal auf Join.
