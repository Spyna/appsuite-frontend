6.5.12. Imprimer des courriels

Pour imprimer des messages, vous pouvez :

  • imprimer le contenu d'un courriel

Pour imprimer le contenu d'un courriel :

  1. Sélectionnez un message dans la barre latérale. Cliquez sur Plus dans la zone d'affichage.

  2. Si le besoin s'en fait sentir, modifiez les réglages d'impression.

  3. Si le besoin s'en fait sentir, modifiez les réglages d'impression. Cliquez sur Imprimer en utilisant la boîte de dialogue système… en bas à gauche.

Conseil : vous pouvez également imprimer plusieurs courriels à la fois.
