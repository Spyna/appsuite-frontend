Kontakte zu einem Termin einladen

Sie können aus einem Adressbuch einen Kontakt, mehrere Kontakte oder die Kontakte einer Verteilerliste zu einem Termin einladen.

So laden Sie Kontakte zu einem Termin ein:

  1. Wählen Sie in der Seitenleiste einen Kontakt oder eine Verteilerliste.

  2. Klicken Sie im Anzeigebereich auf Zu Termin einladen.

  3. Vervollständigen Sie die Angaben zum Anlegen eines Termins.
