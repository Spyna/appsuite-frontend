9.5.7. Datei zum Portal hinzufügen

Sie können eine Datei als Widget zum Portal hinzufügen.

So fügen Sie eine Datei zum Portal hinzu:

  • Wählen Sie im Anzeigebereich eine Datei. Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen . Klicken Sie im Menü auf Zum Portal hinzufügen.
