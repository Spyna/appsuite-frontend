12.2. オブジェクトの印刷

オブジェクトの印刷時には、用紙サイズや余白、インストールされているプリンタの選択など、さまざまなプリンタの設定を利用できます。印刷 ウィンドウでは、画面の左側にプリンタの設定が、また、画面の右側に印刷のプレビューがそれぞれ表示されます。

プリンタの設定を使用する

前提条件: 印刷 ウィンドウが表示されている必要があります。

  1. インストールされているプリンタを選択するには、対象 の 変更 ボタンをクリックします。

  2. 印刷するページを指定するには、ページ で設定を変更します。

  3. 部数 で、印刷部数を指定します。レイアウト で、印刷の向きを指定します。カラー で、色付きのドキュメントをカラーで印刷するのかグレー スケールで印刷するのかを設定します。

  4. 余白 で、余白のサイズを指定します。余白の値をカスタマイズするには、次の手順に従います。

    1. 余白 の カスタム 設定を選択します。

    2. 印刷プレビューで、任意の位置に余白をドラッグします。値をクリックして、任意の値を入力することもできます。

  5. オプション で、ヘッダーとフッターを印刷するかどうかを指定します。
