5.2.2. Portal-Kacheln hinzufügen

Sie können die vorhandenen Kacheln ergänzen, indem Sie neue hinzufügen.

So fügen Sie neue Portal-Kacheln hinzu:

  1. Klicken Sie in der App Portal rechts oben auf Diese Seite personalisieren.

  2. Klicken Sie unterhalb von Portal-Einstellungen auf Widget hinzufügen. Wählen Sie einen Eintrag.

  3. Manche Kacheln benötigen weitere Angaben. In diesem Fall geben Sie die erforderlichen Werte ein. Klicken Sie auf Speichern.

Tipp: In der App Dateien können Sie eine Datei als Portal-Kachel hinzufügen.

Tipp: In der App Mail können Sie eine E-Mail als Portal-Kachel hinzufügen.
