9.6. Uzyskiwanie dostępu do plików przy użyciu WebDAV

Za pomocą protokołu WebDAV można uzyskiwać dostęp do plików aplikacji Pliki tak samo jak w przypadku plików zapisanych lokalnie na dysku. W porównaniu z dostępem uzyskiwanym z poziomu interfejsu użytkownika oprogramowania do pracy grupowej protokół WebDAV ma swoje zalety i wady:

  • Zaletą jest szybki i bezpośredni dostęp za pomocą przeglądarki plików, np. Eksploratora Windows. Konieczne jest zalogowanie się do interfejsu użytkownika oprogramowania do pracy grupowej.

  • Wadą jest to, że jest widoczna tylko bieżąca wersja dokumentu. Nie są pokazywane ani historia wersji, ani dodatkowe informacje, jak komentarze.

Ostrzeżenie: usunięcie pliku podczas korzystania z protokołu WebDAV poskutkuje utratą wszystkich wersji pliku — nie tylko bieżącej. W celu usunięcia określonej wersji należy użyć metod opisanych tutaj: : Jak otworzyć lub zapisać określoną wersję:.

Przed próbą uzyskania dostępu do plików aplikacji Pliki za pomocą przeglądarki plików, np. Eksploratora Windows, należy skonfigurować dostęp przy użyciu WebDAV. Procedura wygląda różnie w zależności od oprogramowania systemu:

  • Konfiguracja WebDAV w systemie Windows XP

  • Konfiguracja WebDAV w systemie Windows Vista

  • Konfiguracja WebDAV w systemie Windows 7
