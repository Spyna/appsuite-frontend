6.5.1. Neuen Kontakt anlegen

Um einen neuen Kontakt anzulegen, müssen Sie im Fenster Kontakt hinzufügen zumindest einen Namen eingeben. Alle weiteren Angaben sind optional.

So legen Sie einen neuen Kontakt an:

  1. Öffnen Sie im Ordnerbaum ein Adressbuch.

    Hinweis: Öffnen Sie ein Adressbuch, in dem Sie die Berechtigung zum Anlegen von Objekten haben.

  2. Klicken Sie in der Werkzeugleiste auf Neu. Klicken Sie auf Kontakt hinzufügen.

  3. Geben Sie die Daten ein.

    Eine Beschreibung der Eingabefelder finden Sie in Ansicht zum Erstellen oder Bearbeiten.

  4. Wenn Sie dem Kontakt Anlagen hinzufügen wollen, klicken Sie unterhalb von Anlagen auf Datei hochladen. Wählen Sie eine oder mehrere Dateien.

    Um eine Anlage zu entfernen, klicken Sie auf das Symbol Löschen .

    Tipp: Sie können eine Anlage auch hinzufügen, indem Sie ein Dokument aus einem Dateibrowser oder vom Desktop in das Kontakt-Fenster ziehen und ablegen.

  5. Klicken Sie auf Speichern.

Ähnliche Aktionen

  • Verteilerliste anlegen, Kontakt aus vCard hinzufügen, Kontakte aus Datei importieren, Kontakte aus sozialem Netzwerk importieren
