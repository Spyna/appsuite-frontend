Organiser les données

11.5. Abonnements

Les abonnements vous permettent d'utiliser les données de vos comptes externes dans le collecticiel. Les options suivantes existent :

  • Importer des contacts en provenance de vos comptes externes en vous abonnant à ces données.

  • Gardez une vue d'ensemble des données auxquelles vous êtes abonné en les gérant sur une page d'aperçu commune.
