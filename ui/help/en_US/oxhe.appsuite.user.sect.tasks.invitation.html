Answering Tasks Invitations

If a user adds you to a task as participant, you are informed about this task in the Info area. You can accept, temporarily accept or refuse your participation in the task.

How to answer a task invitation:

  1. Click the New objects icon in the menu bar. The Info area is displayed.

  2. Under Invitations select an invitation status for the new task.

  3. Click on Change status.
