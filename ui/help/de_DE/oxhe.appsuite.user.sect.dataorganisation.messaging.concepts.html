Wozu dient Social Messaging?

Social Messaging bietet eine einfache Möglichkeit, Nachrichten aus verschiedenen Quellen zusammenzuführen. Sie können

  • Nachrichtenquellen abonnieren,

  • Nachrichten lesen,

  • Nachrichten senden.
