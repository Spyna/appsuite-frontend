11.5.5. 仕事への出欠の状態を変更

仕事への出欠の状態は後で変更することもできます。

仕事への出欠の状態を変更する

  1. サイドバーで、仕事を選択します。

  2. 表示エリアで、さらに表示... をクリックします。メニューで、出欠の状態を変更 をクリックします。

  3. ウィンドウが表示されます。出欠の状態を選択します。必要に応じて、メッセージを入力します。

  4. 状態を変更 をクリックします。
