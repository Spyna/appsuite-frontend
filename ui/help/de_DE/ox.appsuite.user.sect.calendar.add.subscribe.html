8.3.4. Termine aus anderen Kalendern abonnieren

Sie können Termine aus anderen Kalendern, zum Beispiel aus Ihrem Google Kalender, in einen Kalenderordner importieren, indem Sie den anderen Kalender abonnieren. Informationen finden Sie in Daten abonnieren.
