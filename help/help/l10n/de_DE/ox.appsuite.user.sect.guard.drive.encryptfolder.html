13.4.3. Ordnerinhalte automatisch verschlüsseln

Wenn Sie für einen Ordner die automatische Verschlüsselung aktivieren, werden alle Dateien automatisch verschlüsselt, die Sie in diesem Ordner neu anlegen oder die Sie in diesen Ordner verschieben oder kopieren. Hinweise:

  • Wenn Sie die automatische Verschlüsselung aktivieren, werden bereits vorhandene Dateien nicht verschlüsselt.

  • Wenn Sie die automatische Verschlüsselung wieder deaktivieren, bleiben vorhandene verschlüsselte Dateien weiterhin verschlüsselt.

So verschlüsseln Sie Ordnerinhalte automatisch:

  1. In der App Drive wählen Sie in der Ordneransicht einen Ordner.

  2. Verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie auf Encrypt.

    • Wählen Sie im Kontextmenü Encrypt.

Ergebnis: Alle Dateien, die Sie in diesem Ordner neu anlegen oder die Sie in diesen Ordner verschieben oder kopieren, werden automatisch verschlüsselt.
