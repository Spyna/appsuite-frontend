12.2. Imprimer des objets

Différents réglages permettent de régler l'impression d'objets. On peut par exemple régler la taille du papier, les marges ou les imprimantes installées. La fenêtre Imprimer affiche les réglages d'impression sur la gauche ainsi qu'un aperçu avant impression sur la droite.

Pour utiliser les réglages d'impression :

Il faut auparavant que le fenêtre Imprimer soit ouverte.

  1. Pour choisir une imprimante installée, cliquez sur le bouton Modifier dans Cible.

  2. Pour indiquer les pages à imprimer, modifiez les réglages dans Pages.

  3. Dans Copies, définissez le nombre d'exemplaires. Dans Disposition, définissez l'orientation de la page. Dans En couleur, vous pouvez indiquer si les documents en couleur doivent être imprimés en couleur ou en nuances de gris.

  4. Dans Marges, définissez l'épaisseur des marges. Pour adapter les valeurs, procéder ainsi :

    1. Cochez Personnalisé dans Marges.

    2. Dans l'aperçu d'impression, faites glisser les marges à l'emplacement désiré. Vous pouvez également cliquer sur une valeur pour saisir la valeur désirée.

  5. Dans Options, décidez si les en-têtes et pieds de page doivent être imprimés.
