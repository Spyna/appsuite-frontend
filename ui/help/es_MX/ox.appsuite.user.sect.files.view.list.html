9.2.1. Visualización de ficheros en la vista Lista

Cómo ver un archivo en la vista Lista:

  1. Presione el icono Vista de la barra de herramientas. Presione en Lista.

  2. Abra una carpeta del árbol de carpetas.

  3. Para encontrar un determinado archivo rápidamente, use la función de búsqueda.

  4. Presione en un archivo de la barra lateral. Los datos del archivo se muestran en el área de visualización. Se mostrará una vista previa en algunos tipos de ficheros.

  5. Para mostrar otro archivo, use uno de los métodos siguientes:

    • Presione en otro archivo de la barra lateral.

    • Use las teclas de movimiento del cursor para navegar por la lista.
