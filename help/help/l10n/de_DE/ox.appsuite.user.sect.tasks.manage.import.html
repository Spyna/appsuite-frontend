8.7.6. Aufgaben importieren

Sie können Aufgaben aus einer Datei importieren, die im iCal-Format vorliegt.

So importieren Sie Aufgaben aus einer iCal-Datei:

  1. Wählen Sie in der Ordneransicht den Aufgabenordner, in den Sie die Aufgaben importieren wollen.

  2. Klicken Sie neben dem Kalender auf das Symbol Aktionen . Klicken Sie auf Importieren.

  3. Im Fenster Aus Datei importieren klicken Sie auf Datei hochladen. Wählen Sie eine Datei im iCal-Format.

    Um auch Aufgaben zu importieren, die die gleiche Identifikationsnummer haben wie bereits vorhandene Aufgaben, aktivieren Sie Vorhandene Ereignisse ignorieren.

  4. Klicken Sie auf Importieren.

Ergebnis: Die Aufgaben werden dem Ordner hinzugefügt.
