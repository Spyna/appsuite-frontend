9.2.1. Displaying files in the list view

How to view a file in the list view:

  1. Click the View icon in the tool bar. Click on List.

  2. Open a folder in the folder tree.

  3. In order to quickly find a certain file, use the search function.

  4. Click on a file in the side bar. The file data is shown in the display area. For some file types a preview is displayed.

  5. To display another file, use one of the following methods:

    • Click on another file in the side bar.

    • Use the cursor keys to browse the list.
