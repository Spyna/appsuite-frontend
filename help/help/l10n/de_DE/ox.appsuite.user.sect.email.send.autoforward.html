5.4.10. E-Mails automatisch weiterleiten lassen

Sie können eingehende E-Mails automatisch an eine andere Adresse weiterleiten lassen.

So lassen Sie E-Mails automatisch weiterleiten:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Öffnen Sie in der Seitenleiste den Eintrag E-Mail. Klicken Sie auf Automatische Weiterleitung.

  3. Klicken Sie auf Aktivieren.

    Geben Sie die E-Mail-Adresse ein, an die automatisch weitergeleitet werden soll.

    Wenn Sie eine Kopie dieser E-Mail behalten möchten, aktivieren Sie Kopie der Nachricht erhalten.

  4. Die automatische Weiterleitung wird als E-Mail-Filter eingetragen. Wenn nach der automatischen Weiterleitung weitere Filterregeln angewendet werden sollen, aktivieren Sie Gegen nachfolgende Regeln prüfen. Sie können die Abwesenheitsnotiz auch bei den Einstellungen zu E-Mail-Filtern bearbeiten.
