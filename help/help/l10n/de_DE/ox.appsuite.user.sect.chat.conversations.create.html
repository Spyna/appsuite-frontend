10.2.1. Chats anlegen

Sie haben folgende Möglichkeiten:

  • Einen privaten Chat mit einem einzelnen Kontakt anlegen

  • Einen Gruppenchat mit bestimmten Teilnehmern anlegen

  • Einen öffentlichen Kanal anlegen, dem andere Kontakte beitreten können

So legen Sie einen Chat an:

  1. Klicken Sie neben Ihrem Anzeigenamen auf das Symbol Neu .

    Wenn Sie das Chat-Fenster am rechten Rand verankert haben, klicken Sie oben auf Chats. Klicken Sie auf Neuer Chat.

    Ein Menü öffnet sich.

  2. Verwenden Sie eine der folgenden Methoden:

    • Um einen Chat mit einem einzelnen Kontakt anzulegen, wählen Sie Privater Chat. Das Fenster Neue Konversation starten öffnet sich.

      Wählen Sie einen Kontakt.

      Klicken Sie auf Konversation starten.

    • Um einen Chat mit mehreren Teilnehmern anzulegen, wählen Sie Gruppenchat. Das Fenster Gruppenchat erstellen öffnet sich.

      Bestimmen Sie den Gruppennamen. Optional bestimmen Sie ein Bild, indem Sie das vorhandene Symbol anklicken.

      Fügen Sie Kontakte hinzu.

      Klicken Sie auf Chat erstellen.

    • Um einen öffentlichen Kanal anzulegen, dem andere Kontakte beitreten können, wählen Sie Channel. Das Fenster Neuen Kanal erstellen öffnet sich.

      Bestimmen Sie den Kanalnamen. Optional bestimmen Sie ein Bild, indem Sie das vorhandene Symbol anklicken.

      Klicken Sie auf Create Channel.

Der neue Chat wird in der Liste angezeigt.

Tipp: Sie können einen neuen Chat auch anlegen, indem Sie unmittelbar nach dem Öffnen des Chat-Fensters die Symbole im Chat-Feld verwenden. Sobald Sie in der Liste einen Chat gewählt haben, werden diese Symbole nicht mehr angezeigt.
