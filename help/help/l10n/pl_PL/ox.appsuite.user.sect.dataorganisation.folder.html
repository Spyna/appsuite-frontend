10.1. Foldery

Foldery ułatwiają:

  • sprawowanie kontroli nad obiektami

  • udostępnianie informacji innym użytkownikom i partnerom zewnętrznym

  • wyszukiwanie określonych informacji i szybkie ponowne ich znajdowanie

Foldery są różnie nazywane w różnych aplikacjach:

  • W aplikacji Książka adresowa folder jest nazywany książką adresową.

  • W aplikacji Kalendarz folder jest nazywany kalendarzem.
