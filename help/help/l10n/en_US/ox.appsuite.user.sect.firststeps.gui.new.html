2.4.3. The button for creating new objects

Note: Depending on the app, the button label varies. The following is a schematic representation.

Creates app specific objects. The More actions icon opens a menu for creating objects for other apps. You can then e.g. enter a new appointment in the Email app or enter a new contact without leaving the Email app.

The More actions icon will not be available if the folder view is closed. To show the folder view, click the Open folder view icon on the bottom left side.
