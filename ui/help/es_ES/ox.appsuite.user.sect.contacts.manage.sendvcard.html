Envío de contactos como vCard

Puede enviar uno o varios contactos de una vez como vCard adjunto a un correo electrónico.

Cómo enviar un contacto como adjunto vCard:

  1. Seleccione un contacto en la barra lateral.

  2. Pulse en Enviar como vCard en el área de visualización.

  3. Rellene los datos para enviar un nuevo correo electrónico.
