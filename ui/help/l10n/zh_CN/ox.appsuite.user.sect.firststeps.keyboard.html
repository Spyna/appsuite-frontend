3.2. 键盘输入

作为滑鼠的替代您可以使用以下按键：

  • 使用按键组合 [Ctrl]+[F6]，在菜单条、文件夹树、边栏、显示区域之间来回切换。

  • 要输入字段里移动鼠标，使用以下按键：方向键、[Home]、[End]。

  • 要浏览文件夹树或边栏，请使用以下按键：[Page up]、[Page down]·。

  • 要打开或关闭文件夹树里的一个文件夹，请使用 [Space bar] 或 [Enter]。

  • 要选择文件夹树或边栏里的元素，请使用方向键。

  • 为了选择一个功能、一个输入字段、或一个选择框，请使用以下按键：[Tab]、[Shift]+[Tab]。选择的元素将会高亮。

    您可以使用 [Shift]+[Tab] 反向选择元素。

  • 要执行选择功能，按下 [Enter]。

  • 要启用或停用一个选择框，请按下 [Space bar]。

  • 要切换选择框，请使用方向键。

  • 要关闭一个弹出窗口或取消一个对话窗口，请按 [Esc]。

如何使用键盘以发送一个新电子邮件

  1. 如果电子邮件应用程序并没有执行，请进行以下操作：

    • 按下 [Ctrl]+[F6] 以高亮菜单栏。

    • 在菜单里按下 [Tab] 以高亮电子邮件。

    • 按下 [Enter]。电子邮件应用程序将会执行。

  2. 重复按下 [Tab] 直到新建图标高亮。按下 [Enter]。创建新邮件页面打开。会启用To:输入字段。

  3. 输入收件人的电子邮件电子邮件地址。按下 [Enter]。

  4. 按下 [Tab]。会启用主题输入字段。输入主题。

  5. 按下 [Tab]。会启用电子邮件文本输入字段。输入电子邮件文本。

  6. 为了发送电子邮件，按下 [Tab]。会高亮发送按钮。按下 [Enter]。将会发送电子邮件。

  7. 要添加附件，重复按下 [Tab] 直到附件功能高亮。按下 [Enter]。会显示添加文件的控制元素。

    按下 [Enter]。会打开选择文件的对话字段。视系统而定，使用 [Tab] 键、[Enter] 键、方向键一个选择一个文件后关闭系统对话窗口。

    要添加附件，重复按下 [Tab] 直到发送按钮高亮。按下 [Enter]。将会发送电子邮件。

如何使用键盘以回复一个电子邮件

  1. 如果电子邮件应用程序并没有执行，请进行以下操作：

    • 按下 [Ctrl]+[F6] 以高亮菜单栏。

    • 在菜单里按下 [Tab] 以高亮电子邮件。

    • 按下 [Enter]。电子邮件应用程序将会执行。

  2. 重复按下 [Ctrl]+[F6]，直到文件夹树里的一个文件夹高亮。使用方向键以选择需要的文件夹。要检视或隐藏一个子文件夹，按下 [Enter]。

  3. 如果高亮了需要的文件夹，重复按下 [Tab]，直到边栏里一个电子邮件高亮。

  4. 使用方向键以选择需要的电子邮件。

  5. 要回复电子邮件，重复按下 [Tab] 直到回复功能高亮。按下 [Enter] 以激活功能。
