7.2.2. Displaying appointments in the list view

How to display an appointment in the list view:

  1. Click on View in the toolbar. Select List.

  2. Open a calendar folder in the folder tree.

  3. If you use several personal calendar folders you can define the appointments to be displayed.

    • To display all the appointments from all your personal calendars, enable Show appointments from all private calendars in the folder tree.

    • To only display the appointments in the currently selected calendar, disable Show appointments from all private calendars.

  4. Click on an appointment in the list. The appointment's data is displayed in the detail view.

  5. To display another appointment, use one of the following methods:

    • Click on another appointment in the list.

    • Use the cursor keys to browse the list.
