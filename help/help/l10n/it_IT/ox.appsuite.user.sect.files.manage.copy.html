9.5.5. Copiare i file

È possibile copiare file in un'altra cartella: se si copia un file condiviso, i permessi non saranno copiati. Ciò significa che la copia del file non è condivisa.

Come copiare file in un'altra cartella:

Prerequisito: è necessario disporre dei permessi appropriati per creare oggetti nella cartella di destinazione.

  1. Selezionare uno o più file nell'area di visualizzazione. Fare clic sull'icona Azioni nella barra degli strumenti. Selezionare Copia dal menu.

  2. Selezionare una cartella nella finestra Copia. È possibile creare una nuova cartella facendo clic su Crea cartella.

  3. Fare clic su Sposta.
