9.4.1. Dateien als Link senden

Sie können einem Benutzer Links zu Dateien senden. Der Link wird in den Text der E-Mail eingefügt. Wenn der Benutzer diesen Link anklickt, wird die Datei auf einer neuen Seite angezeigt.

So senden Sie Links zu Dateien:

  1. Wählen Sie im Anzeigebereich eine Datei oder mehrere Dateien. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Als internen Link versenden.

  2. Vervollständigen Sie auf der Seite Neue E-Mail verfassen die Angaben zum Senden der E-Mail.
