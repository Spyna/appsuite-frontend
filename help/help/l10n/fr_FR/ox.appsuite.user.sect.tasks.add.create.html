8.4.1. Créer une nouvelle tâche

Pour créer une nouvelle tâche :

  1. Ouvrez un dossier de tâches dans l'arborescence de tâches.

    Remarque : choisissez un dossier dans lequel vous avez le droit de créer des tâches.

  2. Cliquez sur Nouveau dans la barre d'outils.

  3. Saisissez un sujet. Saisissez une description, si nécessaire.

    Pour voir le formulaire complet, cliquez sur Déployer le formulaire. Les fonctions optionnelles suivantes sont disponibles.

    • Indiquez la date de début de la tâche et la date d'échéance.

    • Pour obtenir un rappel de rendez-vous, sélectionnez un paramètre dans Rappel. La date et l'heure respectives seront saisies dans Date du rappel.

      Pour utiliser des valeurs personnalisées, définissez-les dans Date de rappel. Dans ce cas, Saisie manuelle sera présélectionné dans le champ Rappel.

    • Pour ajouter des détails tels que les informations de facturation, cliquez sur Détails. Saisissez les données requises.

    • Vous pouvez utiliser des fonctionnalités supplémentaires : créer des tâches récurrentes, ajouter des participants ou des groupes, ajouter des pièces jointes.

  4. Cliquez sur Créer.
