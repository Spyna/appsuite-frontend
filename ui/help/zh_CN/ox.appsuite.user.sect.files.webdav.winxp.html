9.6.2. 在 Windows XP 设置 WebDAV

如何在 Windows XP 设置 WebDAV 访问：

  1. 单击我的网路位置。在我的主机窗口单击我的网路位置或双击 Windows 桌面里的我的网路位置。

  2. 前往网路任务，打开对话窗口我的网路位置后单击添加一个网路位置。一个用于添加网络资源的向导启动。单击下一步。

  3. 在下一个对话窗口中选择选择另一个网络位置 。

  4. 输入以下地址： webdav://<地址>/servlet/webdav.infostore 其中<地址>需要用组件服务器的 IP 地址或 URL 代替。

  5. 输入您的组件服务器用户名称和密码。

  6. 在下一个对话窗口中，您可以向网络资源指定一个名称。

    备注：注： 如果您已经将您的编码设置为 ISO，请不要在文件夹或文档名称使用任何特殊字符。组件服务器使用 UTF-8 编码。

  7. 在下一个对话窗口中单击 完成。

结果： 您可以使用网路位置访问您的 文件 应用程序。
