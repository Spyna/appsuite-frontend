6.5.12. Printing E-Mails

In order to print E-Mails, you can do the following:

  • print an E-Mail's content

How to print an E-Mail's content:

  1. Select an E-Mail in the sidebar. Click on More in the display area.

  2. If required, change the printer settings.

  3. To print with the currently selected printer, click on Print. To print using the system dialog click on Print Using System Dialog... on the bottom left side.

Tip: You can also print the contents of multiple E-Mails at once.
