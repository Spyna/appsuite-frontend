Kontakte drucken

Um Kontakte zu drucken, haben Sie folgende Möglichkeiten:

  • eine Liste von Kontakten drucken

So drucken Sie eine Liste von Kontakten:

  1. Öffnen Sie im Ordnerbaum ein Adressbuch.

  2. Klicken Sie im Anzeigebereich auf Drucken.

  3. Falls erforderlich, ändern Sie die Druckereinstellungen. Klicken Sie links unten auf Über das Systemdialogfeld drucken....
