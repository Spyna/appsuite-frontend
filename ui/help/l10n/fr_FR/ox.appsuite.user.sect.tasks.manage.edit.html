8.5.1. Modifier des tâches

Vous pouvez modifier par la suite les données d'une tâche.

Pour modifier une tâche :

Vous devez au préalable avoir le droit de créer des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la barre latérale.

  2. Cliquez sur Modifier dans la zone d'affichage. Les données de la tâche s'affichent.

    Pour voir le formulaire complet, cliquez sur Déployer le formulaire.

  3. Modifiez les données.

  4. Cliquez sur Enregistrer.
