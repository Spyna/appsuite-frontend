7.8.2. E-Mail an Teilnehmer senden

Sie können eine E-Mail an alle Teilnehmer eines Termins senden.

So senden Sie eine E-Mail an alle Teilnehmer eines Termins:

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In einer Kalenderansicht klicken Sie auf einen Termin. Klicken Sie im Popup auf E-Mail an alle Teilnehmer senden.

    • In der Listenansicht wählen Sie einen Termin. Klicken Sie im Anzeigebereich auf E-Mail an alle Teilnehmer senden.

  2. Vervollständigen Sie die Angaben zum Senden der E-Mail.
