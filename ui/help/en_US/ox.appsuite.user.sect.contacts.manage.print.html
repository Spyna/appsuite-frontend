7.6.7. Printing contacts

In order to print contacts you can:

  • print a contact's phone data

How to print a contact's phone data:

  1. Open an address book in the folder tree.

  2. Select a contact in the sidebar. Click on More in the display area. Click on Print in the menu.

  3. If required, change the printer settings.

  4. To print with the currently selected printer, click on Print. To print using the system dialog click on Print Using System Dialog... at the bottom left side.

Tip: You can also print a phone list with multiple contacts.
