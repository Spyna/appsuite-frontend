Déplacer des dossiers

Vous pouvez déplacer les sous-dossiers de vos dossiers privés. Pour les autres dossiers, vous avez besoin des droits adéquats.

Pour déplacer un dossier :

Remarque : pour déplacer un dossier, vous devez avoir les droits d'administration sur le dossier ainsi que le droit de créer des sous-dossiers dans le dossier cible.

  1. Dans l'arborescence des dossiers, choisissez le dossier que vous voulez déplacer.

  2. Cliquez sur l'icône Modifier le dossier sous l'arborescence des dossiers. Choisissez Déplacer.

  3. Sélectionnez un dossier dans la fenêtre Déplacer le dossier. Cliquez sur Déplacer.
