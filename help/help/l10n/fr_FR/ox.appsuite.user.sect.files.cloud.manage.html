Comptes de stockage dans le cloud

9.8.1. Définition et modification des comptes de stockage de cloud

Vous disposez des options suivantes :

  • définir des comptes de stockage dans le cloud

  • modifier des comptes de stockage dans le cloud

  • supprimer les comptes de stockage de cloud

Comment définir un compte de stockage dans le cloud :

  1. Cliquez sur Ajouter un compte de stockage dans l'arborescence des dossiers. La fenêtre Ajouter un compte de stockage s'ouvre.

  2. Cliquez sur une icône. La procédure qui suit dépend de votre fournisseur de compte de stockage sur le cloud.

    • Si le fournisseur demande un identifiant de connexion, saisissez celui correspondant au compte de stockage dans le cloud.

    • Si le service de stockage dans le cloud vous demande des droits pour accéder aux données, autorisez-le.

Une entrée pour le compte de stockage sur le cloud apparaît dans l'arborescence des dossiers.

Comment modifier un compte de stockage dans le cloud :

  1. Cliquez sur l'icône menu Système sur la droite de la barre de menu. Cliquez sur Réglages dans le menu.

  2. Dans la barre latérale, cliquez sur Comptes.

  3. Cliquez sur Modifier à côté d'un compte dans la zone d'affichage. Les réglages apparaissent dans une fenêtre.

    Si vous utilisez plusieurs services du même fournisseur, par exemple courriel et stockage sur le cloud, sélectionnez le service à modifier.

  4. Modifier les réglages. Cliquez sur Enregistrer.

Comment supprimer un compte de stockage dans le cloud :

  1. Cliquez sur l'icône Menu Système à droite de la barre de menu. Dans le menu, cliquez sur Réglages.

  2. Dans la barre latérale, cliquez sur Comptes.

  3. Cliquez sur l'icône Supprimer à côté d'un compte.
