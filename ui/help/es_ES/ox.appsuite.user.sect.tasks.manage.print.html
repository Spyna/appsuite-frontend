11.5.6. Impresión de tareas

Para imprimir tareas puede hacer lo siguiente:

  • imprimir los datos de una tarea

Cómo imprimir los datos de una tarea:

  1. Seleccione una tarea en la barra lateral.

  2. Pulse en Más en el área de visualización. Pulse en Imprimir en el menú.

  3. Si fuese necesario, cambie la configuración de la impresora.

  4. Para imprimir usando la impresora actualmente seleccionada, pulse Imprimir. Para imprimir usando la ventana del sistema, pulse Imprimir usando la ventana del sistema... en la parte inferior izquierda.

Consejo: También puede imprimir los datos de múltiples tareas al mismo tiempo.
