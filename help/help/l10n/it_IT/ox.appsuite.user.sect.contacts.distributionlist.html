6.6. Aggiungere liste di distribuzione

Una lista di distribuzione consente l'invio simultaneo di un messaggio di posta a più destinatari digitando la lista di distribuzione nel campo dei destinatari del messaggio. Sono disponibili le seguenti opzioni:

  • Creare nuove liste di distribuzione

  • Salvare i destinatari di un messaggio come una lista di distribuzione

  • Creare una lista di distribuzione dall'elenco dei partecipanti
