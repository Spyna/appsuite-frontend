6.8.4. Kontakte verschieben oder kopieren

Sie können Kontakte oder Verteilerlisten in ein anderes Adressbuch verschieben oder kopieren.

Voraussetzung: Sie haben im Zieladressbuch die Berechtigung zum Anlegen von Objekten.

Hinweise:

  • Sie können keine Kontakte in das Globale Adressbuch verschieben oder kopieren.

  • Sie können keine Kontakte aus dem Globalen Adressbuch verschieben.

So verschieben oder kopieren Sie Kontakte in ein anderes Adressbuch:

  1. Wählen Sie in der Liste einen Kontakt, eine Verteilerliste oder mehrere Kontakte oder Verteilerlisten.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen . Klicken Sie im Menü auf Verschieben oder auf Kopieren.

    Ein Fenster öffnet sich.

  3. Wählen Sie ein Adressbuch. Bei Bedarf legen Sie ein neues Adressbuch an, indem Sie auf Ordner anlegen klicken.

    Klicken Sie auf Verschieben beziehungsweise auf Kopieren.

Tipp: Sie können die gewählten Objekte auch verschieben, indem Sie die Objekte in der Ordneransicht auf ein Adressbuch ziehen.
