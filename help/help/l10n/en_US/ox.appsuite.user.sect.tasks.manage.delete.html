8.6.7. Deleting tasks

You can delete one task or multiple tasks at once.

How to delete a task:

Warning: When deleting a task, this task is irrevocably lost.

  1. Select a task from the list.

  2. Click on Delete in the toolbar.

  3. Confirm that you want to delete the task.

Result: The task will be deleted.
