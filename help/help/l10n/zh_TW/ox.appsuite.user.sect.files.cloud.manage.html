雲端儲存空間帳號

9.8.1. 設定並編輯雲端儲存空間

有下列選項:

  • 設定 雲端儲存空間帳號

  • 編輯 雲端儲存空間帳號

  • 移除 雲端儲存空間帳號

如何設定雲端儲存空間帳號：

  1. 點擊資料夾樹中的 新增儲存空間帳號。新增儲存空間帳號 視窗將開啟。

  2. 點擊圖示。後續的程序視各雲端儲存空間供應商而定。

    • 若供應商詢問帳號密碼，請輸入該帳號的登入帳號密碼。

    • 若供應商詢問授權存取資料，請允許存取。

該雲端儲存空間帳號的項目將顯示於資料夾樹中。

如何編輯雲端儲存空間帳號：

  1. 點下選單列右邊的 系統選單 圖示 。點選選單的 設定 項目。

  2. 點擊側邊欄的 帳號。

  3. 點選顯示區域中帳號的 編輯。設定將顯示在視窗中。

    若您使用所選的供應商的多種服務，例如同時使用郵件與雲端儲存空間，請選擇要編輯的服務項目。

  4. 更改設定。點擊 儲存。

如何移除雲端儲存空間帳號:

  1. 點下選單列右邊的 系統選單 圖示 。點選選單的 設定 項目。

  2. 點擊側邊欄的 帳號。

  3. 點選帳號旁的 刪除 圖示。
