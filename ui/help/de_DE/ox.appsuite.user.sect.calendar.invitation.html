8.4. Termineinladung beantworten

Wenn ein Benutzer Sie als Teilnehmer zu einem Termin hinzufügt, informiert Sie der Infobereich über diesen Termin. Sie können Ihre Teilnahme an dem Termin zusagen, vorläufig zusagen oder ablehnen.

So beantworten Sie eine Termineinladung:

  1. Klicken Sie in der Menüleiste auf das Symbol Neue Objekte. Der Infobereich wird angezeigt.

  2. Klicken Sie unter Einladungen auf Bestätigen/Ablehnen.

  3. Geben Sie im Fenster Change confirmation status eine Anmerkung ein. Klicken Sie auf eine der Schaltflächen Bestätigen, Vorläufig, Ablehnen.

Sie können den Status Ihrer Terminbestätigung später wieder ändern.
