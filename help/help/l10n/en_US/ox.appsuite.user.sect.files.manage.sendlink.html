9.5.1. Sending files as a link

You can send other users links to files. The link is inserted in the E-Mail text. If the user clicks on this link, the file is displayed on a new page.

How to send links to files:

  1. Select one or several files in the display area. Click the Actions icon in the toolbar. Click on Send as internal link in the menu.

  2. In the Compose new E-Mail page, complete the details for sending the E-Mail.
