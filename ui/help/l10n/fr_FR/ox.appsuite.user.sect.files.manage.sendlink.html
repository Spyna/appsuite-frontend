9.5.1. Envoyer les fichiers comme lien

Vous pouvez envoyer à un utilisateur un lien vers un fichier. Vous pouvez envoyer des liens vers plusieurs fichiers à la fois.

Pour envoyer un lien vers un fichier :

  1. Cliquez sur un fichier dans la zone d'affichage. Dans la fenêtre qui s'ouvre, cliquez sur Plus. Choisissez Envoyer en tant que lien interne dans le menu.

  2. Dans la page Rédiger un nouveau courriel, remplissez les détails nécessaires pour envoyer le courriel.
