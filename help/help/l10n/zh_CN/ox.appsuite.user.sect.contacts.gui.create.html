6.1.8. 联系人编辑窗口

此窗口于创建一个新联系人或更改现有的之时使用。

为了打开窗口，进行以下动作之一：

  • 在工具条里单击新建。单击添加联系人。

  • 选择联系人。在工具条里单击编辑。

内容

  • 联系人图片、名称、公司

    单击联系人图像，以编辑或上载一个新联系人图像。

  • 联系人数据

    • 默认会显示那些数据：

      • 名字、姓氏、生日

      • 工作描述

      • 电子邮件、电话

  • 添加附件。添加文件。

  • 添加字段。为额外联系人数据添加字段。
