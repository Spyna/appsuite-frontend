10.1.1. Les composants de Texte et Feuille de calcul

Barre d'outils dans l'app Texte Contient les fonctions suivantes :

  • Bouton Nouveau document de texte. Crée un nouveau document de texte.

  • Bouton Ouvrir un document de texte. Ouvre un document existant.

Barre d'outils dans l'app Feuille de calcul Contient les fonctions suivantes :

  • Bouton Nouvelle feuille de calcul. Crée une nouvelle feuille de calcul.

  • Bouton Ouvrir une feuille de calcul. Ouvre une feuille de calcul existante.

Documents récents Contient une liste des documents ouverts récemment. Cliquez sur l'un des documents pour l'ouvrir.

Nouveau à partir d'un modèle Contient une liste de modèles pour divers documents. Cliquez sur l'un des modèles pour créer un nouveau document. Ce nouveau document sera une copie du modèle.

Lorsque vous créez un nouveau document ou modifiez un document existant, une Page d'édition de texte ou Page d'édition de feuille de calcul s'ouvre.
