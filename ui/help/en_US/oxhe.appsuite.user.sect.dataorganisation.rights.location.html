Where do I find folders shared by other users?

The following example shows where in the folder tree you can find the folders shared by other users. The following applies:

  • The user Timo Meier shares the following personal folders for you:

    • his calendar folder

    • his documents folder

    • his personal Team E-Mail folder

  • The user Tom Green creates the new folder VacationCalendar in the Public folders folder. He grants all users the permission to create objects in this folder.

  • The user Jean Dupont shares the folder from his personal documents folder. He did not share his personal documents folder.

You will see the following folders:

Tasks

Sent items

Shared folders

Timo Meier

Calendar

Shared calendar folder of Timo Meier

Inbox

user

Timo Meier

Team

Shared E-Mail folder of Timo Meier

Documents

UserStore

Your username

Your personal documents folder

Timo Meier

Shared documents folder of Timo Meier

Jean Dupont

MeetingMinutes

Shared subfolder in the documents folder of Jean Dupont

Calendar

Contacts

Public folders

Global address book

VacationCalendar

Folder created by Tom Green

Note: In order to see the shared E-Mail folder, you have to subscribe to this folder. Details can be found in : Subscribing to E-Mail folders.
