6.6.3. Modifier les contacts

Les données d'un contact peuvent être modifiées ultérieurement. La fenêtre de modification affiche les données les plus souvent utilisées. On peut aussi afficher les autres données.

Pour modifier un contact :

Vous devez au préalable avoir les droits appropriés pour modifier des objets dans le dossier contenant le contact.

  1. Choisissez un contact dans la barre latérale.

  2. Cliquez sur Modifier dans la zone d'affichage. Les données du contact s'affichent.

  3. Modifiez les données.

  4. Cliquez sur Enregistrer.
