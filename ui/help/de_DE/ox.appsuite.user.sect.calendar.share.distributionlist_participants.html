8.6.3. Verteilerliste aus Teilnehmern anlegen

Sie können aus den Teilnehmern eines Termins eine Verteilerliste anlegen.

So legen Sie aus den Teilnehmern eines Termins eine Verteilerliste an:

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Tag, Arbeitswoche, Woche oder Monat klicken Sie auf einen Termin. Klicken Sie im Popup auf Als Verteilerliste speichern.

    • In der Ansicht Liste wählen Sie in der Seitenleiste einen Termin. Klicken Sie im Anzeigebereich auf Als Verteilerliste speichern.

  2. Vervollständigen Sie die Angaben zum Anlegen der Verteilerliste.
