9.5.3. Afficher le lien vers un fichier

Vous pouvez afficher le lien vers un fichier qui a été enregistré dans l'application Fichiers.

Pour afficher le lien vers un fichier :

  • Selon la vue choisie, utilisez une des méthodes suivantes :

    • Dans la vue Icônes, cliquez sur un fichier. Dans la fenêtre qui s'ouvre, cliquez sur Afficher le lien.

    • Dans la vue Liste, choisissez un fichier dans la barre latérale. Cliquez sur Afficher le lien dans la zone d'affichage.
