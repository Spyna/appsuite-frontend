3.2. 鍵盤輸入

作為滑鼠的替代，您可以使用以下按鍵：

  • 要在輸入欄位中移動游標，使用以下按鍵： 方向鍵、[Home]、[End]。

  • 要瀏覽文件夾樹或邊欄，使用以下按鍵： [Page up]、[Page down]、[Space bar]、[Shift]+[Space bar]。

  • 要在文件夾樹或邊欄裏選擇元素，使用方向鍵。

  • 要選擇一個功能、一個輸入欄、或一個復選框，使用 [Tab] 鍵、[Shift]+[Tab]；選擇的元素將會被高亮顥示．

    使用 [Shift]+[Tab] 您可以以遞歸的順序選擇元件。

  • 要執行選擇的功能，按下 [Enter]。

  • 要啟用或停用選擇的復選框，按下 [Space bar]。

  • 要切換復選框，使用方向鍵。

展示如何使用鍵盤按制的步驟：

  • 如何使用鍵盤寄出一個新電子郵件

  • 如何使用鍵盤回覆一個電子郵件

如何使用鍵盤寄出一個電子郵件：

  1. 如果電子郵件軟體沒有執行， 進行以下動作；

    • 重複按下 [Tab] 直到 電子郵件 在選單行裏成高亮。

    • 按下 [Enter]。電子郵件 將會執行．

  2. 重複按下 [Tab] 直到 新 圖像 已高亮．按下 [Enter]。撰寫新電子郵件頁打開，輸入欄To啟用。

  3. 輸入收人電子郵件地址，然後按 [Enter]。

  4. 按下 [Tab]。輸入欄 主題 打開後，輸入主題。

  5. 按下 [Tab]。電子郵件文字的輸入欄啟用，輸入電子郵件文字。

  6. 為了寄出電子郵件，按下 [Tab]，寄出 按鍵已高亮；按下 [Enter]。電子郵件將會寄出。

  7. 要添加附件，重複按下 [Tab] 直到 Attachment功能被高亮；按下 [Enter]，添加檔案的控制元素會顯示。

    按下 [Enter]，一個用作選擇檔案的對話輸入欄會打開；視系統而定，使用 [Tab]鍵、[Enter]鍵、和方向鍵以選擇一個檔案和關閉系統對話輸入欄。

    重複按下 [Tab] 直到 寄出 按鍵高亮；按下 [Enter]。電子郵件將會寄出。

如何使用鍵盤以回覆一封電子郵件：

  1. 如果電子郵件軟體沒有起動，請進行以下操作：

    • 重複按下 [Tab] 直到 電子郵件 在選單裏高亮。

    • 按下 [Enter]，電子郵件 軟體將會起動。

  2. 重複按下 [Tab]，直到文件夾樹裏的一個文件夾高亮，使用方向鍵選擇所需的文件夾。要顯示或隱藏子文件夾，按下 [Enter]。

  3. 如果所需的文件夾為高亮，重複按下 [Tab]，直到邊欄裏一封電子郵件高亮。

  4. 使用方向鍵選擇所需的電子郵件。

  5. 要回覆電子郵件，重複按下 [Tab]，直到 回覆 功能高亮。按下 [Enter] 以激活功能。
