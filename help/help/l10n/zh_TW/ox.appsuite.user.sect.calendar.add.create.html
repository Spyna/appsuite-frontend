7.4.1. 建立新約會

如何創建新約會：

  1. 於資料夾樹開啟行事曆。

    註: 開啟您有權限建立約會的行事曆。選擇共享的行事曆後，將詢問您要在何處建立約會：

    • 若您以擁有者身分建立約會，約會將建立於擁有者自己的行事曆。

    • 若您邀請擁有者，約會則將建立於您的行事曆。

  2. 點下工具列中的 新增。

  3. 輸入主題。如果需要，輸入位置與描述。

    若要在其他行事曆建立約會，點選按鈕列中，行事曆 旁邊的行事曆名稱。選擇行事曆。

  4. 要設定約會的開始與結束時間，使用 開始日期 與 結束日期 下方的下列選項。

    • 點擊日期。日期選取器將開啟。輸入日期或在選擇器中選擇日期。

      為全日約會選中 全日。

    • 點擊時間。輸入時間或從清單中選擇時間。

    • 您可以點擊時間旁的時區按鈕，設定開始與結束時間的時區。您可以為開始與結束時間設定不同時區。

  5. 您可以使用其他功能:

    • 設定重複發生的約會

    • 設定約會的顯示方式

    • 使用約會提醒

    • 新增參與者和資源到約會

    • 新增附件

  6. 點選 建立。

提示： 此外，您還可以使用以下方法之一：

  • 在日曆表單中雙擊空閒區域，或用滑鼠拖曳起訖時間來建立新約會。

  • 要建立全日約會，選擇行事曆畫面中的 工作週 或 週。點選行事曆上方的日期。
