9.6. 用 WebDAV 访问信息文件

您可以用 WebDAV 如访问您磁盘的本地文件般，访问 磁盘 应用程序文件。与从组件 GUI 访问相比，WebDAV 有优点也有缺点：

  • 优点是能够以文件浏览器（比如视窗管理员），快速和直接访问。您无需登录到组件图像用户界面。

  • 缺点是您只能看到当前文档版本。版本历史或如注释的额外信息，皆不会显示。

警告：如果使用 WebDAV 删除一个文件，将失去所有版本而只是当前版本。

在以一个文件浏览器（比如视窗管理员）访问 磁盘 应用程序文件之前，您必须设置 WebDAV 访问。视系统软件而定，步骤各异：

  • 在 Linux 下设置 WebDAV

  • 在 Windows XP 下设置 WebDAV

  • 在 Windows Vista 下设置 WebDAV

  • 在 Windows 7 下设置 WebDAV
