11.1.5. Creating folders

You can create any number of subfolders in a personal folder. To create subfolders in a shared or public folder, you need to have the required permissions. Depending on the app, you can create folders or subfolders.

How to create a new subfolder in the E-Mail or Drive app:

  1. Select the folder in which you want to create the new subfolder in the folder tree.

    Note: Select a folder for which you have the permission to create subfolders.

  2. Click the Folder-specific actions icon next to folder name. Click on New subfolder.

  3. Enter a name in the New subfolder window. Click on Add.

How to create a new folder in the apps Address Book, Calendar, Tasks:

  1. In the folder tree, click on New private folder or on New public folder.

  2. Enter a name in the New folder window. Click on Add folder.
