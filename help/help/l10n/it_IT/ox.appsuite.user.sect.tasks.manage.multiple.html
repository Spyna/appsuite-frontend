8.6.8. Modificare più attività contemporaneamente

Le seguenti funzioni possono essere applicate a più attività contemporaneamente:

  • Spostare le attività in un'altra cartella

  • Marcare le attività come completate o incomplete

  • Eliminare le attività

  • Stampare più attività

Come eseguire una funzione per più attività contemporaneamente:

  1. Utilizzare uno o più metodi per selezionare almeno due attività:

    • se nessuna casella è visualizzata accanto alle attività nell'elenco, fare clic su Vista nella barra degli strumenti. Abilitare Caselle di selezione.

      Marcare le caselle di selezione per almeno due attività.

    • È possibile utilizzare anche le funzioni di selezione multipla del proprio sistema.

    Per selezionare tutte le attività, marcare la casella di selezione Seleziona tutto sopra l'elenco.

  2. Selezionare una funzione dalla barra degli strumenti.
