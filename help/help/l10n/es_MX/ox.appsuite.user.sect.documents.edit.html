10.3. Edición de documentos

Al editar documentos, puede hacer lo siguiente:

  • edición de texto, formateo de caracteres, formateo de párrafos

  • edición de hojas de cálculo, formateo de celdas, edición de fórmulas

  • descarga o impresión de documentos seleccionados o envío de los mismos por correo electrónico

  • búsqueda y reemplazo de texto

Se puede editar un documento en cualquiera de las aplicaciones Texto o Hoja de cálculo o en la aplicación Ficheros.

Cómo editar un documento:

  1. En la aplicación Texto, presione en Abrir documento de texto en la barra de herramientas.

    En la aplicación Hoja de cálculo, presione en Abrir hoja de cálculo en la barra de herramientas.

  2. Seleccione un documento en la ventana Abrir documento. Presione en Abrir. El documento se abrirá para su edición.

  3. Para editar el documento, utilice las técnicas habituales de las ya conocidas aplicaciones de Office.

    Para dar formato a caracteres o párrafos, utilice las herramientas de la barra de documento.

    Nota: Todos los cambios serán inmediatamente guardados.

  4. Para descargar o imprimir el documento seleccionado, o enviarlo en un correo electrónico, utilice los iconos que hay en la parte izquierda de la barra de herramientas.

  5. Para buscar y reemplazar elementos de texto, presione en el icono Cambiar búsquedade la parte izquierda de la barra de herramientas. Se mostrarán controles para buscar y reemplazar.

  6. Para terminar de editar y cerrar el documento, presione el icono Cerrar documentode la parte derecha de la barra de documento.

Cómo editar un documento en la aplicación Ficheros:

  1. Lance la aplicación Ficheros.

  2. Abra una carpeta que contenga documentos en el árbol de carpetas.

    Nota: Abra una carpeta en la que tiene permiso para crear objetos.

  3. Presione en un documento en el área de visualización. En la ventana emergente, presione Editar. El documento se abrirá para su edición.

    Use las técnicas descritas en las instrucciones para editar documentos for editing documents.
