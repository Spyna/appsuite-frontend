Modificare diversi file contemporaneamente

È possibile eseguire le seguenti funzioni per più file contemporaneamente:

  • aprire o salvare la versione corrente

  • inviare collegamenti ai file

  • inviare le versioni correnti come allegati di posta

  • visualizzare i collegamenti ai file

  • spostare o copiare file in un'altra cartella

  • eliminare i file

Come eseguire una funzione per più file contemporaneamente:

  1. Nella vista Elenco, fare clic sull'icona Seleziona sotto la barra laterale. Una casella di selezione sarà visualizzata accanto a ogni file nella barra laterale.

  2. Marcare le caselle per almeno due file.

  3. Fare clic su una funzione nell'area di visualizzazione.
