7.6.5. Importare calendari da file

È possibile importate i calendari che sono stati creati all'interno di altre applicazioni. Il calendario che si vuole importare deve essere disponibile come file iCal.

Come importare calendari da un file:

  1. Nell'albero delle cartelle, fare clic su Aggiungi nuovo calendario. Si aprirà un menu.

  2. Selezionare Carica file dal menu. Si aprirà la finestra Importa da file.

    Selezionare un file. Fare clic su Importa.
