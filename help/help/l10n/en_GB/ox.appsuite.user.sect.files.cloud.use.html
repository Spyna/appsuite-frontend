Cloud Storage Accounts

9.8.2. Using cloud storages

The following options are available:

  • You can use the folder tree to access the cloud storage contents.

    Note: Depending on the cloud storage's functionality, working with versions might not be supported.

  • Depending on the content, you can use the menu bar functions, e.g. to view pictures or to copy data.
