13.1.1. Die Bestandteile des Suchfensters

Das Suchfenster enthält die folgenden Bestandteile.

  • Eine Funktionsleiste mit App-Schaltflächen. Anklicken einer App-Schaltfläche bestimmt, in welcher App gesucht werden soll.

  • Eingabefeld Suchen. Hineinklicken oder Eingeben eines Suchbegriffs öffnet das Suchmenü. Je nach App zeigt das Suchmenü unterschiedliche Einträge.

  • Symbol Suchen . Startet eine Suche.

    Sobald eine Suche ausgeführt wurde, zeigt das Suchfenster weitere Elemente:

    • Die Schaltfläche Ordner. Sie zeigt den Ordner, in dem gesucht wurde. Um in einen anderen Ordner zu suchen, klicken Sie auf die Schaltfläche.

    • Eine oder mehrere Schaltflächen mit Suchkriterien. Einige Schaltflächen enthalten ein Auswahlmenü, mit dem Sie die Suchkriterien ändern können.

      Um ein Suchergebnis zurückzusetzen, schließen Sie die Schaltfläche eines Suchkriteriums.

    • Die Liste der gefundenen Objekte. Klicken Sie auf ein Objekt, um seine Daten im Popup anzuzeigen.

      Sie können in der Liste suchen, indem Sie eine neue Suche starten.
