9.1. Los componentes de Ficheros

Se puede encontrar una introducción a la interfaz de usuario en El interfaz de usuario.

Barra de herramientas Contiene los siguientes iconos:

  • Icono Añadir . Contiene las siguientes funciones:

    • Subir fichero nuevo. Crea un nuevo fichero.

    • Añadir nota. Crea un fichero de texto sencillo.

    • Nuevo documento de texto. Crea un nuevo documento de texto.

  • Icono Cambiar carpeta . Abre y cierra el árbol de carpetas.

  • Icono Cambiar búsqueda . Abre un campo para buscar ficheros con un nombre determinado.

  • Icono Vista . Permite seleccionar una de las siguientes vistas para el área de visión:

    • Lista

    • Iconos

    • Panel

Árbol de carpetas Muestra las carpetas. Permite la navegación por la estructura de carpetas. Para abrir el árbol de carpetas, pulse el icono Cambiar carpeta en la barra de herramientas. Están disponibles las siguientes funciones:

  • Si ha pulsado en una carpeta, se muestran sus ficheros.

  • El icono Acciones específicas de carpeta que hay junto a la carpeta seleccionada nos ofrece funciones para organizar datos y para intercambiar datos.

Barra de funciones La barra de funciones de la parte superior del área de visualización contiene funciones que están disponibles en todas las vistas.

  • Si se pulsa sobreSubir fichero nuevose abre la ventana para Subir un fichero nuevo.

  • Si abre una carpeta personal, se muestra el botón Compartir carpetaSi pulsa sobre este botón, se creará una publicación de esta carpeta.

  • Si la carpeta contiene imágenes, o ficheros de audio o vídeo, se mostrarán funciones para la reproducción de ficheros multimedia.

  • Si pulsa sobre uno de los iconos Lista , Iconos , Panel cambiará la vista en el área de visualización.

Área de visualización en la vista Lista Para cada fichero se muestra una fila con el siguiente contenido.

  • Una casilla de verificación para seleccionar uno o varios ficheros.

  • Un icono para cada fichero de la carpeta seleccionada. Se muestra una vista previa del fichero si es posible.

  • Nombre y tamaño del fichero

  • Fecha y hora del último cambio

Si se pulsa sobre un fichero, se abre una ventana emergente. Se muestran más datos y funciones.

Área de visualización en la vista Iconos Se muestra la siguiente información:

  • Ruta de la carpeta seleccionada. Para abrir otra carpeta, pulse en la ruta.

  • El botón Subir nuevo fichero. Abre la ventana para subir un nuevo fichero.

  • El botón Compartir esta carpeta. Crea una publicación para la carpeta seleccionada.

  • Si la carpeta contiene imágenes, ficheros de audio o de vídeo, se muestran funciones para reproducir ficheros multimedia.

  • Un icono por cada fichero de la carpeta seleccionada y, si está disponible, una vista previa del fichero.

Si se pulsa sobre un fichero, se muestra una ventana emergente. Se muestran más datos y funciones.

Área de visualización en la vista Panel Muestra un panel por cada fichero de la carpeta seleccionada. Si está disponible, se muestra una previsualización. Si se pulsa sobre un fichero, se abre una ventana emergente que incluye más datos y funciones.

Ventana emergente Si se pulsa en un fichero, se muestra una ventana emergente. Muestra la siguiente información:

  • el botón Abrir

    Si se pulsa en un documento, se muestra el botón Ver en lugar del botón Abrir.

  • Si se pulsa sobre un área editable, se muestra el botón Editar.

  • los botones Descargar , Borrar , Mover , Copiar

  • Si se pulsa en Más, se abre un menú que muestra las siguientes funciones: Compartir este fichero , Enviar por correo electrónico , Enviar como enlace interno , Mostrar enlace interno , Añadir al portal , Renombrar , Editar descripción , Bloquear.

    Nota: Algunos botones no se muestran si no se tienen los permisos adecuados.

  • Si se pulsa en un documento, el botón Editar como nuevo se muestra en el submenú Más.

  • nombre del fichero

  • una vista previa del fichero, si está disponible

  • comentarios del fichero

  • Ruta de navegación a la carpeta seleccionada. Para abrir otra carpeta, pulse en la ruta.

  • Botones para subir una nueva versión.

  • El botón Mostrar historial de versiones. Si se pulsa sobre este botón, se muestra una lista de versiones. La versión actual se muestra la primera. Para cada versión se muestra la siguiente información:

    • el nombre de fichero de la versión

    • comentarios de la versión, si están disponibles

    • nombre del usuario que subió la versión

    • fecha y hora de la carga de la versión
