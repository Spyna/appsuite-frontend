9.6.1. Configurer WebDAV sous Linux

Pour configurer l'accès par WebDAV sous Linux :

  1. Ouvrez Konqueror de KDE ou un navigateur similaire.

  2. Saisissez l'adresse suivante dans la barre d'adresse : webdav://<adresse>/servlet/webdav.infostore Remplacez <adresse> par l'adresse IP ou l'URL du serveur de travail collaboratif.

  3. Saisissez le nom d'utilisateur et le mot de passe qui vous permettent d'accéder au serveur de travail collaboratif.

Résultat : les fichiers de l'app Fichiers sont affichés dans le navigateur.
