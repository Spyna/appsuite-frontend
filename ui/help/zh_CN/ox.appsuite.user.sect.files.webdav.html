9.6. 以 WebDAV 访问文件

使用 WebDAV，您可以如访问磁盘中的本地文件一样访问 文件 应用程序文件。与从组件 GUI 访问相比，WebDAV 有优点也有缺点：

  • 优点是使用如 Windows Explore 之文件浏览器直接访问。您不用登陆到 Groupware GUI。

  • 缺点是您只能看到当前文档版本。版本信息或额外信息，例如回文皆不会显是。

警告：如果您以 WebDAV 删除一个文件，不只当前版本，所有版本都会丢失。为了删除一个特定版本，特别使用如何删除某版本：描述的方法。

以一个文件浏览器访问 文件 应用程序文件之前，例如 Windows Explorer，您必须设置 WebDAV 访问。是系统软件而定，步骤不一：

  • Linux 下设置 WebDAV

  • Windows XP 下设置 WebDAV

  • Windows Vista 下设置 WebDAV

  • Windows 7 下设置 WebDAV
