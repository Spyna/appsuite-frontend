6.7.5. Kontakte kopieren

Sie können einen Kontakt oder mehrere Kontakte gemeinsam in einen anderen Ordner kopieren.

So kopieren Sie einen Kontakt in einen anderen Ordner:

Voraussetzung: Sie haben für den Zielordner die Berechtigung zum Anlegen von Objekten.

  1. Wählen Sie in der Liste einen Kontakt.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Kopieren.

  3. Wählen Sie im Fenster Kopieren einen Ordner. Bei Bedarf legen Sie einen neuen Ordner an, indem Sie auf Ordner anlegen klicken.

  4. Klicken Sie auf Kopieren.
