10.2. Compartir

Al compartir datos con acceso de lectura o edición, puede cooperar con usuarios internos o colaboradores externos. Puede compartir elementos como correos electrónicos, libretas de direcciones, calendarios, tareas, carpetas o ficheros. Al compartir elementos, se otorgan permisos particulares a las personas con quienes comparte los elementos.

Tenga en cuenta lo siguiente:

  • Para compartir datos, debe tener permisos de administrador para la carpeta respectiva.

  • La libreta de direcciones global no se puede compartir.

  • Los correos electrónicos solo se pueden compartir con usuarios internos, no con colaboradores externos.

  • Las libretas de direcciones, los calendarios y las tareas solo pueden compartirse con colaboradores externos con acceso de lectura, no con acceso de edición.

  • Si comparte una carpeta en la aplicación Ficheros, puede definir si las subcarpetas también deben compartirse.

Tiene las siguientes opciones:

  • Compartición con enlaces públicos

    Puede dar este enlace a otras personas. Los datos se comparten con permiso de lectura. No se pueden otorgar otros permisos.

  • Invitar a un elemento compartido

    Los datos se comparten con usuarios internos o colaboradores externos con permisos de lectura o edición. Usted define los permisos para las personas individualmente.

  • Edición de comparticiones

    Puede cambiar permisos o añadir personas.

  • Reenvío de una invitación

  • Eliminar comparticiones

  • Visualización o edición de comparticiones en la aplicación Ficheros

  • Acceso a comparticiones de otros usuarios
