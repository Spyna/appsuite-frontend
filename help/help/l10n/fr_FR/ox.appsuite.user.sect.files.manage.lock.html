9.5.10. Verrouiller et déverrouiller des fichiers

Quand vous modifiez un fichier, vous pouvez le verrouiller. Le verrou sert à :

  • Informer les autres utilisateurs que le fichier est en train d'être modifié et qu'il pourrait ne plus être à jour.

  • Si le fichier se trouve dans un dossier partagé avec d'autres utilisateurs à qui vous avez donné les droits de modification, empêcher ces utilisateurs de modifier le fichier.

Vous pouvez verrouiller et déverrouiller un fichier isolé ou plusieurs fichiers à la fois.

Pour verrouiller un fichier :

  • Cliquez sur un fichier dans la vue détaillée. Dans la fenêtre qui s'ouvre, cliquez sur Plus. Choisissez Verrouiller dans le menu.

    Vous pouvez aussi sélectionner un ou plusieurs fichiers. Cliquez sur Plus dans la barre d'outils. Choisissez Verrouiller dans le menu.

Pour déverrouiller un fichier :

  • Cliquez sur un fichier verrouillé dans la vue détaillée. Dans la fenêtre qui s'ouvre, cliquez sur Plus. Choisissez Déverrouiller dans le menu.

    Vous pouvez aussi sélectionner un ou plusieurs fichiers. Cliquez sur Plus dans la barre d'outils. Choisissez Déverrouiller dans le menu.
