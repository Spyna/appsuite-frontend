Accedere alle archiviazioni sul cloud

9.7.2. Utilizzare le archiviazioni sul cloud

Sono disponibili le seguenti opzioni:

  • È possibile utilizzare l'albero delle cartelle per accedere ai contenuti dell'archiviazione sul cloud. In base al contenuto dell'archiviazione, è possibile utilizzare le funzioni della barra dei menu, ad es. per visualizzare immagini o copiare dati.

  • In base al contenuto, è possibile utilizzare le funzioni della barra dei menu, ad es. per visualizzare immagini o copiare dati.
