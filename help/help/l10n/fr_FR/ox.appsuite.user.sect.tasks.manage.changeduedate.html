8.6.3. Changer la date d'échéance d'une tâche

Vous pouvez modifier la date et l'heure d'échéance d'une tâche.

Pour changer la date d'échéance d'une tâche :

Pré-requis : vous devez avoir le droit de créer des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la liste.

  2. Cliquez sur Échéance dans la barre d'outils. Choisissez une nouvelle date.
