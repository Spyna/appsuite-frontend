9.2. Visualización de archivos y carpetas

De manera predeterminada, se muestra el contenido de la carpeta Mis archivos. Dependiendo del contenido del archivo están disponibles funciones diferentes:

  • visualizar el contenido de un archivo

  • mantener presentaciones en formato Powerpoint

  • descargar los archivos o contenido de una carpeta
