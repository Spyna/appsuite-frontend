3.4. Passwort ändern

So ändern Sie Ihr Passwort:

  1. Klicken Sie in der Menüleiste auf Portal.

  2. Klicken Sie auf der Portal-Kachel Benutzerdaten auf Mein Passwort.

  3. Geben Sie das aktuelle Passwort ein. Geben Sie zweimal das neue Passwort ein.

  4. Klicken Sie auf Passwort ändern.

Hinweis: Möglicherweise wird diese Kachel nicht angezeigt. In diesem Fall können Sie die Kachel hinzufügen.
