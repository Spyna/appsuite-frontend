9.1. Les composants de Lecteur

Vous trouverez une introduction à l'interface graphique dans L'interface utilisateur.

La barre d'outils Contient les icônes suivantes :

  • L'icône Ajouter . Propose les fonctionnalités suivantes :

    • Télécharger un nouveau fichier. Crée un nouveau fichier.

    • Ajouter une note. Crée un simple fichier de texte.

    • Nouveau document de texte. Crée un nouveau document de texte.

  • L'icône Dossier oui/non . Ouvre ou ferme l'arborescence des dossiers.

  • L'icône Recherche oui/non . Ouvre un champ de saisie pour chercher des fichiers avec un nom donné.

  • L'icône Vue . Permet de choisir une des vues suivantes pour la zone d'affichage :

    • Liste

    • Icônes

    • Pavé

L'arborescence des dossiers Affiche les dossiers. Permet de naviguer dans la structure des dossiers. Pour ouvrir l'arborescence des dossiers, cliquez sur l'icône Dossier oui/non dans la barre d'outils. Les fonctionnalités suivantes sont disponibles :

  • Lorsque l'on clique sur un dossier, ses dossiers sont affichés.

  • L'icône Action sur le dossier à côté du dossier sélectionné permet d'organiser les données et d'échanger les données.

Barre de fonctions La barre de fonctions dans la partie supérieure de la zone d'affichage donne accès aux fonctionnalités qui sont disponibles dans toutes les vues.

  • Lorsque l'on clique sur Télécharger un nouveau fichier, la fenêtre permettant de Télécharger un nouveau fichier s'ouvre.

  • Si vous avez ouvert un dossier privé, le bouton Partager ce dossier s'affiche. Si vous cliquez sur ce bouton, le dossier sera publié.

  • Si le dossier contient des fichiers image, audio ou vidéo, les fonctionnalités pour jouer les fichiers multimédia sont affichées.

  • Lorsque l'on clique sur une des icônes Liste , Icônes ou Pavé , la vue dans la zone d'affichage change.

La zone d'affichage dans la vue Liste Pour chaque fichier, une ligne est affichée, avec le contenu suivant :

  • Une case à cocher, qui permet de sélectionner un ou plusieurs fichiers.

  • Une icône pour chaque fichier du dossier sélectionné. Un aperçu est affiché, s'il est disponible.

  • Le nom est la taille du fichier

  • La date et l'heure de la dernière modification

Lorsque l'on clique sur un fichier, une fenêtre apparaît et affiche d'autres données et offre d'autres fonctionnalités.

La zone d'affichage dans la vue Icônes Elle affiche les informations suivantes :

  • Le chemin de navigation jusqu'au dossier sélectionné. Pour ouvrir un autre dossier, cliquez sur le chemin.

  • Le bouton Télécharger un nouveau fichier, qui ouvre la fenêtre permettant de télécharger un nouveau fichier.

  • Le bouton Partager ce dossier, qui publie le dossier sélectionné.

  • Si le dossier contient des fichiers images, audio ou vidéo, les options permettant de jouer des fichiers multimédia sont affichées.

  • Une icône pour chaque fichier du dossier sélectionné, ainsi qu'un aperçu du fichier s'il est disponible.

Lorsque l'on clique sur un fichier, ses données s'affichent dans une nouvelle fenêtre, où de nouvelles fonctionnalités sont également proposées.

La zone d'affichage dans la vue Pavé Affiche un pavé pour chaque fichier du dossier sélectionnée. Si c'est possible, un aperçu du fichier est affiché. Lorsque l'on clique sur un fichier, une fenêtre apparaît. Celle-ci contient des données supplémentaires et donne accès à de nouvelles fonctions.

La fenêtre qui apparaît Lorsque l'on clique sur un fichier, une fenêtre apparaît. Elle affiche les informations suivantes :

  • le bouton Ouvrir

    lorsque l'on clique sur un document, le bouton Affichage s'affiche à la place du bouton Ouvrir.

  • Lorsque l'on clique sur un document modifiable, le bouton Modifier s'affiche.

  • les boutons Télécharger , Supprimer , Déplacer et Copier

  • Lorsque l'on clique sur Plus, un menu s'ouvre et propose les fonctionnalités suivantes : Partager ce fichier , Envoyer par courriel , Envoyer en tant que lien interne , Afficher le lien interne , Ajouter au portail , Renommer , Modifier la description et Verrouiller.

    Remarque : certains boutons ne sont pas affichés si vous n'avez pas les droits adéquats.

  • Lorsque l'on clique sur un document, le choix Modifier en tant que nouveau apparaît dans le sous-menu Plus….

  • le nom du fichier

  • un aperçu du fichier, s'il est disponible

  • les commentaires sur le fichier

  • Le chemin de navigation vers le dossier choisi. Pour ouvrir un autre dossier, cliquez sur le chemin.

  • Un bouton pour télécharger une nouvelle version.

  • Le bouton Afficher l'historique des versions. Si l'on clique sur ce bouton, une liste de versions est affichée. La version courante se trouve en haut de la liste. Pour chaque version, les informations suivantes sont affichées :

    • le nom de fichier de la version

    • les commentaires de la version, si disponibles

    • le nom de l'utilisateur qui a mis en ligne la version

    • la date et l'heure de la mise en ligne de la version
