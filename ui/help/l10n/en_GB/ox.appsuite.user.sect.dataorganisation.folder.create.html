12.2.3. Creating folders

You can create any number of subfolders in a personal folder. To create subfolders in a shared or public folder, you need to have the required permissions. Depending on the app, you can create folders or subfolders.

  • In the E-Mail app you can create folders and subfolders.

  • In the Drive app you can only create subfolders, not folders.

  • In the Address Book, Calendar, Tasks apps you can only create folders, not subfolders.

How to create a new folder:

  1. Depending on the app, the procedure differs:

    • In the E-Mail app click on New folder below the folder tree.

    • In the Address Book, Calendar, Tasks apps click on Add private folder or on Add public folder in the folder tree.

      Note: In the Drive app you cannot create folders, only subfolders.

  2. Enter a name in the New folder window. Click on Add folder.

How to create a new subfolder:

  1. In the E-Mail or Drive app's folder tree select the folder in which you want to create a new subfolder.

    Note: Select a folder for which you have the permission to create subfolders.

  2. Click the Folder-specific actions icon next to the folder name. Click on New subfolder.

  3. Enter a name in the New folder window. Click on Add folder.
