6.5.10. Salvare i messaggi di posta

È possibile salvare un messaggio di posta come file di testo. Il file di testo ha estensione EML.

Come salvare un messaggio di posta:

  1. Selezionare un messaggio di posta nella barra laterale.

  2. Fare clic su Altro nell'area di visualizzazione. Selezionare Salva come file dal menu.

  3. Seguire le istruzioni per salvare il file EML.
