12.2.1. Tipi di cartella

I seguenti tipi di cartella esistono nell'albero delle cartelle:

  • Cartelle personali

    • Le cartelle personali contengono messaggi di posta, contatti, appuntamenti, attività e file. Gli altri utenti non possono vedere le vostre cartelle personali, a meno che non le abbiate condivise con loro.

    • Le cartelle personali di contatti, appuntamenti, attività e file sono posizionate in Privato nell'albero delle cartelle.

    • La cartella personale dei file è posizionata in I miei file.

  • Cartelle pubbliche

    • Le cartelle pubbliche contengono contatti, appuntamenti e documenti di interesse comune a tutti gli utenti. Ogni utente può creare cartelle pubbliche e condividerle con gli altri utenti.

    • Le cartelle pubbliche sono posizionate nell'albero delle cartelle in Pubblico. Qui troverete la rubrica globale con i dati di contatto di tutti gli utenti.

    • Le cartelle pubbliche dei file sono posizionate in File pubblici.

  • Cartelle condivise

    • La cartelle condivise sono state condivise con voi da altri utenti con permessi di lettura e scrittura.

    • Le cartelle condivise sono posizionate nell'albero delle cartelle in Condiviso.

    • Le cartelle condivise dei file sono posizionate in File condivisi.
