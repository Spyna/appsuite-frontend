Kalender-Einstellungen

So verwenden Sie die Kalender-Einstellungen:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf Calendar.

  3. Ändern Sie die Einstellungen.

Die folgenden Einstellungen sind verfügbar.

  • Interval in minutes

  • Start of working time

  • End of working time

  • Ansicht

  • Time range for the calender view

  • Time range for the team view

  • Time range for the list view

  • Default time for reminder

  • E-Mail notification for new, changed, deleted

  • E-Mail notification for appointment creator

  • E-Mail notification for appointment participant

Interval in minutes. Bestimmt die Unterteilung des Zeitrasters in den Kalenderansichten Tag, Arbeitswoche, Woche.

Start of working time. Bestimmt den Beginn der Arbeitszeit.

End of working time. Bestimmt das Ende der Arbeitszeit.

Ansicht. Bestimmt die Standard-Ansicht der Termine.

Time range for the calender view. Bestimmt den Standard-Zeitabschnitt für die Ansicht Kalender.

Time range for the team view. Bestimmt den Standard-Zeitabschnitt für die Ansicht Team.

Time range for the list view. Bestimmt den Standard-Zeitabschnitt für die Ansicht Liste.

Default time for reminder. Bestimmt die Standard-Zeit für die Termin-Erinnerung.

E-Mail notification for new, changed, deleted. Bestimmt, ob Sie in den folgenden Fällen eine E-Mail-Nachricht erhalten: Ein Termin, bei dem Sie Teilnehmer sind, wurde neu angelegt, geändert oder gelöscht.

E-Mail notification for appointment creator. Bestimmt, ob Sie in den folgenden Fällen eine E-Mail-Nachricht erhalten: Zu einem Termin, bei dem Sie Organisator sind, hat ein Teilnehmer zugesagt oder abgesagt.

E-Mail notification for appointment participant. Bestimmt, ob Sie in den folgenden Fällen eine E-Mail-Nachricht erhalten: Zu einem Termin, bei dem Sie Teilnehmer sind, hat ein anderer Teilnehmer zugesagt oder abgesagt.
