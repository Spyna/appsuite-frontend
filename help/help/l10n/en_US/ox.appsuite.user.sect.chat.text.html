10.2. Sending or Reading Messages

The following options exist:

  • send messages and files to a group or single contact

  • search for and read messages of a conversation

How to send a message in the Chat window:

  1. Select a contact or a group from the sidebar.

  2. Enter a message in the input field.

    To send the message, press [Enter]. An icon on the right border of the message indicates its delivery status:

    • A gray check mark: The message has been arrived on the server.

    • Two gray check marks: The message has been transferred by the server to all group members.

    • Two blue check marks: The message has been viewed by all group members.

Options:

  • To create a line break within a message, press [Shift]+[Enter].

  • If you send the message to a group, you can address a contact specifically. To do so, enter [@] followed by the person's name and the message. The contact addressed will see a red marker at the group name in the list.

  • In order to send a file, click the Attachment icon next to the input field. Select a file.

How to read a message in the Chat windows:

  1. Select a conversation in the sidebar.

    Conversations with unread messages will be displayed as follows:

    • The name of the contact or group will be displayed in bold letters.

    • A figure indicates the number of unread messages.

  2. Read the conversation's messages. To view earlier messages in this conversation, use the icons on the right border to scroll up.

Options:

  • To search for conversations, enter a search term in the Search or start a new chat input field.

  • To view recently used conversations, click on Recent conversations in the sidebar.

  • To view all public channels, click on All channels in the sidebar.
