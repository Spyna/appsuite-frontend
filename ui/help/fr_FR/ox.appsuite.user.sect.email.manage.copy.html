6.5.2. Copier des messages

Vous pouvez copier un message isolé ou un fil de discussion complet dans un autre dossier.

Pour copier un courriel :

Il faut auparavant que vous disposiez du droit de créer des objets dans le dossier cible.

  1. Choisissez un message dans la barre latérale.

  2. Cliquez sur Copier dans la zone d'affichage. Choisissez Copier dans le menu.

  3. Sélectionnez un dossier dans la fenêtre Copier. Cliquez sur Copier.

Conseil : vous pouvez aussi copier plusieurs messages à la fois ou tous les messages d'un fil de discussion.
