Adressbuch-Einstellungen

So verwenden Sie die Adressbuch-Einstellungen:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf Adressbuch.

  3. Ändern Sie die Einstellungen.

Die folgenden Einstellungen sind verfügbar.

  • Kontakte aus Administrator-Gruppe anzeigen

Kontakte aus Administrator-Gruppe anzeigen Bestimmt, ob auch Kontakte angezeigt werden, die zur Administrator-Gruppe gehören.
