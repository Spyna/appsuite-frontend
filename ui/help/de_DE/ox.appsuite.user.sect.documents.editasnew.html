Dokument unter neuem Namen bearbeiten

Sie können ein neues Dokument als Kopie eines vorhandenen Dokuments anlegen, indem Sie das vorhandene Dokument unter einem neuen Namen bearbeiten.

So bearbeiten Sie ein Dokument unter einem neuen Namen:

  1. Starten Sie die App Dateien .

  2. Öffnen Sie im Ordnerbaum einen Ordner, der Dokumente enthält.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Bearbeiten von Objekten haben.

  3. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Symbole klicken Sie auf ein Dokument. Klicken Sie im Popup auf Weitere. Klicken Sie im Menü auf Edit as new.

    • In der Ansicht Liste wählen Sie in der Seitenleiste ein Dokument. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Edit as new.

    Das Dokument wird zur Bearbeitung geöffnet.

  4. Klicken Sie rechts oben in der Dokumentleiste auf Unnamed. Geben Sie einen Namen für das Dokument ein.

  5. Sie können jetzt die Funktionen zum Bearbeiten eines Dokuments verwenden.
