Los componentes de las Apps

Se puede encontrar una introducción a la interfaz de usuario en El interfaz de usuario.

En la página Sus aplicaciones podrá ver:

  • el botón Administrar aplicaciones. Lanza el gestor de aplicaciones, que le ayuda a Organizar sus apps.

  • una vista general de todas las aplicaciones disponibles. Para lanzar una aplicación, presione el icono correspondiente.
