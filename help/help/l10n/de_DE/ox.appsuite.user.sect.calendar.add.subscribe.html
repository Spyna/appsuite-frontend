7.4.11. Termine aus externen Kalendern abonnieren

Sie können Termine aus externen Kalendern in einen Kalenderordner importieren, indem Sie den externen Kalender abonnieren. Sie können diese Termine importieren:

  • Termine aus Ihrem Google-Kalender

  • Termine aus Kalendern, die von SchedJoules bereitgestellt werden.

    Dazu zählen u. a. internationale Feiertage, Sportereignisse, Medienereignisse, Börsen.

Dabei gilt:

  • Die abonnierten Termine werden in der Groupware in einem eigenen Kalender gespeichert. Änderungen, die Sie in diesem Kalender vornehmen, werden nicht in den externen Kalender übertragen.

  • Änderungen, die im externen Kalender stattfinden, können Sie in die Groupware übernehmen, indem Sie abonnierte Daten aktualisieren.

So abonnieren Sie Termine aus Ihrem Google Kalender:

  1. Klicken Sie im Ordnerbaum auf Kalender abonnieren. Das Fenster Kalender abonnieren wird geöffnet.

  2. Klicken Sie auf das Symbol Google. Ein neues Browserfenster öffnet sich. Erlauben Sie den Zugriff auf Ihren Google-Kalender.

    Die Termine werden in einen neuen Kalender importiert. Sobald der Import beendet ist, wird das Fenster Kalender abonnieren geschlossen.

Tipp: Ein Kalender mit abonnierten Terminen wird durch das Symbol Abonnements gekennzeichnet. Anklicken des Symbols öffnet die Einstellungen für Abonnements.

So abonnieren Sie Termine von Schedjoules:

  1. Klicken Sie im Ordnerbaum auf Subscribe schedjoules calendar. Das Fenster Add calendar wird geöffnet.

  2. Wählen Sie die Sprache und das Land des Kalenders, den Sie abonnieren möchten.

    Wählen Sie einen Kalender oder meherere Kalender.

  3. Klicken Sie auf Speichern.

    Die Termine werden in einen neuen Kalender importiert.

Tipp: Ein Kalender mit abonnierten Terminen wird durch das Symbol Abonnements gekennzeichnet. Anklicken des Symbols öffnet die Einstellungen für Abonnements.
