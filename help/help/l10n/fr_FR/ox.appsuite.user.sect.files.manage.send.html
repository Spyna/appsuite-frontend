9.4.2. Envoyer des fichiers comme pièces jointes

Vous pouvez envoyer les versions courantes des fichiers comme pièces jointes à un courriel.

Pour envoyer des fichiers comme pièces jointes à un courriel :

  1. Sélectionner un ou plusieurs fichiers dans la zone d'affichage. Cliquez sur l'icône Actions dans la barre d'outils. Cliquez sur Envoyer par courriel dans le menu.

    Vous pouvez également utiliser l'icône Actions dans l'Afficheur.

  2. Dans la page Rédiger, remplissez les détails nécessaires pour envoyer le courriel.
