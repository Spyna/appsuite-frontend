6.5.12. Mehrere E-Mails gemeinsam bearbeiten

Sie können folgende Funktionen für mehrere E-Mails gemeinsam ausführen:

  • E-Mails weiterleiten

  • E-Mails in einen anderen Ordner verschieben oder kopieren

  • E-Mails als gelesen oder als ungelesen markieren

  • E-Mails mit einem Label kennzeichnen

  • E-Mails löschen

So führen Sie eine Funktion für mehrere E-Mails gemeinsam aus:

  1. Klicken Sie unterhalb der Seitenleiste auf das Symbol Auswählen . In der Seitenleiste wird vor jeder E-Mail ein Kontrollfeld angezeigt.

  2. Aktivieren Sie die Kontrollfelder bei mindestens 2 E-Mails.

  3. Klicken Sie im Anzeigebereich auf eine Funktion.
