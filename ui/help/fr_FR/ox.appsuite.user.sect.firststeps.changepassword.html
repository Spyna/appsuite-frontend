3.4. Changer le mot de passe

Pour changer votre mot de passe :

  1. Cliquez sur Portail dans la barre de menu

  2. Cliquez sur Mon mot de passe dans le pavé de portail Données utilisateur.

  3. Saisissez un mot de passe. Saisissez le nouveau mot de passe deux fois.

  4. Cliquez sur Changer le mot de passe.

Remarque : ils se peut que ce pavé ne soit pas affiché. Si c'est le cas, vous pouvez l'ajouter.
