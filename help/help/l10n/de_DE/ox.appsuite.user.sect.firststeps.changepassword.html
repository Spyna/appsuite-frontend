2.3. Passwort ändern

Hinweis: Je nach Konfiguration weichen die Schritte zum Ändern des Passworts möglicherweise von dieser Anleitung ab. Wenden Sie sich in diesem Fall an Ihren Administrator oder Hoster.

So ändern Sie Ihr Passwort:

  1. Klicken Sie in der Menüleiste auf das Symbol App Launcher . Klicken Sie im App Launcher auf Portal.

  2. Wenn das Widget Benutzerdaten nicht angezeigt wird, klicken Sie rechts oben auf Widget hinzufügen. Klicken Sie auf Benutzerdaten.

    Klicken Sie im Widget Benutzerdaten auf Mein Passwort.

  3. Ändern Sie Ihr Passwort.

Benutzeroberfläche:

  • Widget Benutzerdaten
