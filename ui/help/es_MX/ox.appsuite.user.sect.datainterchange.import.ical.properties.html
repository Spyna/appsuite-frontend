14.1.1.2. ¿Qué propiedades del objeto iCal no están admitidas?

Tenga en cuenta las siguientes limitaciones:

  • Se permiten citas tales como "El último domingo de un mes". No se permiten citas en días contados desde el fin del mes. Ejemplo: El penúltimo domingo de un mes. Si una cita incluye tal información, la cita no se importará.

  • No se admite la repetición de alarmas. Ejemplo: "Recuérdamelo cuatro veces". Si una cita contiene información como esa, tal información será ignorada.
