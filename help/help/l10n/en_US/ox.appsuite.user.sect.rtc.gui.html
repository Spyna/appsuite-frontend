12.1. The Messenger Components

An introduction to the user interface can be found in The User Interface.

Toolbar Contains the following icons:

  • Start new group-conversation icon . Starts a new group chat.

Sidebar Shows a list of chats. The following details are displayed: the contact's name, the date of the last conversation, the first line of the last conversation A colored dot next to the contact indicates the current status. The following functions are available:

  • If clicking on a chat, its content is displayed in the display area.

Display area Displays the chat's content and the functions available:

  • The contact's name and status

  • Buttons: Video, Call

  • When chatting in real-time:

    • Sent and received messages

    • Input field for your real-time message

    • Send button for sending a real-time message

  • When communicating via calls:

    • The contact's name and picture

    • The duration of the call

    • Button for hanging up, button for muting the microphone

    • Send button for sending a real-time message

    Tip: The contact name, the call duration and a button for hanging up are also displayed in the menu bar.

  • When communicating via video calls:

    • A large live picture from your contact's camera in the background

    • A small live picture from your camera in the foreground

    • Button for switching off your camera on your chat partner's screen, button for hanging up, button for muting your microphone

    • Send button for sending a real-time message

    Tip: The contact name, the call duration and a button for hanging up are also displayed in the menu bar.

Buddy list Shows a list of contacts with which you can chat. In order to open the buddy list, click the Buddy list icon on the right side of the menu bar. The following information are displayed in the buddy list:

  • The contact's name If the contact's personal data include a picture, it is displayed.

  • The current connection status of the contact. The connection status is also indicated by a color.

  • If you click on a contact that is on-line, icons for starting a conversation with this contact will be displayed.
