7.2.2. Mehrere Zeitzonen anzeigen

Zusätzlich zur Standard-Zeitzone können Sie Zeitzonen anzeigen, die Sie als Favoriten gekennzeichnet haben. Die Standard-Zeitzone können Sie bei Bedarf in den Grundeinstellungen ändern.

So zeigen Sie mehrere Zeitzonen in einer Kalenderansicht an:

  1. Klicken Sie in der Werkzeugleiste auf Ansicht. Wählen Sie einen der folgenden Einträge: Tag, Arbeitswoche, Woche.

  2. Klicken Sie links oberhalb des Kalenderblatts auf die Schaltfläche Zeitzone.

    Ein Menü öffnet sich. Aktivieren Sie im Menü unterhalb von Favoriten eine Zeitzone.

    Durch Anklicken von Favoriten verwalten können Sie Zeitzonen zu Favoriten hinzufügen oder entfernen.
