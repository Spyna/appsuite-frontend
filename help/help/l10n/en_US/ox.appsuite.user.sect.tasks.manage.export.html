8.6.5. Exporting tasks

You can export tasks in the iCalendar format. You can e.g. use this format to exchange tasks with other calendar applications.

How to export tasks:

  1. Select a task or multiple tasks from the list.

  2. Click the More actions icon in the toolbar. Click the Export menu item. The Export selected dialog window opens.

  3. Click on Export. Complete the entries.
