13.1.3. Objekte suchen

Sie haben die folgenden Möglichkeiten. Sie können bestimmen, in welcher App gesucht werden soll:

  • Sie können in der aktuellen App suchen.

  • Sie können das Suchfenster öffen, um in einer beliebigen App zu suchen.

So suchen Sie nach Objekten:

  1. Klicken Sie in der Menüleiste auf das Symbol Suchen . Das Suchfenster öffnet sich.

  2. Wählen Sie im Suchfenster die App, in der sie suchen möchten.

  3. Verwenden Sie eine dieser Methoden:

    • Wenn Sie einen Suchbegriff verwenden möchten, geben Sie in das Eingabefeld Suchen den Suchbegriff ein. Das Suchmenü wird angezeigt.

    • Wenn Sie nach Objekten mit einem bestimmten Merkmal suchen möchten, klicken Sie in das Eingabefeld Suchen. Das Suchmenü wird angezeigt.

  4. Wählen Sie einen Eintrag im Suchmenü.

    Ergebnis: Eine Liste der gefundenen Objekte wird im Suchfenster angezeigt.

  5. Um in der Liste der gefundenen Objekte weiter zu suchen, starten Sie eine weitere Suche.

    Um das Ergebnis zurück zu setzen, schließen Sie die Schaltfläche des Suchkriteriums.

So suchen Sie in der aktuellen App nach Objekten:

  1. Wenn Sie nach E-Mails suchen wollen, öffnen Sie im Ordnerbaum den E-Mail-Ordner, den Sie durchsuchen wollen.

    In den anderen Apps werden stets alle Ordner durchsucht.

  2. Verwenden Sie eine dieser Methoden:

    • Wenn Sie einen Suchbegriff verwenden möchten, geben Sie in der Menüleiste in das Eingabefeld Suchen den Suchbegriff ein. Das Suchmenü wird angezeigt.

    • Wenn Sie nach Objekten mit einem bestimmten Merkmal suchen möchten, klicken Sie in der Menüleiste in das Eingabefeld Suchen. Das Suchmenü wird angezeigt.

  3. Wählen Sie einen Eintrag im Suchmenü.

    Ergebnis: Eine Liste der gefundenen Objekte wird im Suchfenster angezeigt.

  4. Um in der Liste der gefundenen Objekte weiter zu suchen, starten Sie eine weitere Suche.

    Um das Ergebnis zurück zu setzen, schließen Sie die Schaltfläche des Suchkriteriums.
