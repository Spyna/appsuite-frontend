9.2. Ver ficheros

De manera predeterminada se muestra el contenido de su carpeta de ficheros personales. La carpeta se denomina Mis ficheros.

Cómo mostrar un fichero:

  1. Pulse el icono Ver de la barra de herramientas. Seleccione una entrada.

    También puede utilizar los iconos de la barra de funciones para seleccionar una vista.

  2. Abra una carpeta del árbol de carpetas.

  3. Para encontrar un determinado fichero rápidamente, use la función de búsqueda.

  4. Pulse en un fichero del área de visualización. Los datos se muestran en la ventana emergente.
