12.2.2. 在文件夹结构里导航

以下功能可供使用：

  • 打开或关闭文件夹树

  • 更改文件夹树宽度

  • 在文件夹树里打开或选择一个文件夹。

如何打开文件夹树：

  • 使用以下方法之一：

    • 在工具栏里单击切换文件夹图标。

    • 单击边栏里列表下的文件夹名称。

如何更改文件夹树宽度：

  1. 移动光标到文件夹树的右边框。一个双箭头将会显示。

  2. 拖动边框到左方或右方。

如何关闭文件夹树：

  • 使用以下方法之一：

    • 单击文件夹树下面的关闭图标。

    • 在工具栏里单击切换文件夹图标。

如何打开某个文件夹：

  1. 如果文件夹树关闭，verwenden Sie eine der folgenden Methoden, um ihn zu öffnen:

    • 在工具栏里单击切换文件夹图标。

    • 单击边栏里列表下的文件夹名称。

  2. 要检视一个文件夹的子文件夹，单击文件夹名称旁边的箭头。

  3. 要打开一个文件夹，单击它。

结果：显示文件夹内容
