Carnet d'adresses

6.2. Afficher des contacts

Votre carnet d'adresses peut être trouvé dans l'arborescence des dossiers sous Contacts. Dans les réglages du carnet d'adresses, vous pouvez définir le carnet d'adresses à ouvrir par défaut.

Pour afficher un contact :

  1. Ouvrez un carnet d'adresses dans l'arborescence des dossiers.

  2. Pour afficher les contacts avec une initiale donnée, cliquez sur la lettre dans la barre de navigation.

  3. Cliquez sur un contact dans la liste. Les données du contact sont affichées dans la vue détaillée.

    Vous pouvez ouvrir le contact dans une nouvelle fenêtre en faisant un double-clic sur le contact apparaissant dans la liste.

  4. Il y a plusieurs méthodes pour afficher un autre contact :

    • Cliquez sur un autre contact dans la liste.

    • Utiliser les flèches pour naviguer dans la liste.
