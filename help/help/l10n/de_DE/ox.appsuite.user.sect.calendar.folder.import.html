7.6.3. Kalender aus Datei importieren

Sie können Kalender importieren, die von anderen Anwendungen erstellt wurden. Der Kalender, den Sie importieren wollen, muss als Datei im iCal-Format vorhanden sein.

So importieren Sie einen Kalender aus einer Datei:

  1. Klicken Sie innerhalb der Ordneransicht auf Neuen Kalender hinzufügen. Ein Menü öffnet sich.

    Klicken Sie auf Datei hochladen. Das Fenster Aus Datei importieren öffnet sich.

  2. Klicken Sie auf Datei hochladen. Wählen Sie eine Datei.

  3. Bestimmen Sie einen Namen für den neuen Kalender. Klicken Sie auf Importieren.
