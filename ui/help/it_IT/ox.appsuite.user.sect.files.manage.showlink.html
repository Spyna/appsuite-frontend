Mostrare il collegamento a un file

È possibile mostrare il collegamento a un file che viene salvato nell'applicazione File.

Come mostrare il collegamento a un file:

  • In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Mostra collegamento.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic su Mostra collegamento nell'area di visualizzazione.
