12.5.2. Marquer des tâches comme terminées

Vous pouvez marquer une ou plusieurs tâches comme terminées.

Pour marquer une tâche comme terminée :

Vous devez au préalable avoir le droit de modifier des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la barre latérale.

  2. Cliquez sur Terminé dans la zone d'affichage.
