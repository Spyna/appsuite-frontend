Aufgaben verschieben

Sie können eine Aufgabe oder mehrere Aufgaben gemeinsam in einen anderen Ordner verschieben.

So verschieben Sie eine Aufgabe:

Voraussetzung: Sie haben für den Zielordner die Berechtigung zum Anlegen von Objekten.

  1. Wählen Sie in der Seitenleiste eine Aufgabe.

  2. Klicken Sie im Anzeigebereich auf Verschieben.

  3. Wählen Sie im Fenster Verschieben einen Ordner. Klicken Sie auf OK.
