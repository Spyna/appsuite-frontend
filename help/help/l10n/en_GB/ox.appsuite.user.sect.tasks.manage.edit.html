8.6.1. Editing tasks

You can edit a task's data at a later point.

How to edit a task:

Prerequisite: You have the permission to create objects in the folder containing the task.

  1. Select a task from the list.

  2. Click on Edit in the tool bar. The task's data are displayed.

  3. Edit the data.

    A description of the input fields can be found in View for creating or editing.

  4. Click on Save.
