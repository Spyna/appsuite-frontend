8.6.2. Marquer des tâches comme terminées

Vous pouvez marquer une ou plusieurs tâches comme terminées.

Pour marquer une tâche comme terminée :

Pré-requis : vous devez avoir le droit de créer des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la liste.

  2. Cliquez sur Terminé dans la barre d'outils.

    L'intitulé du bouton devient Défait. Un clic sur le bouton fait passer la tâche à l'état non terminé.
