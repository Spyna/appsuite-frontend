9.6. 用 WebDAV 访问文件

使用 WebDAV，您可以如访问您硬盘上本地文件一样，访问 文件 应用程序文件。与从组件 GUI 访问相比，WebDAV 有优点也有缺点：

  • 优点是使用文件浏览器（文件管理员）快速直接访问。您不需要登陆组件图形界面。

  • 缺点是您只能看到当前文档版本。既不会显示版本历史，也不会显示像意见的额外信息。

警告：用 WebDav 删除文件时，将删除所有文件版本，而不仅仅是当前版本。 要删除具体版本，只能使用 : 如何删除某版本： 所描述的方法。

在使用文件浏览器（文件管理员）访问 文件 应用程序文件前，您必须设置 WebDAV 访问。是系统软件而定，步骤不一：

  • 在 Linux 设置 WebDAV

  • 在 Windows XP 设置 WebDAV

  • 在 Windows Vista 设置 WebDAV

  • 在 Windows 7 设置 WebDAV
