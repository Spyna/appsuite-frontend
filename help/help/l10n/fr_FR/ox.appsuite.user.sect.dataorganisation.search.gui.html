12.1.1. Les composants de la fenêtre de recherche

La fenêtre de recherche inclut les éléments suivant :

  • Une barre d'outils avec des boutons correspondant aux applications. Un clic sur l'un de ces boutons définit l'application à utiliser pour la recherche.

  • Un champ de saisie Chercher. Lorsque l'on saisit un terme de recherche, les éléments du menu de recherche sont adaptés en conséquence.

  • Le bouton Dossier. Celui-ci affiche le dossier utilisé pour la recherche. Pour effectuer une recherche dans un autre dossier, cliquez sur ce bouton.

  • Un menu de recherche dont les éléments dépendent des applications. Si vous cliquez sur l'un de ces éléments, la recherche est lancée.

  • Icône Chercher . Lance la recherche.

    Dès que la recherche est terminée, la fenêtre de recherche affiche d'autres éléments :

    • Un ou plusieurs boutons avec des critères de recherche. Certains boutons comportent une liste déroulante dans laquelle vous pouvez changer le critère de recherche.

      Pour réinitialiser un résultat de recherche, fermez le bouton d'un critère de recherche.

    • La liste d'objets trouvés. Cliquez sur un objet pour afficher ses détails dans une nouvelle fenêtre.

      Vous pouvez effectuer une recherche parmi la liste des résultats en démarrant une nouvelle recherche.
