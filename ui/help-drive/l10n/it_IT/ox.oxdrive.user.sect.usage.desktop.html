2.1. Utilizzare OX Drive su una workstation

Come utilizzare le funzioni di OX Drive su una workstation:

  1. Aprire il menu contestuale dell'icona OX Drive.

    In base al sistema, esistono diversi modi di aprire il menu contestuale di OX Drive.

    • In MS Windows, fare clic con il tasto destro sull'icona OX Drive nella barra delle attività a destra dell'area informazioni. Potrebbe essere necessario visualizzare prima l'icona facendo clic su Mostra icone nascoste.

  2. Selezionare una funzione dal menu contestuale.

Sono disponibili le seguenti funzioni.

  • Sospendere la sincronizzazione

  • Aprire la cartella di OX Drive

  • File modificati di recente

  • Preferenze

  • Errori di sincronizzazione

  • Uscire da OX Drive

Sospendere la sincronizzazione Ferma la sincronizzazione. Per procedere con la sincronizzazione, fare clic su Riprendi sincronizzazione.

Aprire la cartella di OX Drive Apre la cartella locale nel navigatore dei file del proprio sistema.

File modificati di recente Visualizza un elenco di file modificati di recente.

Preferenze Apre la finestra Preferenze. Saranno disponibili le seguenti impostazioni.

  • Scheda Generale

    • Quota. Mostra informazioni sullo spazio disponibile.

      Per ottenere ulteriore spazio, fare clic su Acquista quota.

    • Lingua

  • Scheda Avanzate

    • Avvio automatico. Specifica se OX Drive è eseguito automaticamente all'avvio del sistema.

    • Integrazione Windows Explorer. Se questa funzione è installata, icone in Windows Explorer indicano lo stato delle sincronizzazione di cartelle e file.

    • Scollega. Termina la sincronizzazione e disconnette l'utente. Poi avvia la procedura guidata di installazione di OX Drive. Qui è possibile specificare nuovamente le proprie credenziali, la cartella locale e la cartella remota.

  • La scheda Informazioni. Mostra le informazioni sulla versione di OX Drive e del server di groupware utilizzata.

Errori di sincronizzazione Mostra possibili errori che si sono verificati durante l'installazione.

Uscire da OX Drive Esce da OX Drive.
