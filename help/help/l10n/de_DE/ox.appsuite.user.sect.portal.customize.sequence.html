4.2.1. Reihenfolge der Widgets ändern

Sie können festlegen, in welcher Reihenfolge die Widgets angezeigt werden.

So ändern Sie die Reihenfolge der Widgets:

  1. Ziehen Sie ein Widget an eine andere Position.

  2. Legen Sie das Widget an der neuen Position ab.
