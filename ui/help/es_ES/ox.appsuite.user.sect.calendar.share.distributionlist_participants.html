7.6.4. Creación de una lista de distribución a partir de la lista de participantes

Puede crear una lista distribución a partir de la lista de participantes de una cita.

Cómo crear una lista distribución a partir de la lista de participantes de una cita:

  1. Dependiendo del modo de visualización, utilice uno de los siguientes métodos:

    • En las vistas Día, Semana laboral, Semana o Mes, pulse sobre una cita. En la ventana emergente, pulse Guardar como lista de distribución.

    • En la vista Lista, seleccione una cita de la barra lateral. Pulse sobre Guardar como lista de distribución en el área de visualización.

  2. Complete los detalles para crear la lista de distribución.
