5.5. E-Mail-Ordner hinzufügen

Sie können unterhalb Ihres primären E-Mail-Kontos weitere E-Mail-Ordner anlegen.

So legen Sie einen neuen E-Mail-Ordner an:

  1. Wählen Sie in der Ordneransicht den Ordner, in dem Sie einen neuen Unterordner anlegen wollen.

  2. Klicken Sie neben dem Ordnernamen auf das Symbol Aktionen . Klicken Sie auf Neuen Ordner hinzufügen.

    Ein Fenster öffnet sich.

  3. Geben Sie einen Namen ein. Klicken Sie auf Hinzufügen.

Benutzeroberfläche:

  • Die E-Mail-Ordneransicht

Verwandte Themen:

  • E-Mails verwalten

  • Daten verwalten mit Ordnern
