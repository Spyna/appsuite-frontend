8.5.3. Terminbestätigung ändern

Sie können Ihre Terminbestätigung nachträglich ändern.

So ändern Sie Ihre Terminbestätigung:

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Tag, Arbeitswoche, Woche oder Monat klicken Sie auf einen Termin. Klicken Sie im Popup auf Status ändern.

    • In der Ansicht Liste wählen Sie in der Seitenleiste einen Termin. Klicken Sie im Anzeigebereich auf Status ändern.

  2. Geben Sie im Fenster Change confirmation status eine Anmerkung ein. Klicken Sie auf eine der Schaltflächen Ablehnen, Vorläufig, Akzeptieren.
