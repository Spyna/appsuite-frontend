8.9.8. Aufgaben löschen

Sie können Aufgaben löschen.

Achtung: Wenn Sie eine Aufgabe löschen, ist diese Aufgabe endgültig gelöscht.

So löschen Sie Aufgaben:

  1. Wählen Sie in der Liste eine Aufgabe oder mehrere Aufgaben.

  2. Klicken Sie in der Werkzeugleiste auf Löschen.

  3. Bestätigen Sie, dass Sie die Aufgaben löschen möchten.
