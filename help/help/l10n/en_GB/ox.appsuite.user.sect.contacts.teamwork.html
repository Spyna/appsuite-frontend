Address Book

6.10. Contacts in Teams

You can share your contacts with internal and external partners. Depending on the requirements, different methods exist.

  • The Global address book provides the contact data for all users. Each user can use those contacts.

  • To make additional contacts available for internal users or external partners, proceed as follows:

    • Create a new personal or public address book where you can copy or move the wanted contacts to.

    • Share this address book.

    You can also share an existing address book.

  • If another internal user shared an address book with you, you can access this address book in the folder tree.
