9.6.7. Aggiungere file al portale

È possibile aggiungere un file come widget del portale.

Come aggiungere un file al portale:

  • Selezionare un file nell'area di visualizzazione. Fare clic sull'icona Altre azioni nella barra degli strumenti. Fare clic su Aggiungi al portale nel menu.
