9.5.1. Inviare file come un collegamento

È possibile inviare un collegamento a un file come allegato di posta a un utente. È possibile inviare collegamenti a più file contemporaneamente come allegati di posta.

Come inviare un collegamento a un file:

  1. Fare clic su un file nell'area di visualizzazione. Fare clic su Altro nella finestra a comparsa. Fare clic su Invia come collegamento interno nel menu.

  2. Nella pagina Componi un nuovo messaggio, completare i dettagli per inviare il messaggio di posta.
