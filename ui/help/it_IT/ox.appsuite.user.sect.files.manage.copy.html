9.5.7. Copiare i file

È possibile copiare un file o più file contemporaneamente in un'altra cartella.

Come copiare un file:

Prerequisito: è necessario disporre dei permessi appropriati per creare oggetti nella cartella di destinazione.

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Altro. Nel menu, fare clic su Copia.

    • Nella vista Elenco, fare clic su un file nella barra laterale. Nell'area di visualizzazione, fare clic su Altro. Nel menu, fare clic su Copia.

  2. Selezionare una cartella nella finestra Copia. Fare clic su Copia.
