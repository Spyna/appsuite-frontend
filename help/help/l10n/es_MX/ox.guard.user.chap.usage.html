Capítulo 12. Cifrado de datos

Guard es un componente de seguridad del software colaborativo que permite cifrar correos electrónicos y archivos.

  • Cifre sus comunicaciones de correo electrónico con otros usuarios o colaboradores externos.

  • Cifre archivos individuales. Comparta los datos cifrados con otros usuarios.

  • Usar las opciones de seguridad para definir el nivel de cifrado.

  • Los datos cifrados están protegidos con contraseña. Use la función de restablecimiento de contraseña para protegerse contra las consecuencias de una contraseña perdida.
