Address Book

The Address Book Components

6.1.1. The New contact button

Creates a new contact. The More actions icon opens a menu with additional functions: new distribution list, new email, new appointment, new task

The More actions icon will not be available if the folder view is closed. To show the folder view, click the Open folder view icon on the bottom left side.

General description of the user interface:
