8.6.1. Linux 環境での WebDAV のセットアップ

Linux 環境で WebDAV アクセスをセットアップする

  1. KDE Konqueror または同等のブラウザを起動します。

  2. アドレス バーに次のアドレスを入力します。 webdav://<アドレス>/servlet/webdav.infostore <アドレス> は、グループウェア サーバの IP アドレスか URL に置き換える必要があります。

  3. グループウェア サーバにアクセスするためのユーザ名とパスワードを入力します。

結果: ファイル アプリケーションのファイルがブラウザに表示されます。
