E-Mail aus Adressbuch senden

Sie können aus Ihrem Adressbuch eine E-Mail an einen Kontakt, an mehrere Kontakte oder an eine Verteilerliste senden.

So senden Sie eine E-Mail aus einem Adressbuch:

  1. Wählen Sie in der Seitenleiste einen Kontakt oder eine Verteilerliste.

  2. Klicken Sie im Anzeigebereich auf E-Mail senden.

  3. Vervollständigen Sie die Angaben zum Senden einer neuen E-Mail.
