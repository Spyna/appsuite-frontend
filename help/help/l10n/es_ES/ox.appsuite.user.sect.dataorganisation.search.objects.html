12.1.3. Búsqueda de objetos

Cómo buscar objetos:

  1. Pulse el icono Buscar de la barra de menús. Se abre la ventana de búsqueda.

  2. En la barra de herramientas, seleccione la aplicación que debería usarse para buscar.

  3. Pulse el botón Carpeta. Selecciona la carpeta que desea usar para la búsqueda.

  4. Utilice uno de los siguientes métodos:

    • Para usar un término de búsqueda, introdúzcalo en el campo Buscar. Pulse en una entrada del menú de búsqueda.

    • Para buscar objetos con una propiedad determinada, pulse en el campo Buscar. Pulse en una entrada del menú de búsqueda.

    Resultado: Se mostrará una lista de los objetos encontrados en la ventana de búsqueda.

  5. Puede ajustar aún más el resultado de la búsqueda:

    • Para buscar en otra carpeta, pulse en Carpeta.

    • Para mejorar su búsqueda en la lista de objetos encontrados, inicie una búsqueda adicional.

    • Para reiniciar el resultado, cierre el botón de criterio de búsqueda.
