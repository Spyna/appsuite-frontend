External E-Mail Accounts

If you use external E-Mail accounts e.g., Google Mail, you can access those E-Mail accounts from within the groupware provided you have set up the external E-Mail accounts in the settings. During the setup each external E-Mail account gets an own E-Mail folder.

  • First you have to set up or edit external E-Mail accounts.

  • Then you can use the external E-Mail accounts.
