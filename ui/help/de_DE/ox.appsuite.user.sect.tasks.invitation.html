Aufgabeneinladung beantworten

Wenn ein Benutzer Sie als Teilnehmer zu einer Aufgabe hinzufügt, informiert Sie der Infobereich über diese Aufgabe. Sie können Ihre Teilnahme an der Aufgabe zusagen, vorläufig zusagen oder ablehnen.

So beantworten Sie eine Aufgabeneinladung:

  1. Klicken Sie in der Menüleiste auf das Symbol Neue Objekte. Der Infobereich wird angezeigt.

  2. Wählen Sie unter Einladungen einen Bestätigungsstatus für die neue Aufgabe.

  3. Klicken Sie auf Status ändern.
