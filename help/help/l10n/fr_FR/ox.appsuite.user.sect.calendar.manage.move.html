7.6.6. Déplacer des rendez-vous vers un autre dossier

Vous pouvez déplacer un rendez-vous individuel ou plusieurs rendez-vous à la fois vers un autre dossier.

Pour déplacer un rendez-vous vers un autre dossier :

Pré-requis : vous devez avoir le droit de créer des objets dans le dossier cible.

  1. Selon la vue courante, utilisez l'une des méthodes suivantes :

    • Cliquez sur un rendez-vous dans la vue en calendrier. Dans la fenêtre, cliquez sur l'icône Actions . Dans le menu, choisissez Déplacer.

    • Sélectionnez un rendez-vous dans la vue en liste. Cliquez sur l'icône Actions dans la barre d'outils. Dans le menu, choisissez Déplacer.

  2. Choisissez un dossier dans la fenêtre Déplacer. Cliquez sur Déplacer.
