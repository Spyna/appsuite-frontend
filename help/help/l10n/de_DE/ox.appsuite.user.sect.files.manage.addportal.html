9.5.8. Datei zum Portal hinzufügen

Sie können eine Datei als Widget zum Portal hinzufügen.

So fügen Sie eine Datei zum Portal hinzu:

  • Klicken Sie in der Detailansicht auf eine Datei. Klicken Sie im Popup auf Weitere. Klicken Sie im Menü auf Zum Portal hinzufügen.

    Alternativ wählen Sie eine Datei. Klicken Sie in der Werkzeugleiste auf Weitere. Klicken Sie im Menü auf Zum Portal hinzufügen.
