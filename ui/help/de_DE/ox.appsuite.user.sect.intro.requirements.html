2.2. Systemanforderungen

Um mit der OX App Suite arbeiten zu können, muss Ihr Arbeitsplatzrechner die folgenden Voraussetzungen erfüllen.

Bildschirmauflösung. Die minimale Bildschirmauflösung beträgt 1024 x 768 Pixel.

Browser. 

  • Microsoft Internet Explorer 9, 10

  • Mozilla Firefox, neueste Version

  • Apple Safari Mac OS X 6.0

  • Google Chrome, neueste Version

Browser-Einstellungen. 

  • Cookies müssen aktiviert sein

  • JavaScript muss aktiviert sein

  • Popup-Fenster müssen zugelassen sein
