Aggiungere file al portale

È possibile aggiungere un file come riquadro del portale.

Come aggiungere un file al portale:

  • In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Fare clic su un file nella vista Icone. Fare clic su Altro nella finestra a comparsa. Fare clic su Aggiungi al portale nel menu.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic su Altro nell'area di visualizzazione. Fare clic su Aggiungi al portale nel menu.
