Kacheln aktivieren oder deaktivieren

Sie können die Sichtbarkeit einer Kachel festlegen, indem Sie die Kachel aktivieren oder deaktivieren.

So aktivieren oder deaktivieren Sie eine Kachel:

  1. Klicken Sie rechts unterhalb der Menüleiste auf Diese Seite personalisieren.

  2. Um eine Kachel zu aktivieren, klicken Sie neben der Kachel auf Aktivieren.

    Um eine Kachel zu deaktivieren, klicken Sie neben der Kachel auf Deaktivieren.
