Chapter 1. Qual è lo scopo di OX Drive?

OX Drive consente di sincronizzare i dati tra la propria macchina locale o un dispositivo mobile e il server OX App Suite . È possibile specificare i dati da sincronizzare:

  • su una workstation specificherete due cartelle che saranno sincronizzate:

    • Una cartella remot sul serverOX App Suite . La cartella remota è scelta dalle cartelle personali nell'applicazione File .

    • Una cartella locale sul proprio desktop o dispositivo mobile.

  • Su un dispositivo mobile, potrete scegliere se sincronizzare tutti i contenuti o i preferiti selezionati.

. Ogni modifica applicata sul server è trasferita alla workstation o al dispositivo mobile. Ogni modifica applicata sulla workstation o su un dispositivo mobile sarà trasferita al server e agli atri dispositivi. trasferita alla cartella remota.

Le seguenti modifiche sono sincronizzate.

  • nuovi file

  • modifiche del contenuto dei file

  • rinomina dei file

  • eliminazione dei file

  • nuove cartelle

  • rinomina delle cartelle

  • eliminazione delle cartelle

OX Drive supporta i seguenti sistemi operativi per workstation o dispositivi mobili:

  • MS Windows 7, 8

  • Mac OS X 10.8, 10.9

  • iOS 6, 7

  • Android 4.1, 4.2
