Toont de link naar een bestand

U kan de link tonen naar een bestand welke is opgeslagen in de Bestanden app.

U kan de link naar het bestand als volgt opvragen:

  • Afhankelijk van het gekozen overzicht, kan u één van de volgende methoden kiezen:

    • Klik in het Pictogrammenoverzicht op een bestand. In de pop-up klikt u op Toon link.

    • In het Lijst overzicht, kiest u een bestand uit de zijbalk. Klik op Toon link in het hoofdgebied.
