9.4.8. Datei als PDF speichern

Sie können Dokumente und Textdateien im PDF-Format speichern. Die PDF-Datei wird in dem gleichen Ordner gespeichert, in dem sich das Dokument oder die Textdatei befindet.

So speichern Sie eine Datei im PDF-Format:

  1. Wählen Sie im Anzeigebereich eine Datei. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Als PDF speichern.

  2. Im Fenster Als PDF speichern bestimmen Sie den Namen der PDF-Datei. Klicken Sie auf Speichern.
