9.5.9. Eliminare i file

È possibile eliminare un file o più file contemporaneamente.

Avviso: un file eliminato non può essere più recuperato.

Come eliminare un file:

  1. Fare clic su un file nell'area di visualizzazione. Fare clic su Elimina nella finestra a comparsa.

  2. Confermare che si desidera eliminare il file.

Risultato: il file sarà eliminato.
