11.2. Créer des fichiers de texte

Cous pouvez créer des fichiers de texte obéissant à différents formats de fichiers, p. ex. texte pur, HTML, CSS, JavaScript.

Pour créer un nouveau fichier de texte :

  1. Lancez l'application Fichiers.

  2. Ouvrez un dossier dans l'arborescence des dossiers.

    Remarque : ouvrez un dossier dans lequel vous avez le droit de créer des objets.

  3. Cliquez sur l'icône Nouveau dans la barre d'outils. Cliquez sur Nouveau texte.

  4. Cliquez sur Sans nom en haut à droite de la barre de texte. Saisissez un nom pour le texte.

  5. Saisissez le texte ou collez-le depuis le presse-papiers.

  6. Pour modifier le texte, utilisez les techniques usuelles des applications familières.

  7. Pour finir, cliquez sur l'icône Fermer le document sur la partie droite de la barre de texte.
