6.5.11. Importer des courriels

Vous pouvez importer un message au format EML.

Pour importer un courriel :

  1. Ouvrez le dossier de messages dans lequel vous souhaiter importer le courriel.

  2. Faites glisser le fichier EML depuis le bureau de votre système d'exploitation vers la fenêtre de l'app Courriel. Déposez-le dans la fenêtre de l'app.
