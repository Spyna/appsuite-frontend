10.4. Documenten Bewerken met een Nieuwe Naam

U kan nieuwe documenten, op basis van een kopie van een huidige bestand, maken door een bestaand document te bewerken onder een andere naam.

Zo bewerkt u een document met een nieuwe naam:

  1. Start de Bestanden app.

  2. Open een map met documenten vanuit de mappenboom.

    Opmerking: Selecteer eerst een map waarin u rechten heeft om objecten te maken.

  3. Klik op een document in het hoofdgebied. Klik in de popup op Meer. Selecteer Bewerken als nieuw. Het document wordt geopend voor bewerken.

  4. Klik op Naamloos in de rechterbovenhoek van de documentbalk. Geef de naam van het document.

  5. U kan de volgende functies gebruiken voor het bewerken van een document.
