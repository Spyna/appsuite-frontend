9.8. Cloud Opslag gebruiken

Als u de externe cloud opslag zoals Google Drive, Dropbox, Box of OneDrive gebruikt, kan u toegang tot uw cloud opslag krijgen vanuit de groupware. De volgende opties zijn beschikbaar:

  • Eerst moet u de externe cloud opslag accounts aanmaken en bewerken in de groupware.

  • Dan kan u uw cloud opslag gebruiken met de hulp van de mappenboom.
