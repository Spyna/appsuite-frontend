Customizing the Start Page

The following possibilities exist:

  • Changing the tiles' order

  • Enabling or disabling tiles

Changing the tiles' order

You can define the order of the tiles.

How to change the tiles' order:

  1. Click on Customize this page on the right side below the menu bar.

  2. Drag a tile to another position.
