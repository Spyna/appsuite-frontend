10.2.2. Chats bearbeiten

Gruppenchats und öffentliche Kanäle können von allen Teilnehmern bearbeitet werden. Sie haben folgende Möglichkeiten:

  • Das Bild ändern

  • Den Namen ändern

  • Bei einem Gruppenchat: Teilnehmer hinzufügen oder löschen

So bearbeiten Sie einen Gruppenchat oder einen öffentlichen Kanal:

  1. Wählen Sie in der Chat-Liste einen Gruppenchat oder einen öffentlichen Kanal.

  2. Klicken Sie rechts oben auf das Symbol Weitere Aktionen .

    Wählen Sie Chat bearbeiten beziehungsweise Kanal bearbeiten. Ein Bearbeitungsfenster öffnet sich.

  3. Bei Bedarf können Sie Folgendes ändern:

    • Ändern Sie das Bild, indem Sie das Bild anklicken.

    • Ändern Sie den Namen.

    • Bei einem Gruppenchat: Fügen Sie Teilnehmer hinzu oder löschen Sie Teilnehmer.

  4. Klicken Sie auf Speichern.
