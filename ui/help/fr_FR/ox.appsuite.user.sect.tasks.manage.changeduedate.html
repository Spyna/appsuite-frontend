12.5.3. Changer la date d'échéance d'une tâche

Vous pouvez modifier la date et l'heure d'échéance d'une tâche.

Pour changer la date d'échéance d'une tâche :

Vous devez au préalable avoir le droit de créer des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la barre latérale.

  2. Cliquez sur Modifier la date d'échéance dans la zone d'affichage. Choisissez un élément dans la liste.
