9.9. Impostazioni di Drive

Come utilizzare le impostazioni di Drive:

  1. Fare clic sull'icona Menu di sistema a destra della barra dei menu. Fare clic su Impostazioni nel menu.

  2. Se Drive non è visualizzato nella barra laterale, abilitare Impostazioni avanzate nella parte bassa della barra laterale.

    Fare clic su Drive nella barra laterale.

  3. Cambiare le impostazioni.

Sono disponibili le seguenti impostazioni.

  • Mostra i file e le cartelle nascoste

Mostra i file e le cartelle nascoste Specifica se i file e le cartelle nascoste saranno visualizzate. Questi file e cartelle potrebbero essere creare dai client locali di Drive.
