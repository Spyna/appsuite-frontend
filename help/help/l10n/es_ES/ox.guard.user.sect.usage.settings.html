12.6. Ajustes de seguridad de Guard

Cómo usar los ajustes de Guard:

  1. Pulse el icono Ajustes a la derecha de la barra de menú. Pulse en el elemento de menú Configuración.

  2. Seleccione Seguridad en la barra lateral. Pulse en Seguridad de Guard.

  3. Cambie los ajustes.

Están disponibles los siguientes ajustes.

Predeterminados

  • De manera predeterminada, enviar cifrado al redactar un correo electrónico Determina si un correo nuevo se cifra de manera predeterminada.

  • De manera predeterminada, añadir firma a correos salientes Determina si un correo nuevo se cifra de manera predeterminada.

  • PGP ajustado inicialmente a PGP incrustado en nuevos correos Determina si el cifrado PGP se hace de manera predeterminada incrustado en un correo cifrado. Utilice este ajuste solo si el cliente de correo de algún destinatario no admite PGP, y aún así el mensaje debe ser legible. Si utiliza esta característica, no puede enviar correos en formato html.

  • Recordar contraseña de manera predeterminada Define el rango de tiempo predeterminado durante el que Guard recordará la contraseña. Se puede cambiar este valor predeterminado cuando se le pida su contraseña de Guard.

Avanzado

  • Mostrar los ajustes avanzados Define si se muestra la sección Claves. Esas funciones permiten administrar claves propias o públicas.

Claves

Para enviar o recibir mensajes cifrados, las funciones para administrar claves generalmente no son necesarias. Sin embargo, dichas funciones se pueden usar en los siguientes casos:

  • Desea utilizar sus claves PGP de Guard en otros clientes de correo electrónico, por ejemplo, en clientes de correo electrónico locales.

  • Tiene claves PGP de otras aplicaciones y desea usar esas claves en Guard.

  • Usted tiene la clave pública de un colaborador externo. Para leer mensajes cifrados de este colaborador externo sin tener que acceder a un servidor de claves, desea importar la clave pública del colaborador en Guard.

  • Desea proporcionar su clave pública a un destinatario para darle a este último acceso de lectura a sus mensajes cifrados sin la necesidad de acceder a un servidor de claves.

Existen las siguientes opciones:

  • Botón Sus claves Abre el cuadro de diálogo Sus claves para administrar sus propias claves.

  • Botón Claves públicas Abre el cuadro de diálogo Sus claves para administrar claves públicas.
