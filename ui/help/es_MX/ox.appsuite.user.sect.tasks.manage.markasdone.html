Marcado de tareas como realizadas

Puede marcar una tarea o varias de una vez como realizadas.

Cómo marcar una tarea como realizada:

Requisito: Debe tener permiso para editar objetos en la carpeta que contiene la tarea.

  1. Seleccione una tarea en la barra lateral.

  2. Presione sobre Completada en el área de visualización.
