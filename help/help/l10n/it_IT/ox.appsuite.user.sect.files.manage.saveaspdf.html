9.5.6. Salvare file come PDF

È possibile salvare documenti e file di testo in formato PDF, Il file PDF sarà salvato nella cartella che contiene il documento o file di testo.

Come salvare un file nel formato PDF:

  1. Selezionare un file nell'area di visualizzazione. Fare clic sull'icona Azioni nella barra degli strumenti. Selezionare Salva come PDF nel menu.

  2. Nella finestra Salva come PDF, è possibile digitare un nome per il file PDF. Fare clic su Salva.
