9.7.1. Cloud Speicher einrichten und bearbeiten

Sie haben die folgenden Möglichkeiten:

  • Cloud Speicher einrichten

  • Cloud Speicher bearbeiten

  • Cloud Speicher Account entfernen

So richten Sie Cloud Speicher ein:

  1. Klicken Sie unterhalb des Ordnerbaums unter Account hinzufügen auf ein Symbol. Ein neues Browserfenster öffnet sich.

  2. In dem neuen Browserfenster geben Sie Ihre Zugangsdaten für den Cloud Speicher Account ein.

    Wenn Sie der Cloud Speicher Dienst um die Erlaubnis für den Datenzugriff fragt, erteilen Sie diese Erlaubnis.

    Hinweis: Je nach Anbieter sind die Anfragen nach Zugriffsdaten und Erlaubnis unterschiedlich gestaltet.

Im Ordnerbaum erscheint ein Eintrag für den Cloud Speicher. Sie können den Cloud Speicher jetzt verwenden.

So bearbeiten Sie die Einstellungen eines Cloud Speicher Accounts:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf Accounts.

  3. Klicken Sie im Anzeigebereich neben einem Account auf Bearbeiten. Ein Popup zeigt die Einstellungen.

  4. Ändern Sie die Einstellungen.

  5. Klicken Sie unten im Popup auf Speichern.

So entfernen Sie einen Cloud Speicher Account:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf Accounts.

  3. Klicken Sie neben einem Account auf das Symbol Löschen
