Kapitel 1. Wozu dient OX Drive?

Mit OX Drive synchronisieren Sie Daten zwischen Ihrem lokalen Arbeitsplatzrechner oder Mobilgerät und dem OX App Suite Server. Sie können bestimmen, welche Daten synchronisiert werden.

  • Auf einem Arbeitsplatzrechner bestimmen Sie zwei Ordner, deren Inhalte synchronisierte werden:

    • Einen entfernten Ordner auf dem OX App Suite Server. Als entfernter Ordner dient ein persönlicher Ordner in der App Dateien.

    • Einen lokalen Ordner auf Ihrem Arbeitsplatzrechner oder auf Ihrem Mobilgerät.

  • Auf einem Mobilgerät bestimmen Sie, dass entweder alle Inhalte oder die Favoriten synchronisiert werden, die Sie auf dem Mobilgerät auswählen.

Alle Änderungen, die auf dem Server vornehmen, werden auf den Arbeitsplatzrechner und auf die Mobilgeräte übertragen. Alle Änderungen, die auf dem Arbeitsplatzrechner oder auf einem Mobilgerät vornehmen, werden auf den Server und auf andere Geräte übertragen.

Synchronisiert werden die folgenden Änderungen.

  • Anlegen einer neuen Datei

  • Änderungen am Inhalt einer Datei

  • Umbenennen einer Datei

  • Löschen einer Datei

  • Anlegen eines neuen Ordners

  • Umbenennen eines Ordners

  • Löschen eines Ordners

Als Arbeitsplatzrechner oder als Mobilgerät unterstützt OX Drive die folgenden Systeme:

  • MS Windows 7, 8

  • Mac OS X 10.8, 10.9

  • iOS 6, 7

  • Android 4.1, 4.2
