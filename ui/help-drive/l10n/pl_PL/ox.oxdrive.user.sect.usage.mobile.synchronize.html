2.2.1. Wybieranie danych do synchronizacji

Jak wybrać dane do synchronizacji na urządzeniu mobilnym:

  1. W aplikacji OX Drive stuknij ikonę menu.

  2. Wybierz pozycję Wszystko. Otwórz folder zawierający elementy, które chcesz zsynchronizować.

  3. Przesuń palcem obok nazwy folderu — od prawej strony do środka ekranu. Otworzy się menu folderu.

  4. Stuknij ikonę Ulubione. Zamknij menu folderu.
