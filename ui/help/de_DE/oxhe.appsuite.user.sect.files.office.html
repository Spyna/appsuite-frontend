Office-Dokumente anlegen

Sie können Office-Dokumente anlegen, die formatierten Text, Tabellen und Bilder enthalten können.

So legen Sie ein Office-Dokument an:

  1. Öffnen Sie im Ordnerbaum einen Ordner.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Anlegen von Objekten haben.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Neu . Klicken Sie auf Neues Office Dokument.

  3. Geben Sie den Text ein, oder fügen Sie Text aus der Zwischenablage ein.

  4. Geben Sie einen Titel ein. Klicken Sie auf Speichern.
