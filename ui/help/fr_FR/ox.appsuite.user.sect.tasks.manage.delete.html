11.5.7. Supprimer des tâches

Vous pouvez supprimer une ou plusieurs tâches.

Pour supprimer une tâche  :

Attention : lorsque l'on supprime une tâche, celle-ci est perdue à jamais.

  1. Choisissez une tâche dans la barre latérale.

  2. Cliquez sur Supprimer dans la zone d'affichage.

  3. Confirmez que vous voulez supprimer la tâche.

Résultat : la tâche est supprimée.
