9.5.8. Adding files to the portal

You can add a file to the portal as a portal square.

How to add a file to the portal:

  • Depending on the view selected, use one of the following methods:

    • Click on a file in the Icons view. Click on More in the pop-up window. Click on Add to portal in the menu.

    • In the List view, select a file in the side bar. Click on More in the display area. Click on Add to portal in the menu.
