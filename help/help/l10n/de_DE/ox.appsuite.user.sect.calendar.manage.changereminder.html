7.8.2. Terminerinnerung ändern

Sie können Ihre Terminerinnerung nachträglich ändern, indem Sie eine Erinnerung anpassen, entfernen oder eine weitere Erinnerung hinzufügen.

So ändern Sie eine Terminerinnerung:

  1. Je nach Ansicht verwenden Sie eine der folgenden Methoden:

    • In einer Kalenderansicht klicken Sie auf einen Termin. Ein Popup öffnet sich.

    • In der Listenansicht wählen Sie einen Termin. Doppelklicken Sie auf den Termin. Der Termin wird in einem Fenster angezeigt.

    Klicken Sie auf Erinnerungen ändern. Das Fenster Erinnerungen ändern öffnet sich.

  2. Klicken Sie auf eine Erinnerung. Das Fenster Erinnerungen bearbeiten öffnet sich.

    Je nach Bedarf können Sie eine vorhandene Erinnerung bearbeiten oder löschen oder eine neue Erinnerung hinzufügen.
