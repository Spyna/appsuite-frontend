5.5.12. Imprimer des courriels

Pour imprimer des messages, vous pouvez :

  • imprimer le contenu d'un courriel

Comment imprimer le contenu d'un courriel :

  1. Sélectionnez un courriel. Cliquez sur l'icône Plus dans la barre d'outils. Dans le menu, choisissez Imprimer. Une fenêtre s'ouvre et présente un aperçu avant impression.

    Vous pouvez aussi utiliser l'icône Actions dans la vue détaillée.

  2. Si nécessaire, changez les réglages d'impression. Cliquez sur le bouton Imprimer.

  3. Fermez la fenêtre d'aperçu d'impression.

Conseil : vous pouvez également imprimer plusieurs courriels à la fois.
