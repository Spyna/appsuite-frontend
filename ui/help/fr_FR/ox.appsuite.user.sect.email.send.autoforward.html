6.4.4. Faire suivre automatiquement des courriels

Vous pouvez faire suivre automatiquement les courriers électroniques à une autre adresse.

Pour faire suivre vos messages automatiquement :

  1. Cliquez sur l'icône Menu système à droite de la barre de menu. Choisissez Réglages dans le menu.

  2. Cliquez sur Transfert automatique dans la barre latérale.

  3. Saisissez l'adresse de courrier électronique où vous souhaitez faire suivre les messages.

  4. Cliquez sur Activé.
