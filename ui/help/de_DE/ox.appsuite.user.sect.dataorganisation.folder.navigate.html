12.2.2. Navigieren in der Ordnerstruktur

Die folgenden Techniken sind verfügbar:

  • den Ordnerbaum öffnen und schließen

  • die Breite des Ordnerbaums ändern

  • im Ordnerbaum einen Ordner öffnen oder auswählen

So öffnen Sie den Ordnerbaum:

  • Verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Werkzeugleiste auf das Symbol Ordner an/aus

    • Klicken Sie in der Seitenleiste unterhalb der Liste auf den Ordnernamen.

So ändern Sie die Breite des Ordnerbaums:

  1. Bewegen Sie den Zeiger an den rechten Rand des Ordnerbaums. Ein Doppelpfeil wird angezeigt.

  2. Ziehen Sie den Rand nach links oder nach rechts.

So schließen Sie den Ordnerbaum:

  • Verwenden Sie eine der folgenden Methoden:

    • Klicken Sie unterhalb des Ordnerbaums auf das Symbol Schließen .

    • Klicken Sie in der Werkzeugleiste auf das Symbol Ordner an/aus .

So öffnen oder wählen Sie einen bestimmten Ordner:

  1. Wenn der Ordnerbaum geschlossen ist, verwenden Sie eine der folgenden Methoden, um ihn zu öffnen:

    • Klicken Sie in der Werkzeugleiste auf das Symbol Ordner an/aus

    • Klicken Sie in der Seitenleiste unterhalb der Liste auf den Ordnernamen.

  2. Wenn ein Ordner weitere Unterordner enthält, können Sie die Unterordner anzeigen, indem Sie vor dem Ordnernamen auf das Dreieck klicken.

  3. Klicken Sie auf den Ordner, den Sie öffnen oder wählen wollen.

Ergebnis: Die Objekte des Ordners werden angezeigt.
