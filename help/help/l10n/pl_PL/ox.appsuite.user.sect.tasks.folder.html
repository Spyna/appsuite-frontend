8.6. Dodawanie folderów zadań

Dodatkowe osobiste foldery zadań można utworzyć w sekcji poniżej elementu Moje zadania.

Jak utworzyć nowy osobisty folder zadań:

  1. W widoku folderów kliknij pozycję Dodaj nowy folder. Otworzy się okno.

  2. Wprowadź nazwę. Jeśli nowy folder ma być publiczny, zaznacz opcję Dodaj jako folder publiczny. Kliknij polecenie Dodaj.

Interfejs użytkownika:

  • Widok folderów w aplikacji Zadania

  • Zarządzanie zadaniami

  • Zarządzanie danymi z folderami
