9.6.3. Freigaben verwalten

Um eigene Freigaben zu verwalten, haben Sie die folgenden Möglichkeiten:

  • bei vorhandenen Freigaben die Art der Berechtigung wie Leserecht oder Bearbeitungsrecht ändern

  • Andere Benutzer oder externe Partner zu vorhandenen Freigaben hinzufügen

  • Andere Benutzer oder externe Partner von einer Freigabe entfernen

  • Vorhandene Freigaben zurückziehen

So verwalten Sie Ihre Freigaben:

  1. In der App Drive öffnen Sie im Ordnerbaum den Eintrag Shared files.

    Öffnen Sie den Eintrag Shared by me. In der Detailansicht werden die Ordner und Dateien angezeigt, die Sie freigegeben haben.

  2. Verwenden Sie diese Methoden:

    • Um die Art der Berechtigung zu ändern, klicken Sie neben einem freigegebenen Ordner oder einer freigegebenen Datei auf Optionen. Wählen Sie einen Eintrag.

    • Um eine Freigabe zurückzuziehen, klicken Sie klicken Sie neben einem freigegebenen Ordner oder einer freigegebenen Datei auf das Symbol Remove .
