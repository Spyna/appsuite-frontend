Edición de tareas

Puede editar los datos de una tarea en otro momento.

Cómo editar una tarea:

Requisito: Debe tener permiso para crear objetos en la carpeta que contiene la tarea.

  1. Seleccione una tarea en la barra lateral.

  2. Pulse en Editar en el área de visualización. Se mostrarán los datos de la tarea.

    Para ver el formulario completo pulse en Expandir formulario.

  3. Edite los datos.

  4. Pulse en Guardar.
