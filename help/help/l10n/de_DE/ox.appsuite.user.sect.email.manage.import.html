5.7.11. E-Mails importieren

Sie können eine E-Mail importieren, die als eml-Datei vorliegt.

So importieren Sie eine E-Mail:

  1. Öffnen Sie den E-Mail-Ordner, in den Sie die E-Mail importieren wollen.

  2. Ziehen Sie die eml-Datei aus einem Dateibrowser oder vom Desktop in die E-Mail-Liste.
