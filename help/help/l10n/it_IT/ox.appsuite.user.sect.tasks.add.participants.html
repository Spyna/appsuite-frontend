8.4.3. Aggiungere partecipanti a un'attività

Come aggiungere partecipanti nella finestra di modifica delle attività:

  1. Digitare gli indirizzi di posta elettronica dei partecipanti, il nome di un gruppo o il nome di una lista di distribuzione nel campo di inserimento sotto Partecipanti. Suggerimenti:

    • Mentre si digita un indirizzo di posta, sono visualizzati diversi suggerimenti corrispondenti. Per accettare un suggerimento, utilizzare uno dei seguenti metodi:

      • Utilizzare la barra di scorrimento per scorrere l'elenco. Fare clic su un suggerimento.

      • Utilizzare le frecce direzionali per selezionare un suggerimento. Premere Invio.

    • Per selezionare contatti da una rubrica, fare clic sull'icona Rubricaa destra del campo di inserimento.

  2. Per rimuovere un partecipante, fare clic sull'icona accanto al nome.
