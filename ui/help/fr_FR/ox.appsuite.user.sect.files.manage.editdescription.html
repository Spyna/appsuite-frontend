9.5.5. Modifier des descriptions

Vous pouvez modifier la description d'un fichier.

Pour modifier la description d'un fichier :

Vous devez au préalable avoir les droits permettant de modifier des objets dans le dossier contenant le fichier.

  1. Selon la vue choisie, utilisez une des méthodes suivantes :

    • Dans la vue Icônes, cliquez sur un fichier. Dans la fenêtre qui s'affiche, cliquez sur Plus.... Dans le menu, cliquez sur Modifier la description.

    • Dans la vue Liste, cliquez sur un fichier dans la barre latérale. Dans la zone d'affichage, cliquez sur Plus.... Dans le menu, cliquez sur Modifier la description.

  2. Modifiez la description.

  3. Cliquez sur Enregistrer.
