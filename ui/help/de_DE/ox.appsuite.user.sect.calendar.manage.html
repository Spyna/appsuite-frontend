8.5. Termine verwalten

Einige der Techniken zum Verwalten von Terminen setzen voraus, dass Sie eigene Kalenderordner eingerichtet haben. Informationen zum Anlegen von Ordnern finden Sie in Ordner.

Sie haben folgende Möglichkeiten:

  • Termine bearbeiten

  • Termine mit Drag&Drop bearbeiten

  • Terminbestätigung ändern

  • Termine in einen anderen Ordner verschieben

  • Termine löschen

  • Mehrere Termine gemeinsam bearbeiten
