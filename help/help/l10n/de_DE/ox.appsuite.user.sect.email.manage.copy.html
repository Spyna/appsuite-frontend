5.6.3. E-Mails kopieren

Sie können einzelne E-Mails oder eine vollständige Konversation in einen anderen Ordner kopieren.

So kopieren Sie E-Mails:

Voraussetzung: Sie haben im Zielordner die Berechtigung zum Anlegen von Objekten.

  1. Wählen Sie eine E-Mail oder mehrere E-Mails.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen . Klicken Sie im Menü auf Kopieren.

    Alternativ verwenden Sie eine der folgenden Methoden:

    • Verwenden Sie in der Detailansicht das Symbol Weitere Aktionen .

    • Verwenden Sie in der E-Mail-Liste das Kontextmenü.

  3. Wählen Sie im Fenster Kopieren einen Ordner. Bei Bedarf legen Sie einen neuen Ordner an, indem Sie auf Ordner anlegen klicken.

  4. Klicken Sie auf Kopieren.
