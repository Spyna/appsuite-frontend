12.3.5. Datei entschlüsseln

Sie können die Verschlüsselung von einer Datei entfernen, indem Sie die Datei entschlüsseln.

So entschlüsseln Sie eine Datei:

  1. In der App Drive wählen Sie im Anzeigebereich eine verschlüsselte Datei.

  2. Verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Verschlüsselung entfernen.

    • Wählen Sie im Kontextmenü Verschlüsselung entfernen.

  3. Wenn sich das Fenster Guard-Sicherheitspasswort eingeben öffnet, geben Sie das Guard-Sicherheitspasswort ein.

    Bei Bedarf können Sie festlegen, wie lange das Guard-Sicherheitspasswort gelten soll. Dazu aktivieren Sie Mein Passwort merken. Wählen Sie in der Liste einen Wert.

    In den Guard-Sicherheitseinstellungen können Sie für den Zeitraum einen Vorgabewert festlegen.

    Klicken Sie auf OK.
