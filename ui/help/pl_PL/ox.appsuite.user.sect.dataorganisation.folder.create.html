12.3.3. Tworzenie folderów

W folderze osobistym można utworzyć dowolną liczbę dalszych podfolderów. Aby utworzyć podfoldery w folderze udostępnionym lub publicznym, musisz mieć odpowiednie uprawnienia.

Jak utworzyć nowy folder:

  1. W drzewie folderów wybierz folder, w którym chcesz utworzyć nowy podfolder. Jeśli nie wybierzesz folderu, nowy podfolder zostanie utworzony w głównym folderze osobistym.

    Uwaga: wybierz folder, w przypadku którego masz uprawnienia do tworzenia podfolderów.

  2. Kliknij ikonę Dodaj folder poniżej drzewa folderów. Kliknij polecenie Dodaj podfolder.

  3. W oknie Dodaj nowy podfolder wpisz nazwę i kliknij przycisk Dodaj folder.
