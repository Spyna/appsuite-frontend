9.4.1. Envoyer les fichiers comme lien

Vous pouvez envoyer à d'autres utilisateurs des liens vers des fichiers. Le lien est inséré dans le texte du courrier électronique. Si l'utilisateur clique sur le lien, le fichier est alors affiché dans une nouvelle page.

Pour envoyer des liens vers des fichiers :

  1. Sélectionner un ou plusieurs fichiers dans la zone d'affichage. Cliquez sur l'icône Actions dans la barre d'outils. Cliquez sur Envoyer comme lien interne dans le menu.

  2. Dans la page Rédiger un nouveau courriel, remplissez les détails nécessaires pour envoyer le courriel.
