8.6.1. Ajout de dossiers de tâches personnelles

Vous pouvez créer des dossiers de tâches personnelles sous Mes tâches.

Pour créer un nouveau dossier de tâches personnelles :

  1. Dans l'arborescence des dossiers, cliquez sur Ajouter un nouveau dossier. Une fenêtre s'ouvre.

  2. Saisissez un nom. Si le nouveau dossier doit être public, activez l'option Ajouter comme dossier public. Cliquez sur Ajouter.
