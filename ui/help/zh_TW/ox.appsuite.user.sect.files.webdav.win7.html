9.6.4. 在 Windows 7 下存取 WebDAV

透過下列方式在 Windows 7 設定 WebDAV 存取：

  • 使用 OX Updater。可在 OX 更新器的文件中找到相關信息。

  • 首先安裝微軟的網頁資料夾 軟體更新。然後設定 WebDav 存取。

如何在 Windows 7 下設定 WebDAV 存取：

  1. 在 Windows 資源管理器中打開 計算機。

  2. 在圖示列中單擊 連線磁碟機。連線磁碟 窗口打開。

  3. 單擊 連線到讓您可以儲存文件的網站。添加網絡位置 對話窗口打開。單擊 下一步。

  4. 選擇 選擇自定義網絡位置。單擊 下一步 。

  5. 在 網際網路或內部網絡位置 欄位中輸入以下地址： https://<address>/servlet/webdav.infostore 其中 <address> 必須要用組件伺服器的 IP 或網址代替。單擊 下一步。

  6. 輸入您的用戶名和密碼。使用您組件伺服器的登入資料。單擊 確定。

  7. 在下一頁中您可以向網絡資源指定一個名稱。單擊 下一步。

  8. 單擊 完成。

結果： 您可以在 電腦 下存取您的 檔案 應用程式檔案。
