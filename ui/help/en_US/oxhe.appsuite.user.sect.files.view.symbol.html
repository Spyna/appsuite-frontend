Displaying files in the icon view

How to display a file in the icon view:

  1. Click the View icon in the toolbar. Click on Icons.

  2. Select a folder from the navigation path on top of the display area or open a folder in the folder tree.

  3. In order to quickly find a particular file, use the search function.

  4. Click a file's icon in the display area. The data are displayed in the pop-up. For some file types a preview is shown.
