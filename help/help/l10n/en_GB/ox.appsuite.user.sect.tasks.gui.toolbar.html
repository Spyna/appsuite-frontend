8.1.3. The Tasks tool bar

Contains the following:

  • New. Creates a new task.

  • Edit. Edits a task's data.

  • Due. Changes a task's due date.

  • Done. Marks a task as done.

  • Delete. Deletes the tasks selected by you.

  • Actions icon . Opens a menu with additional functions:

  • View. Opens a menu with check boxes for controlling the view.

    • Folder view. Opens or closes the folder tree.

    • Checkboxes. Displays checkboxes in the list, for marking objects.
