The Drive Components

9.1.1. The New button

Opens a menu with further functions:

  • create new appointments

  • upload files or folders

  • create text note

  • Depending on the configuration, functions for creating text documents, spreadsheets or presentations will be available.

  • new email, new appointment, new contact, new task

Those function will not be available if the folder view is closed. To show the folder view, click the Open folder view icon on the bottom left side.

General description of the user interface:

  • The button for creating new objects
