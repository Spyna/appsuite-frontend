Come avviare l'applicazione File:

  • Fare clic sull'icona Esecutore delle applicazioni nella barra dei menu. Fare clic su File nell'esecutore delle applicazioni.

    In base alla configurazione, è possibile inoltre fare clic sull'icona di avvio rapido per eseguire l'applicazione File nella barra dei menu.
