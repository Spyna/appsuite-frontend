5.6.11. Saving E-Mails

You can save an E-Mail as a text file. The text file has the file extension EML. If you select multiple E-Mails, an eml file will be created for each E-Mail. The eml file will be saved as zip archive then.

How to save an E-Mail:

  1. Select one or several E-Mails.

  2. Click the More actions icon in the toolbar. Select Save as file from the menu.

    You can also use the More actions iconin the detail view.

  3. Fill in the details for saving the file.
