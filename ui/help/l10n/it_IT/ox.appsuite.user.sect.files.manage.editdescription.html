9.5.5. Modificare le descrizioni

È possibile modificare la descrizione di un file.

Come modificare la descrizione di un file:

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella contenente il file.

  1. Fare clic su un file nell'area di visualizzazione. Fare clic su Altro nella finestra a comparsa. Fare clic su Modifica descrizione nel menu.

  2. Modificare la descrizione.

  3. Fare clic su Salva.
