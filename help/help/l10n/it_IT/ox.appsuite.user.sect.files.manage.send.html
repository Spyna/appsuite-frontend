9.5.1. Inviare file come allegati di posta

È possibile inviare le versioni correnti dei file come allegati di posta.

Come inviare file come allegati di posta elettronica:

  1. Selezionare uno o più file nell'area di visualizzazione. Fare clic sull'icona Azioni nella barra degli strumenti. Fare clic su Invia tramite posta nel menu.

    È possibile utilizzare anche l'icona Azioni nel Visualizzatore.

  2. Nella pagina Componi, completare i dettagli per inviare il messaggio di posta.
