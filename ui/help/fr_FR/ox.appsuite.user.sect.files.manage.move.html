Déplacer des fichiers

Vous pouvez déplacer un fichier ou plusieurs fichiers à la fois dans un autre dossier.

Pour déplacer un fichier :

Vous devez au préalable avoir les droits permettant de créer des objets dans le dossier cible.

  1. Selon la vue choisie, utilisez une des méthodes suivantes :

    • Dans la vue Icônes, cliquez sur un fichier. Dans la fenêtre qui s'ouvre, cliquez sur Plus.... Dans le menu, cliquez sur Déplacer.

    • Dans la vue Liste, cliquez sur un fichier dans la barre latérale. Dans la zone d'affichage, cliquez sur Plus.... Dans le menu, cliquez sur Déplacer.

  2. Choisissez un dossier dans la fenêtre Déplacer. Cliquez sur Déplacer.

Conseil : pour déplacer des fichiers par glisser-déposer, activez la vue en Liste view. Sélectionnez un ou plusieurs fichiers dans la barre latérale. Faites glisser les fichiers sélectionnés vers un dossier de l'arborescence des dossiers.
