6.5.5. Anhänge hinzufügen

So fügen Sie im Kontakte-Bearbeitungsfenster Anhänge hinzu:

  1. Klicken Sie unterhalb von Anhänge auf Anhänge hinzufügen.

  2. Wählen Sie eine Datei oder mehrere Dateien.

    Um einen Anhang zu entfernen, klicken Sie auf das Symbol Löschen .

Tipp: Sie können einen Anhang auch hinzufügen, indem Sie ein Dokument aus einem Dateibrowser oder vom Desktop in das Kontakt-Fenster ziehen und ablegen.
