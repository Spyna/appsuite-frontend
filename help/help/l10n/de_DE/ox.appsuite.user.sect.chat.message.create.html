10.3.1. Nachrichten lesen, senden, beantworten

Sie haben folgende Möglichkeiten:

  • Nachrichten eines Chats suchen und lesen. Das Eintreffen von Nachrichten können Sie folgendermaßen erkennen:

    • Wenn ungelesene Nachrichten vorhanden sind, enthält das Chat-Symbol in der Menüleiste einen roten Punkt.

    • In den Chat-Einstellungen können Sie folgende Benachrichtigungen festlegen:

      • Beim Eintreffen neuer Nachrichten wird ein Ton abgespielt.

      • Wenn sich die Groupware im Hintergrund befindet, wird eine Desktopbenachrichtigung angezeigt.

      • Wenn Sie offline sind, erhalten Sie eine E-Mail-Benachrichtigung, die den Inhalt der Chat-Nachricht enthält. Wenn Sie diese E-Mail beantworten, wird Ihre Antwort in den Chat eingefügt.

  • Nachrichten und Dateien in einer Gruppe oder an einen einzelnen Kontakt senden

  • Eine frühere Nachricht beantworten. Dabei wird die frühere Nachricht in Ihrer Antwort zitiert.

So lesen Sie eine Nachricht:

  1. Wählen Sie in der Chat-Liste einen Chat.

    Chats mit ungelesenen Nachrichten werden folgendermaßen dargestellt:

    • Der Namen des Chats wird in Fettschrift angezeigt.

    • Eine Zahl nennt die Anzahl ungelesener Nachrichten.

  2. Lesen Sie die Nachrichten der Konversation. Um ältere Nachrichten dieser Konversation zu sehen, blättern Sie mit Hilfe der Symbole am rechten Rand nach oben.

Optionen:

  • Um nach Chats zu suchen, geben Sie in das Eingabefeld Suchen oder einen neuen Chat starten einen Suchbegriff ein.

So senden Sie eine Nachricht:

  1. Verwenden Sie eine der folgenden Alternativen:

    • Um einen neuen Chat zu beginnen, klicken Sie auf das Symbol Neu .

      Wählen Sie im Menü die gewünschte Form der Konversation.

    • Um einen vorhandenen Chat weiterzuführen, wählen Sie in der Chat-Liste einen Chat. Die bisher ausgetauschten Nachrichten dieses Chats werden angezeigt.

  2. Tippen Sie eine Nachricht in das Nachrichtenfeld.

    Wenn Sie während der Eingabe das Chat-Fenster schließen oder einen anderen Chat öffnen, bleiben Ihre Eingaben erhalten, solange Sie in der Groupware angemeldet sind.

  3. Um die Nachricht zu senden, drücken Sie [Eingabe] oder klicken Sie auf das Symbol Senden .

    Bei einem privaten Chat zeigt ein Symbol rechts neben der Nachricht den Lesestatus Ihrer Nachricht:

    • Ein grauer Haken: Die Nachricht ist auf dem Server angekommen.

    • Zwei graue Haken: Die Nachricht wurde vom Server an den Empfänger übertragen.

    • Zwei blaue Haken: Die Nachricht wurde vom Empfänger gelesen.

Optionen:

  • Um innerhalb einer Nachricht einen Zeilenumbruch zu erzeugen, drücken Sie [Umschalt]+[Eingabe].

  • Wenn Sie die Nachricht in einer Gruppe senden, können Sie in der Nachricht einen Kontakt gezielt ansprechen. Dazu tippen Sie [@], dahinter den Namen der Person, dahinter die Nachricht. Der angesprochene Kontakt sieht in der Chat-Liste eine rote Markierung am Namen der Gruppe.

  • Um Emojis zu senden, klicken Sie neben dem Eingabefeld auf das Symbol Emoji hinzufügen . Ein Fenster mit Emojis öffnet sich. Wählen Sie ein Emoji oder mehrere Emojis.

    Um das Fenster zu schließen, klicken Sie erneut auf das Symbol Emoji hinzufügen .

  • Um Dateien zu senden, klicken Sie neben dem Eingabefeld auf das Symbol Datei hochladen .

    Alternativ ziehen Sie eine Datei oder mehrere Dateien aus einem Dateibrowser oder vom Desktop in das Chat-Feld.

    Hinweis: Aus Sicherheitsgründen können ausführbare Dateien nicht gesendet werden.

  • Um alle bisher gesendeten oder empfangenen Dateien zu sehen, klicken Sie in der Chat-Liste auf Alle Dateien.

  • Um den Text einer Nachricht zu formatieren, verwenden Sie die folgenden Zeichen.

    • Dieses Wort ist _kursiv_ geschrieben.

    • Dieses Wort ist *fett* geschrieben.

    • Dieses Wort ist ~durchgestrichen~.

    • Das ist `Programmcode` innerhalb einer Zeile.

    • Das ist ```Programmcode in einem Block

      Codezeile 1

      Codezeile 2

      ```

      Hinweis: Für die Zeilenumbrüche zwischen den Zeilen des Programmcodes verwenden Sie [Umschalt]+[Eingabe].

So beantworten Sie eine Nachricht:

  1. Zeigen Sie im Chat-Feld auf eine Nachricht.

    Klicken Sie neben der Nachricht auf das Symbol Nachrichtenaktionen . Wählen Sie Antworten.

    Die Nachricht und ihr Absender werden als Zitat in das Nachrichtenfeld kopiert.

  2. Tippen Sie Ihre Antwort in das Nachrichtenfeld.

    Um die Antwort zu senden, drücken Sie [Eingabe] oder klicken Sie auf das Symbol Senden .
