Copier des fichiers

Vous pouvez copier un fichier ou plusieurs fichiers à la fois dans un autre dossier.

Pour copier un fichier :

Vous devez au préalable avoir les droits permettant de créer des objets dans le dossier cible.

  1. La méthode à utiliser dépend de la vue courante :

    • Dans la vue Icônes, cliquez sur un fichier. Dans la fenêtre qui s'ouvre, cliquez sur Plus.... Dans le menu, cliquez sur Copier.

    • Dans la vue Liste, cliquez sur un fichier dans la barre latérale. Dans la zone d'affichage, cliquez sur Plus.... Dans le menu, cliquez sur Copier.

  2. Choisissez un fichier dans la fenêtre Copier. Cliquez sur Copier.
