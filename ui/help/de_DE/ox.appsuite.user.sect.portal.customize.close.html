4.2.2. Portal-Kachel entfernen

Sie können eine Kachel vom Portal entfernen, indem Sie die Kachel schließen.

So entfernen Sie eine Portal-Kachel:

  • Klicken Sie in der Kachel auf das Symbol Schließen .

    Um die Portal-Kachel wieder anzuzeigen, verwenden Sie die Portal-Einstellungen.
