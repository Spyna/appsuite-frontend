9.5.3. création ou modification des descriptions

Vous pouvez créer ou modifier la description d'un fichier.

Pour créer ou modifier la description d'un fichier :

Pré-requis : vous devez avoir les droits permettant de modifier les objets dans le dossier contenant le fichier.

  1. Sélectionnez un fichier dans la zone d'affichage. Si aucun détail n'est affiché, cliquez sur Affichage dans la barre d'outils. Activez Détails du fichier.

  2. Afin d'ajouter une description, cliquez sur Ajouter une description. Afin de modifier une description, double-cliquez sur la description.

    Saisissez un nouveau nom ou modifier le nom existant. Cliquez sur Enregistrer.

Vous pouvez également utiliser la fonction dans la barre latérale de l'Afficheur.
