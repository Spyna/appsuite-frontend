12.2.3. Creating folders

You can create any number of subfolders in a personal folder. To create subfolders in a shared or public folder, you need to have the required permissions.

How to create a new folder:

  1. In the folder tree, select the folder in which you want to create a new subfolder. If you do not select a folder, the new subfolder will be created in your personal root folder.

    Note: Select a folder for which you have the appropriate permissions to create subfolders.

  2. Click the Add folder icon below the folder tree. Click on Add subfolder.

  3. Enter a name in the Add new subfolder window. Click on Add folder.
