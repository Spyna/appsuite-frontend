Eliminar tareas

Puede eliminar una tarea o varias tareas de una vez.

Cómo borrar una tarea:

Advertencia: Cuando elimine una tarea, se pierde definitivamente.

  1. Seleccione una tarea en la barra lateral.

  2. Pulse Borrar en el área de visualización.

  3. Confirme que desea borrar la tarea.

Resultado: La tarea es eliminada.
