10.1.3. Los componentes de la página Editar de Hoja de cálculo

Barra de herramientas La barra de herramientas se encuentra a la izquierda. Contiene los siguientes iconos:

  • Icono Cambiar búsqueda . Muestra controles para buscar o reemplazar contenido de celdas.

  • Icono Descargar . Descarga el documento seleccionado.

  • Icono Imprimir . Muestra el documento actual en un cuadro de diálogo de impresión. Utilice los controles de la parte superior del cuadro de diálogo para imprimir el documento o guardarlo como un fichero PDF.

  • Icono Enviar como correo electrónico . Envía el documento seleccionado como adjunto en un correo electrónico.

Barra de documento La barra de documento está ubicada en la derecha. Contiene los siguientes componentes:

  • Herramientas básicas

    • ocultar la barra de documento

    • deshacer la última acción

    • rehacer la última acción deshecha

    • cerrar el documento

  • herramientas de ficheros

    • El nombre de fichero del documento. Para renombrar el documento, pulse en el nombre del fichero

    • funciones de ficheros: guardar con un nombre diferente, descargar, descargar como pdf, activar Autoguardado

  • fuente herramientas de formateo

    • selección de tipo de letra y tamaño

    • iconos de formato de fuente

  • Formateo de celdaHerramientas de formateo de celda

    • Seleccionar estilo

    • Alineación horizontal y vertical de celdas. Fusionar y separar celdas. Ajuste de línea automático.

    • Color de relleno, borde, copiar formato, borrar formato

  • Formato numéricoHerramientas para asignar formatos numéricos

    • Formato numérico. Define el formato del contenido de la celda, por ejemplo numérico.

    • Códigos de formato. Define el subformato, por ejemplo el número de dígitos decimales.

  • Formato numéricoHerramientas para filas y columnas

    • Insertar fila, eliminar filas seleccionadas, establecer la altura de fila.

    • Insertar columna, eliminar columna seleccionada, establecer el ancho de columna.

  • InsertarHerramientas de inserción

    • Calcular la suma de las celdas seleccionadas, insertar hiperenlace, insertar imagen, insertar gráfica

  • Herramientas para el control de la vista

    • Dividir la hoja, congelar fila o columna
