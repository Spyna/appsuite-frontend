9.5.3. Link zu einer Datei anzeigen

Sie können den Link zu einer Datei anzeigen, die in der App Dateien gespeichert ist.

So zeigen Sie den Link zu einer Datei an:

  • Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Symbole klicken Sie auf eine Datei. Klicken Sie im Popup auf Link anzeigen.

    • In der Ansicht Liste wählen Sie in der Seitenleiste eine Datei. Klicken Sie im Anzeigebereich auf Link anzeigen.
