2.2. Utiliser le client OX Drive sur un appareil mobile

Les fonctionnalités suivantes sont disponibles sur un appareil mobile :

  • Sélectionnez les données à synchroniser

  • Consultez le contenu du dossier photostream

  • Utilisez d'autres fonctionnalités, comme la création, la recherche dans et le tri de dossiers

  • Personnalisez les réglages
