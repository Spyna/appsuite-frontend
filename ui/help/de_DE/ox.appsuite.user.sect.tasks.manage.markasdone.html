Aufgaben als erledigt kennzeichnen

Sie können eine Aufgabe oder mehrere Aufgaben gemeinsam als erledigt kennzeichnen.

So kennzeichnen Sie eine Aufgabe als erledigt:

Voraussetzung: Sie haben für den Ordner, der die Aufgabe enthält, die Berechtigung zum Bearbeiten von Objekten.

  1. Wählen Sie in der Seitenleiste eine Aufgabe.

  2. Klicken Sie im Anzeigebereich auf Erledigt.
