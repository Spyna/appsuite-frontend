6.6.1. E-Mails teilen

Sie können Ihre E-Mails mit internen Benutzern teilen. Je nach Anforderung können verschiedene Methoden sinnvoll sein.

  • Wenn Sie E-Mails für interne Benutzer zur Verfügung stellen möchten, gehen Sie folgendermaßen vor:

    • Legen Sie einen neuen persönlichen Ordner an, in den Sie die gewünschten E-Mails kopieren oder verschieben.

    • Geben Sie diesen Ordner für alle Benutzer oder für bestimmte Benutzer frei.

    • Die Benutzer müssen Ihren freigegebenen E-Mail-Ordner abonnieren, um auf die E-Mails zugreifen zu können.

  • Wenn ein anderer Benutzer einen E-Mail-Ordner für Sie freigegeben hat, müssen Sie den freigegebenen E-Mail-Ordner abonnieren, um auf die E-Mails zugreifen zu können.
