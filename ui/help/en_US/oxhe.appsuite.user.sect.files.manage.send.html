Sending files as E-Mail attachment

You can send the current version as E-Mail attachment. You can send the current versions of multiple files together as E-Mail attachments.

How to send the current version as E-Mail attachment:

  1. Depending on the view selected, use one of the following methods:

    • In the Icons view, click on a file. In the pop-up, click on Send by E-Mail.

    • In the List view, select a file in the sidebar. Click on Send by E-Mail in the display area.

  2. Complete all data for sending the E-Mail in the Compose new E-Mail page.
