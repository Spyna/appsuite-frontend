4.1. Die Bestandteile von Apps

Eine Einführung in die Benutzeroberfläche finden Sie in Die Benutzeroberfläche.

Die Seite Ihre Anwendungen enthält die folgenden Bestandteile:

  • Schaltfläche Anwendungen verwalten. Startet den Application Manager, der Sie beim Organisieren ihrer Apps unterstützt.

  • Eine Übersicht der verfügbaren Apps. Um eine App zu starten, klicken Sie auf das entsprechende Symbol.
