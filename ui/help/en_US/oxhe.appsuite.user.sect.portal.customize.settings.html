Changing the portal tiles' settings

How to use the portal tiles settings:

  1. Click the System menu icon in the menu bar. In the menu click on Settings.

  2. In the sidebar click on Portal.

  3. Change the Settings.

Tip: You can also display the portal tiles settings page by clicking on Customize this page in the Portal app.

The following settings are available.

  • Color button

  • Options button

Color button. Defines the color used for displaying a portal tile's name.

Options button. Opens a menu with the following functions: Edit, Disable, Delete.
