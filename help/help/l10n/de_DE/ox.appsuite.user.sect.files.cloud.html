9.8. Cloud Speicher verwenden

Wenn Sie Cloud Speicher anderer Anbieter verwenden, zum Beispiel Google Drive, Dropbox, Box oder OneDrive, können Sie innerhalb der Groupware auf Ihren Cloud Speicher zugreifen. Sie haben die folgenden Möglichkeiten.

  • Zunächst müssen Sie den Cloud Speicher eines anderen Anbieters innerhalb der Groupware einrichten und bearbeiten.

  • Danach können Sie Ihren Cloud Speicher mit Hilfe des Ordnerbaums verwenden.
