12.2.4. Renommer des dossiers

Vous pouvez renommer les sous-dossiers de vos dossiers privés. Pour les autres dossiers, vous avez besoin des droits adéquats.

Pour renommer un dossier :

  1. Dans l'arborescence des dossiers, choisissez le dossier que vous voulez renommer.

  2. Cliquez sur l'icône Actions sur le dossier près du nom du dossier. Choisissez Renommer.

  3. Modifiez le nom ou saisissez un nouveau nom. Cliquez sur Renommer.
