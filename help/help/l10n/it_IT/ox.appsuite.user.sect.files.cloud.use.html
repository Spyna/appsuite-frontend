Account di archiviazione sul cloud

9.8.2. Utilizzare le archiviazioni sul cloud

Sono disponibili le seguenti opzioni:

  • È possibile utilizzare l'albero delle cartelle per accedere ai contenuti dell'archiviazione sulla nuvola.

    Nota: in base alla funzionalità del servizio di archiviazione sulla nuvola, il controllo di versione potrebbe non essere supportato.

  • In base al contenuto, è possibile utilizzare le funzioni della barra dei menu, ad es. per visualizzare immagini o copiare dati.
