10.3.2. Managing messages

The following options exist:

  • edit own messages

  • delete own messages

How to edit a message sent by you:

  1. In the chat field, go to a message sent by you.

  2. Click the Message actions icon next to the message. Select Edit message.

  3. Edit the message.

    To finish and send the message, press [Enter] or click the Send icon .

    To cancel the editing, click the Cancel icon .

Tip: To edit the previously sent message, press the up arrow key.

How to delete a message sent by you:

  1. In the chat field, go to a message sent by you.

  2. Click the Message actions icon next to the message. Select Delete message.
