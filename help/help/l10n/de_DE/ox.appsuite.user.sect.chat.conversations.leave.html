10.2.5. Eine Konversation verlassen

Sie können eine Gruppenkonversationen oder einen öffentlichen Kanal jederzeit verlassen.

So verlassen Sie eine Gruppenkonversation oder einen öffentlichen Kanal:

  1. Wählen Sie in der Liste eine Gruppenkonversation oder einen öffentlichen Kanal.

  2. Klicken Sie rechts oben auf das Symbol More actions .

    Wählen Sie Leave chat.
