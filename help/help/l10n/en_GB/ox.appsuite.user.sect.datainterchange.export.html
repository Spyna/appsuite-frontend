Interchanging Data

12.2. Exporting Data

With the Export function you can export certain data to a file for use with other programs. You can export the following data:

  • Contacts and distribution lists in the following formats:

    • CSV

    • vCard

  • Appointments in the following formats:

    • iCalendar

  • Tasks in the following formats:

    • iCalendar

Those formats are standardised and can be imported by many other programs. The following options are available:

  • exporting a folder's data.
