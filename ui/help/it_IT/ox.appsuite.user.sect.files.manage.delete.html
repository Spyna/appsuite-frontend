Eliminare i file

È possibile eliminare un file o più file contemporaneamente.

Avviso: un file eliminato non può essere più recuperato.

Come eliminare un file:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Elimina.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic su Elimina nell'area di visualizzazione.

  2. Confermare che si desidera eliminare il file.

Risultato: il file sarà eliminato.
