Ordnerarten

Es gibt folgende Ordnerarten:

  • Persönliche Ordner

    • Die persönlichen Ordner enthalten Ihre E-Mails, Kontakte, Termine und Dokumente. Andere Benutzer können Ihre persönlichen Ordner nicht sehen, solange Sie diese nicht freigeben.

    • Sie finden Ihre persönlichen Ordner im Ordnerbaum unterhalb von Privat.

    • Ihr persönlicher Dokumente-Ordner befindet sich unterhalb von InfoStore/Userstore.

  • Öffentliche Ordner

    • Die öffentlichen Ordner enthalten Kontakte, Termine und Dokumente, die für alle Benutzer von Interesse sind. Jeder Benutzer kann öffentliche Ordner anlegen und diese für andere Benutzer freigeben.

    • Sie befinden sich im Ordnerbaum unterhalb von Public. Hier finden Sie das Globale Adressbuch mit den Kontaktdaten aller Benutzer.

    • Der öffentliche Dokumente-Ordner befindet sich in Öffentlicher InfoStore.

  • Freigegebene Ordner

    • Freigegebene Ordner sind Ordner, die andere Benutzer für Sie zum Lesen oder Schreiben freigegeben haben.

    • Sie befinden sich im Ordnerbaum unterhalb von Shared.

    • Freigegebene Dokumente-Ordner befinden sich unterhalb von Userstore.
