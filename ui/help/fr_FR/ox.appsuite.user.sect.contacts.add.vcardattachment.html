6.4.2. Ajouter un contact depuis une vCard

Vous pouvez ajouter un contact depuis une vCard jointe à un courrier électronique. Par défaut, une telle pièce jointe a un nom qui se termine par vcf.

Pour ajouter un contact depuis une vCard jointe :

  1. Dans l'application Courriel, choisissez un message avec une vCard en pièce jointe.

  2. Cliquez sur le nom de la pièce jointe dans la zone d'affichage. Choisissez Ajouter au carnet d'adresses dans le menu.
