7.5.7. Modificare più appuntamenti contemporaneamente

È possibile eseguire le seguenti funzioni per più appuntamenti contemporaneamente:

  • Spostare gli appuntamenti in un'altra cartella

  • Stampare più appuntamenti

Come eseguire una funzione per più appuntamenti contemporaneamente:

  1. Nella vista Elenco, fare clic sull'icona Seleziona sotto la barra laterale. Una casella di selezione sarà visualizzata accanto a ogni appuntamento nella barra laterale.

  2. Abilitare le caselle di selezione per almeno due appuntamenti.

    È possibile utilizzare anche le funzioni di selezione multipla del proprio sistema.

    Per selezionare tutti gli appuntamenti nella cartella attualmente selezionata, marcare la casella di selezione Seleziona tutto in basso a sinistra della barra laterale.

  3. Fare clic su una funzione nell'area di visualizzazione.
