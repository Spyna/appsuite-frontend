7.2.1. Termine in einer Kalenderansicht anzeigen

So zeigen Sie Termine in einer Kalenderansicht an:

  1. Klicken Sie in der Werkzeugleiste auf Ansicht. Wählen Sie einen der folgenden Einträge: Tag, Arbeitswoche, Woche, Monat.

  2. Öffnen Sie im Ordnerbaum einen Kalenderordner.

    Wenn Sie alle Ihre Termine aus allen Kalenderordnern sehen möchten, öffnen Sie den Ordner Alle meine Termine.

  3. Klicken Sie im Kalenderblatt auf einen Termin. Die Daten des Termins werden im Popup angezeigt.

  4. Um im Kalender zu blättern, verwenden Sie die Navigationselemente oberhalb des Kalenderblatts.

    Um den Zeitraum anzuzeigen, der den aktuellen Tag enhält, klicken Sie in der Werkzeugleiste auf Heute.
