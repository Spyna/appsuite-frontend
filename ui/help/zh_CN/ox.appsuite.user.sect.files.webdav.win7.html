9.6.4. 在 Windows ７ 下设置 WebDAV

为了在 Windows 7 下设置 WebDAV 访问，使用以下选项之一：

  • 使用 OX 更新器。可以在 OX 更新器用户文档里找到信息。

  • 首先，为网路文件夹安装微软软件更新器。然后，设置WebDav 访问。

如何在 Windows 7 下设置 WebDAV 访问：

  1. 在Windows 资源管理器的导航区域里，选择计算机项目。

  2. 在图标条里单击连接硬盘。连接硬盘窗口打开。

  3. 单击连接一个您可以使用储存文档的网页。添加网路位置窗口打开。单击下一步。

  4. 选择选择自定义网络位置。单击下一步。

  5. 在互联网或网路地址对话窗口中输入以下地址：webdav://<地址>/servlet/webdav.infostore 其中<地址>必须要用组件服务器的 IP 地址或 URL 代替。单击下一步。

  6. 输入您的用户名和密码。使用您的针对组件服务器的登录数据。单击 确定。

  7. 在下一页上，您可以为网路位置分配一个名称。单击下一步。

  8. 单击芬兰文.

结果：计算机下面，您可以访问您的 文件 应用程序文件。
