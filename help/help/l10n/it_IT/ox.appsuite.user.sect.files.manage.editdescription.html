9.4.5. Creare o modificare descrizioni

È possibile creare o modificare la descrizione di un file.

Come creare o modificare la descrizione di un file:

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella contenente il file.

  1. Selezionare un file nell'area di visualizzazione. Se non viene visualizzato alcun dettagliato, fare clic su Vista nella barra degli strumenti. Abilitare Dettagli file.

  2. Per aggiungere una descrizione, fare clic su Aggiungi una descrizione. Per modificare una descrizione, fare doppio clic sulla descrizione.

    Digitare un nuovo testo o cambiare quello esistente. Fare clic su Salva.

È possibile inoltre utilizzare la funzione nella barra laterale del Visualizzatore.
