10.5. Chat-Einstellungen

So verwenden Sie die Chat-Einstellungen:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Einstellungen. Klicken Sie auf Einstellungen.

  2. Klicken Sie in der Liste auf Chat.

Je nach Konfiguration sind die folgenden Einstellungen verfügbar:

Anzeigeoptionen

  • Beim Start letzten Chat auswählen. Bestimmt, ob nach der Anmeldung an der Groupware der zuletzt verwendete Chat im Chat-Fenster angezeigt wird.

  • Chats nach Typ gruppieren. Bestimmt, ob in der Chat-Liste die Chats in private Chats, Gruppenchats und Kanäle gruppiert werden.

  • Chats sortieren nach. Bestimmt, wie die Chats in der Chat-Liste sortiert werden.

  • Kompaktheitsgrad. Bestimmt, wie detailliert die Chats in der Chat-Liste dargestellt werden.

Benachrichtigungen

  • Für neue Nachrichten Desktopbenachrichtigungen anzeigen. Bestimmt, ob Sie eine Desktopbenachrichtigung erhalten, wenn Sie eine neue Nachricht empfangen, während sich die Groupware im Hintergrund befindet. Damit diese Einstellung wirksam ist, müssen Sie Desktop-Benachrichtigungen auch in den Grundeinstellungen aktivieren, siehe Mit Benachrichtigungen arbeiten

  • Für neue Nachrichten Benachrichtigungston abspielen. Bestimmt, ob ein Benachrichtigungston abgespielt wird, wenn Sie eine neue Nachricht empfangen. Wenn Benachrichtigungstöne aktiviert sind, sind weitere Einstellungen verfügbar:

    • Der Ton, der abgespielt werden soll

    • Das Verhalten, unter welchen Bedingungen der Ton abgespielt wird

  • Offline-E-Mail-Benachrichtigungen. Bestimmt, ob und bei welchen Chats Sie eine E-Mail-Benachrichtigung erhalten, wenn Sie abgemeldet sind.

Verwandte Themen:

  • Das Chat-Fenster
