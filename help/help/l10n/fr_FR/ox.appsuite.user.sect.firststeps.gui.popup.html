2.4.7. Les fenêtres

Remarque : le contenu de la nouvelle fenêtre varie en fonction de l'application. Les éléments ci-dessous constituent une représentation schématique.

Affiche les détails relatifs à un objet. Les actions suivantes ouvrent une nouvelle fenêtre :

  • un clic sur l'expéditeur ou le destinataire d'un courriel affiché en vue détaillée

  • un clic sur un participant d'un rendez-vous ou d'une tâche

  • un clic sur un rendez-vous dans la vue en agenda

Si vous lisez un courriel dans la zone d'affichage et que vous cliquez sur le nom de l'expéditeur, les éléments suivants s'affichent dans la nouvelle fenêtre :

  • des boutons pour effectuercertaines actions, par exemple copier les données de contact de l'expéditeur dans un dossier

  • les données de contact de l'expéditeur

  • des informations connexes, par exemple vos rendez-vous partagés ou vos échanges de courriels avec l'expéditeur

  • les données de cette personne en provenance de réseaux sociaux. Vous pouvez utiliser les boutons disponibles pour afficher le profil de cette personne.

Ces données associées à cette personne sont désignées par le terme vue en halo. Cliquer sur certains éléments de la fenêtre peut ouvrir une fenêtre supplémentaire. Pour fermer cette fenêtre, cliquez sur l'icône Fermer en haut à droite.
