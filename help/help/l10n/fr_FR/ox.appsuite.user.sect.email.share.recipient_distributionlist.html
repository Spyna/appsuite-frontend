Courriels au sein d'une équipe

5.7.4. Enregistrer tous les destinataires d'un message en tant que liste de diffusion.

Si un message s'adresse à plusieurs destinataires, vous pouvez les enregistrer tous dans une nouvelle liste de diffusion.

Pour enregistrer les destinataires d'un courrier électronique en tant que liste de diffusion :

  1. Sélectionnez un courriel.

  2. Cliquez sur l'icône Actions dans la vue détaillée.

    Choisissez Enregistrer comme liste de diffusion dans le menu.

  3. Complétez les données pour créer une liste de diffusion.
