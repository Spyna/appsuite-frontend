9.5. Gestire i file

Alcune delle tecniche per gestire i file richiedono la preventiva configurazione delle proprie cartelle. Informazioni sulla creazione delle cartelle sono disponibili in Cartelle.

Esistono le seguenti opzioni:

  • Inviare file come un collegamento

  • Inviare file come allegati di posta

  • Mostrare il collegamento a un file

  • Modificare i nomi dei file

  • Modificare le descrizioni

  • Spostare i file

  • Copiare i file

  • Aggiungere file al portale

  • Eliminare i file

  • Modificare diversi file contemporaneamente

  • Lavorare con le versioni
