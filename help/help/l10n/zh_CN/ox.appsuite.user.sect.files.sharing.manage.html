9.6.4. 管理自己的分享

为了组织自己的分享，您可以做以下的事：

  • 编辑现有权限，例如：更改某位用户的读取权限成写入权限。

  • 批准权限到一个额外用户，以添加新权限

  • 为某个用户移除现有权限

  • 为所有用户移除现有权限

如何编辑现有权限：

  1. 单击在文件夹树里的我的分享。

  2. 在详情视图里选择一个分享。在工具条里单击编辑分享。会打开一个窗口。它显示每一个用户批准了的权限。

  3. 使用以下方法：

    • 要调整一个用户角色，在当前角色里单击项目。从菜单选择一个项目。

    • 要调整权限详情，在详细访问权里单击项目。从菜单里选择一个权限。

    备注：您不能更改一个个人文件夹的管理权。

    备注：权限不能为公共链接更改。

  4. 单击 保存。

如何添加新权限：

  1. 单击在文件夹树里的我的分享。

  2. 在详情视图里选择一个分享。在工具条里单击编辑分享。会打开一个窗口。它显示每一个用户批准了的权限。

  3. 在添加人员里输入一个电子邮件地址。电子邮件地址将会添加到列表成内部用户、群组、或访客。已批准预设权限。

  4. 单击 保存。

如何为单一用户移除权限：

  1. 单击在文件夹树里的我的分享。

  2. 在详情视图里选择一个分享。在工具条里单击编辑分享。会打开一个窗口。它显示每一个用户批准了的权限。

  3. 为了移除一个用户的权限，单击用户名称旁边的动作图标。在菜单里单击收回访问权。

  4. 单击 保存。

如何移除所有权限：

  1. 单击在文件夹树里的我的分享。

  2. 在详情视图里选择一个分享。在菜单条里单击收回访问权。
