10.2.1. Creating chats

The following options exist:

  • create a private chat with a single contact

  • create a group chat with specific participants

  • create a public channel which other contacts can join

How to create a chat:

  1. Click the New icon above the chat list or on New chat.

    A menu opens.

  2. Use one of the following methods:

    • To create a chat with a singe contact, select Private chat. The Start new conversation window opens.

      Select a contact.

      Click on Start conversation.

    • To create a chat with multiple participants, select Group chat. The Create group chat window opens.

      Enter a group name. You can add a picture by clicking on the icon.

      Add contacts.

      Click on Create chat.

    • To create a public channel that other contacts can join, select Channel. The Create new channel window opens.

      Enter a channel name. You can add a picture by clicking on the icon.

      Click on Create channel.

The new chat will be displayed in the list.
