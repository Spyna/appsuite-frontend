Organiser les données

11.2. Droits

Les droits définissent les actions qu'un utilisateur interne ou un partenaire externe est autorisé à faire sur un dossier particulier et son contenu.

  • Vous pouvez trouver une description des droitsdans 11.2.1: Quels droits peuvent être accordés ?.

  • Vous pouvez trouver une liste des droits prédéfinis pour des dossiers particuliers dans 11.2.2: Droits sur les dossiers existants et 11.2.3: Droits sur les nouveaux dossiers.

Vous pouvez définir les permissions en partageant des courriels, carnets d'adresses, agendas, tâches, dossiers ou fichiers.
