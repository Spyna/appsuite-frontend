5.9.4. Saving all email recipients as a distribution list

If an email contains multiple recipients, you can save all the recipients as a new distribution list.

How to save the recipients of an email as a distribution list:

  1. Select an E-Mail.

  2. Click the More actions icon in the detail view.

    Select Save as distribution list from the menu.

  3. Complete the details for creating the distribution list.
