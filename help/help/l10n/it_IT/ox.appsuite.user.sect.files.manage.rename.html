9.4.4. Modificare i nomi dei file

È possibile modificare il nome di un file.

Come modificare il nome di un file:

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella contenente il file.

  1. Selezionare un file nell'area di visualizzazione. Fare clic sull'icona Azioni nella barra degli strumenti. Fare clic su Rinomina nel menu.

  2. Modificare il nome di un file nella finestra Rinomina. Notare l'estensione del nome. Fare clic su Rinomina.

È possibile utilizzare anche l'icona Azioni o fare clic sul nome del file in alto a destra del visualizzatore.
