5.4.7. Signatur hinzufügen

Wenn Sie Signaturen erstellt haben, können Sie dem E-Mail Text eine Signatur hinzufügen.

So fügen Sie auf der Seite Verfassen dem E-Mail Text eine Signatur hinzu:

  1. Klicken Sie auf Signaturen.

  2. Wählen Sie in der Liste eine Signatur.
