2.3. Operating Instructions

The following techniques are required:

  • Using the left mouse button for clicking on elements

  • Using drag and drop

  • Multi-selection by pressing the <ctrl> key or the <cmd> key on Mac systems

  • Working with several windows

  • Opening, saving, and closing files

  • Opening, closing, and selecting files in a folder tree

Information on these techniques can be found in your operating system documentation.
