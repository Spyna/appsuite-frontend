E-Mail an Teilnehmer senden

Sie können eine E-Mail an alle Teilnehmer eines Termins senden.

So senden Sie eine E-Mail an alle Teilnehmer eines Termins:

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Tag, Arbeitswoche, Woche oder Monat klicken Sie auf einen Termin. Klicken Sie im Popup auf Send mail to all participants.

    • In der Ansicht Liste wählen Sie in der Seitenleiste einen Termin. Klicken Sie im Anzeigebereich auf E-Mail an alle Teilnehmer senden.

  2. Vervollständigen Sie die Angaben zum Senden der E-Mail.
