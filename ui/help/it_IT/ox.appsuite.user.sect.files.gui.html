8.1. I componenti di File

Un'introduzione all'interfaccia utente è disponibile in L'interfaccia utente.

Barra degli strumenti Contiene le seguenti icone:

  • L'icona Aggiungi . Contiene le seguenti funzioni:

    • Carica un nuovo file. Crea un nuovo file.

    • Nuovo documento di testo. Crea un nuovo documento di testo.

  • L'icona Cartella . Apre o chiude l'albero delle cartelle.

  • L'icona Ricerca . Apre un campo di inserimento per cercare i file con un determinato nome.

  • L'icona Vista . Consente di selezionare una delle viste seguenti per l'area di visualizzazione:

    • Icone

    • Elenco

Albero delle cartelle Visualizza le cartelle. Consente la navigazione all'interno della struttura delle cartelle. Per aprire l'albero delle cartelle, fare clic sull'icona Cartella nella barra degli strumenti. Sono disponibili le seguenti funzioni:

  • Facendo clic su una cartella, i file contenuti sono visualizzati.

  • Le icone Aggiungi il menu delle cartelle e Azioni delle cartelle sotto le cartelle offrono funzioni per organizzare i dati.

  • L'icona Aggancia definisce se l'albero delle cartelle deve sovrapporsi alla barra laterale o all'area di visualizzazione o deve essere visualizzato accanto alla barra laterale o all'area di visualizzazione.

Area di visualizzazione nella vista Icone Visualizza le seguenti informazioni:

  • Percorso di navigazione alla cartella selezionata. Per aprire un'altra cartella, fare clic sul percorso.

  • Il pulsante Carica un nuovo file. Apre la finestra per caricare un nuovo file.

  • Il pulsante Condividi cartella. Crea una pubblicazione per la cartella selezionata.

  • Se la cartella contiene immagini, file audio o video, saranno visualizzate funzioni per riprodurre file multimediali.

  • Un'icona per ogni file nella cartella selezionata e se disponibile, un'anteprima del file.

Facendo clic su un file, i suoi dati sono visualizzati nella finestra a comparsa. Tale finestra mostra le stesse informazioni dell'area di visualizzazione nella vista Elenco.

La barra laterale nella vista Elenco Visualizza un elenco di file nella cartella selezionata. Le seguenti funzioni sono disponibili:

  • Facendo clic su un file, i suoi dati sono visualizzati nell'area di visualizzazione.

  • Facendo clic sull'icona Seleziona sotto l'elenco, una casella di selezione sarà visualizzata accanto a ogni file. È possibile selezionare più file per modificarli contemporaneamente.

  • Il testo sotto l'elenco contiene il nome della cartella selezionata. Facendo clic su di esso, si aprirà l'albero delle cartelle. Il numero corrisponde al numero di oggetti nella cartella selezionata.

Area di visualizzazione nella vista Elenco Visualizza i dati del file:

  • I pulsanti: Apri , Scarica , Elimina .

    Facendo clic su un file modificabile nella barra laterale, il pulsante Modifica viene visualizzato.

    Facendo clic su Altro, si apre un menu che mostra le seguenti funzioni: Invia come collegamento , Invia tramite posta , Mostra collegamento , Rinomina , Modifica descrizione , Lock , Sposta , Copia e Aggiungi al portale .

    Nota: alcuni pulsanti non sono visualizzati se non si dispone dei permessi appropriati.

    Facendo clic su un documento nella barra laterale, il pulsante Modifica come nuovo viene visualizzato nel menu sotto Altro.

  • Nome, nome file

  • Un'anteprima dei file, se disponibile

  • Commenti dei file

  • Il percorso di navigazione alla cartella selezionata. Per aprire un'altra cartella, fare clic sul percorso.

  • I pulsanti per caricare una nuova versione.

  • Un elenco di versioni se il file ha più versioni. La versione corrente è elencata in alto. Per ogni versione, sono visualizzate le seguenti informazioni:

    • il nome file della versione

    • commenti della versione, se disponibili

    • nome dell'utente che ha caricato la versione

    • data e ora del caricamento della versione
