9.2.2. E-Mail Anhänge anzeigen

Die Anhänge von E-Mails, die Sie gesendet oder empfangen haben, werden in einem eigenen Ordner angezeigt.

So zeigen Sie die Anhänge von E-Mails an:

  1. Öffnen Sie den Ordner Meine Anhänge.

  2. Im den Inhalt eines Anhangs anzuzeigen, verwenden Sie die Methoden zum Anzeigen von Dateien.

    Um die E-Mail anzuzeigen, die zu der gewählten Anlage gehört, klicken Sie in der Werkzeugleiste auf Ansicht. Aktivieren Sie Datei-Details. Klicken Sie auf Nachricht anzeigen.
