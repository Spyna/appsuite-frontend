9.2.2. Afficher les pièces jointes des courriels

Les pièces jointes des courriels que vous avez reçus ou envoyés sont présentées dans un dossier séparé.

Pour afficher les pièces jointes aux courriels :

  1. Ouvrez le dossier Mes pièces jointes.

  2. Pour afficher le contenu d'une pièce jointe, utilisez les méthodes d'affichage des fichiers.

    Pour afficher le courriel correspondant à la pièce jointe sélectionnée, cliquez sur Vue dans la barre d'outils. Activez Détails du fichier. Cliquez sur Afficher le message.
