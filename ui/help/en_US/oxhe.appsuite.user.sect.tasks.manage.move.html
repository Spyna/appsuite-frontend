Moving tasks

You can move a task or multiple tasks at once to another folder.

How to move a task:

Prerequisite: You have the permission to create objects in the target folder.

  1. Select a task in the sidebar

  2. Click on Move in the display area .

  3. Select a folder in the Move window. Click on OK.
