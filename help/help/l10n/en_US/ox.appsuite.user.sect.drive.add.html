9.4. Drive: Creating Folders

You can create new folders in the current folder.

How to create a new folder:

  1. Open a folder.

    Note: Open a folder for which you have the appropriate permissions to create objects.

  2. Click on New in the toolbar. Click on Folder.

    You can also click the Actions icon next to the folder name, in the folder view. Click on Add new folder.

  3. Enter a name in the Add new folder window. Click on Add.
