8.5.1. Edición de tareas

Puede editar los datos de una tarea en otro momento.

Cómo editar una tarea:

Requisito: Debe tener permiso para crear objetos en la carpeta que contiene la tarea.

  1. Seleccione una tarea en la barra lateral.

  2. Presione en Editar en el área de visualización. Se mostrarán los datos de la tarea.

    Para ver el formulario completo presione en Expandir formulario.

  3. Edite los datos.

  4. Presione en Guardar.
