5.6.2. Creating a new rule when moving

You can create a new rule when moving an E-Mail to another folder. Incoming E-Mails for which this rule applies, will then be moved to this folder automatically.

How to create a new rule when moving an E-Mail

  1. Select an E-Mail.

  2. Click the Actions icon in the toolbar. Select Move from the menu.

    You can also use the Actions icon in the detail view.

    Select a folder in the Move window. Click on the Move button.

  3. Enable Create filter rule. A note with details about the filter is displayed.

    Click on the Move button. The Create new rule window opens.

  4. Enter a rule name. You can adjust the conditions or actions. Click on Save.

    The E-Mail will be moved. The new rule will be created.
