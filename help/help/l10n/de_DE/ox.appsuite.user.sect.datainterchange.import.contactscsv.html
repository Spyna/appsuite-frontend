11.1.3. Kontakte im CSV-Format importieren

Die folgenden CSV-Dateien werden unterstützt:

  • Standard CSV-Dateien, deren Werte durch Kommas voneinander getrennt sind

  • CSV-Dateien aus den folgenden Microsoft Outlook Versionen:

    • Microsoft Outlook 2003, 2007

    • Sprachversionen Deutsch, Englisch, Französisch

Das korrekte Format der CSV-Datei wird automatisch erkannt. Die Zuordnung der Daten zu bestimmten Datenfeldern ist abhängig von der Ihrer Konfiguration. Mehr erfahren Sie von Ihrem Administrator oder Hoster.

So importieren Sie Kontakte aus einer CSV-Datei:

  1. Starten Sie die App Adressbuch.

  2. Wählen Sie im Ordnerbaum das Adressbuch, in das Sie die Kontakte importieren wollen.

  3. Klicken Sie neben dem Adressbuch auf das Symbol Aktionen . Klicken Sie auf Import.

  4. Wählen Sie das passende Datenformat. Klicken Sie auf Import.

Ergebnis: Die Kontakte werden dem Adressbuch hinzugefügt.
