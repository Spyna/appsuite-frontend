5.8. E-Mail-Filter verwenden

E-Mail-Filter helfen Ihnen, eingehende E-Mails zu organisieren. Ein E-Mail-Filter enthält eine oder mehrere Regeln. Mit Hilfe von Regeln können Sie zum Beispiel folgende Aktionen auslösen:

  • Die E-Mail wird in einem bestimmten E-Mail-Ordner abgelegt.

  • Die E-Mail wird an eine andere E-Mail-Adresse weitergeleitet.

  • Die E-Mail wird als gelesen markiert.

Um E-Mail-Filter zu verwenden, müssen Sie folgende Schritte ausführen:

  • Legen Sie E-Mail-Ordner an.

  • Erstellen Sie eine oder mehrere Regeln.

  • Bestimmen Sie die Reihenfolge der Regeln.

  • Legen Sie fest, ob nachfolgende Regeln bearbeitet werden, wenn eine Regel zutrifft.

Sie haben folgende Möglichkeiten:

  • Neue Regel erstellen

  • Eine Bedingung hinzufügen

  • Eine Aktion hinzufügen

  • Neue Regel erstellen beim Verschieben

  • Neue Regel erstellen für den Betreff und den Absender

  • Eine Regel ändern

  • Eine Regel auf vorhandene E-Mails anwenden

Hinweis: Die folgenden Anleitungen und Beispiele gehen davon aus, dass der vollständige Funktionsumfang der E-Mail-Filter verfügbar ist. Je nach Konfiguration sind einige Funktionen möglicherweise nicht verfügbar.

Verwandte Themen:

  • E-Mail automatisch weiterleiten lassen

  • Abwesenheitsbenachrichtigung automatisch senden
