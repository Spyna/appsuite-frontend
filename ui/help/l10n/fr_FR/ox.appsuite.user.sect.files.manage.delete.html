9.5.9. Supprimer des fichiers

Vous pouvez supprimer un seul fichier ou plusieurs fichiers à la fois.

Attention : Un fichier supprimé ne peut pas être récupéré.

Pour supprimer un fichier :

  1. Cliquez sur un fichier dans la zone d'affichage. Dans la fenêtre qui s'ouvre, cliquez sur Supprimer.

  2. Confirmez que vous voulez supprimer le fichier.

Résultat : le fichier sera supprimé.
