7.5. Gestión de citas

Algunos de los métodos para organizar las citas requieren tener totalmente configuradas las carpetas de su calendario. Puede encontrar información sobre la creación de carpetas en Carpetas.

Existen las siguientes posibilidades:

  • Edición de citas

  • Edición de citas con arrastrar y soltar

  • Cambiar el estado de una cita

  • Traslado de citas a otra carpeta

  • Impresión de citas

  • Borrado de citas

  • Edición de varias citas de una vez
