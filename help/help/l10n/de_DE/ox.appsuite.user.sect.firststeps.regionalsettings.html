3.4. Regionale Einstellungen ändern

Mit Hilfe der regionalen Einstellungen bestimmen Sie die Sprache, die Formatierung von Daten und Grundeinstellungen des Kalenders:

  • Zeitformat

  • Datumsformat

  • Zahlenformate

  • Erster Wochentag einer Kalenderwoche

  • Erste Kalenderwoche eines Jahres

Sie haben folgende Möglichkeiten:

  • Eine regionale Einstellung wählen

  • Die regionale Einstellung individuell anpassen

So ändern Sie die regionale Einstellung:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Einstellungen. Klicken Sie im Menü auf Meine Kontaktdaten.

  2. Klicken Sie in der Seitenleiste auf Grundeinstellungen.

  3. Wählen Sie unterhalb von Region im Listenfeld eine vorgegebene Region.

    Um die regionalen Einstellungen individuell anzupassen, klicken Sie auf Customize regional settings.

    Um die individuellen Anpassungen zurück zu nehmen, wählen Sie im Listenfeld eine vorgegebene Region.
