9.6.2. Windows XP 下设置 WebDAV

如何在 Windows XP 下设置 WebDAV 访问：

  1. 单击我的网路位置。在我的计算机窗口单击我的网路位置，或在视窗桌面里双击我的网路位置图标。

  2. 前往网路任务，打开对话窗口我的网路位置后单击添加一个网路位置。一个用于添加网络资源的向导启动。单击下一个。

  3. 在下一个窗口里选择选择别的网路位置。

  4. 输入以下地址： https://<adresse>/servlet/webdav.infostore，以组件服务器的 IP 地址或 URL取代 <address>。

  5. 为组件服务器输入您的用户名称和密码。

  6. 在下一个对话窗口中，您可以向网络资源指定一个名称。

    备注：如果您已经将您的编码设置为 ISO，请不要在文件夹或文档名称中使用任何特殊字符。组件服务器使用 UTF-8 编码。

  7. 在下一个对话窗口中单击完成 。

结果：您可以使用网上邻居访问您的 文件 app files.
