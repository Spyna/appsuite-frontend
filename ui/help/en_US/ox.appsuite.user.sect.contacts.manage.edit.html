Editing contacts

Contact data can be edited at a later point of time. The editing window shows the data that are most frequently used. Other data can be displayed.

How to edit a contact:

Prerequisite: You have the permission to edit objects in the folder containing the contact.

  1. Select a contact in the sidebar.

  2. Click on Edit in the display area. The contact's data are displayed.

  3. Edit the data.

  4. Click on Save.
