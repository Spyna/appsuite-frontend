9.6.3. Creación o edición de descripciones

Puede crear o editar la descripción de un fichero.

Requisito: Tener los permisos adecuados para editar objetos en la carpeta que contiene el fichero.

Cómo crear o editar la descripción de un fichero:

  1. Seleccione un fichero en el área de visualización.

    Si no se muestran detalles, pulse Vista en la barra de herramientas. Marque Detalles del fichero.

  2. Para añadir una descripción, pulse Añadir una descripción.

    Para editar una descripción, pulse dos veces en la descripción.

    Introduzca un texto nuevo o cambie el existente. Pulse Guardar.

También puede usar la función de la barra lateral del Visor.
