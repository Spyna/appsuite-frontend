7.4.7. Aggiungere allegati

Come aggiungere allegati nella finestra di modifica degli appuntamenti:

  1. Fare clic su Aggiungi allegati sotto Allegati.

  2. Selezionare uno o più file.

    Per rimuovere un allegato, fare clic sull'icona Elimina.

Suggerimento: è inoltre possibile aggiungere un allegato trascinando e rilasciando un documento da un navigatore di file o dal desktop alla finestra degli appuntamenti.
