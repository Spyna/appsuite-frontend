9.5.1. Office-Dokumente anlegen

Sie können Office-Dokumente anlegen, die formatierten Text, Tabellen und Bilder enthalten.

So legen Sie ein Office-Dokument an:

  1. Öffnen Sie im Ordnerbaum einen Ordner.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Anlegen von Objekten haben.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Neu . Klicken Sie auf Neues Office Dokument.

  3. Klicken Sie links oben in der Seitenleiste auf Unnamed. Geben Sie einen Namen für das Dokument ein.

  4. Geben Sie den Text ein, oder fügen Sie Text aus der Zwischenablage ein.

  5. Um den Text zu editieren, verwenden Sie die gängigen Techniken aus bekannten Office-Anwendungen.

    Um Zeichen oder Absätze zu formatieren, nutzen Sie die Textwerkzeuge in der Seitenleiste.

  6. Zum Beenden klicken Sie rechts in der Office-Werkzeugleiste auf das Symbol Schließen .
