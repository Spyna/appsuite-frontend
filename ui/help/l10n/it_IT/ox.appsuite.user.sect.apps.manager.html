Il Gestore delle applicazioni

Barra laterale Contiene diverse viste delle applicazioni:

  • Le tue applicazioni. Offre viste di

    • tutte le applicazioni installate

    • tutte le applicazioni marcate come preferite

    • applicazioni che installano aggiornamenti

  • Categorie. Offre viste di

    • applicazioni di base

    • applicazioni di ufficio

    • applicazioni di sviluppo

Il numero accanto a una vista rappresenta il numero di applicazioni incluso nella vista.

Area di visualizzazione  Visualizza le applicazioni che sono contenute nella vista selezionata. Per ogni applicazione saranno visualizzate le seguenti informazioni:

  • icona, nome e origine dell'applicazione

  • una descrizione degli scopi per i quali è possibile utilizzare l'applicazione

  • un pulsante per avviare l'applicazione

  • un pulsante per marcare l'applicazione come preferita o per rimuovere un'etichetta. Un asterisco accanto al nome indica che l'applicazione è stata marcata come preferita.
