4.2.3. Ajout de composants au portail

Vous pouvez compléter les composants existants en en ajoutant de nouveaux.

Comment ajouter de nouveaux composants :

  1. Dans l'app Portail, cliquez sur Ajouter un composant en haut à droite. Sélectionnez une entrée.

  2. Certains composants ont besoin d'informations supplémentaires. Saisissez les valeurs demandées. Cliquez sur Enregistrer.

Conseil : dans l'app Fichiers vous pouvez ajouter un fichier en tant que composant du portail.

Conseil : dans l'app Courriel vous pouvez ajouter un composant du portail.
