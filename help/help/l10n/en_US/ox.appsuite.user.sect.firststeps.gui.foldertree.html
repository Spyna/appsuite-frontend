The user interface

3.1.4. The folder tree

To open or close the folder tree, use one of the following methods:

  • Click on View in the toolbar. Enable or disable Folder view.

  • On the bottom left side, click the Open folder view icon or the Close folder view icon.

Content

  • the app specific folders

  • depending on the application, sections for personal, public and shared folders

    Note: Depending on the groupware configuration, those sections might not be displayed if there are no public or shared folders.

  • The Folder-specific actions icon next to the selected folder. It contains functions for organizing folders. Depending on the folder, further functions might be available.

    Tip: You can also access the folder-specific actions by right-clicking on a folder.

  • Depending on the app, further functions might be available.
