2.2.3. Impostazioni su un dispositivo mobile

Come utilizzare le impostazioni di OX Drive su un dispositivo mobile:

  1. Nell'applicazione OX Drive, toccare l'icona del menu.

  2. Selezionare Impostazioni.

Sono disponibili le seguenti impostazioni.

  • Disconnettere

  • Generale

  • Photostream

  • File non in linea

  • Batteria

  • Informazioni su OX Drive

Disconnettere Disconnettersi. Fermare la sincronizzazione.

Generale Mostra varie impostazioni e informazioni.

  • Sincronizza solo all'avvio dell'applicazione. Imposta se la sincronizzazione è eseguita automaticamente all'avvio dell'applicazione.

  • Sincronizza le modifiche automaticamente. Imposta se la sincronizzazione è avviata automaticamente dopo una modifica.

  • Limita la dimensione della cache a. Specifica lo spazio che dovrebbe essere riservato ai dati sincronizzati.

  • URL server. Mostra l'indirizzo del server utilizzato per la sincronizzazione.

Photostream Imposta se le foto scattate con il dispositivo mobile sono trasferite al server.

  • Importa foto automaticamente. Abilita o disabilita Photostream.

  • Nuove foto. Limita l'importazione alle foto scattate dopo l'abilitazione di Photostream.

  • Tutte le foto. Trasferisce tutte le foto sul dispositivo mobile e tutte le foto scattate da quel momento in poi.

File non in linea Specifica i file da sincronizzare.

  • Tutti. Sincronizza tutti i file e le cartelle che sono posizionati sotto I miei file sul server di groupware.

  • Preferiti. Sincronizza solo i file e le cartelle che sono marcate come Preferiti sul dispositivo mobile.

Batteria Specifica lo stato della batteria che deve essere verificato per consentire la sincronizzazione.

  • Sincronizza solo in fase di carica. Sincronizza solo durante la fase di carica delle batterie.

  • Sospendi la sincronizzazione se inferiore a. Imposta se la sincronizzazione è sospesa se la capacità della batteria scende al di sotto di un determinato livello.

Informazioni su OX Drive Mostra le note di versione di OX Drive.
