Capitolo 12. Cifrare dati

Guard un componente di sicurezza del groupware che consente di cifrare messaggi di posta e file.

  • Cifrare le proprie comunicazioni di posta elettronica con altri utenti o collaboratori esterni.

  • Cifrare file singoli. Condividere i dati cifrati con altri utenti.

  • Utilizzare le opzioni di sicurezza per definire il livello di cifratura.

  • I dati cifrati sono protetti con password. Utilizzare la funzione di ripristino della password per proteggersi dalle conseguenze della perdita di una password.

Scoprire come funzione l'applicazione Guard.

  • applicare le impostazioni di base

  • cifrare le comunicazioni di posta elettronica

  • cifrare file

  • cifrare documenti di Office

  • applicare le impostazioni di sicurezza
