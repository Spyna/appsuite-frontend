6.6.7. Printing contacts

In order to print contacts you can:

  • print a contact's phone data

How to print a contact's phone data:

  1. Open an address book in the folder tree.

  2. Select a contact in the sidebar. Click on More in the display area. Click on Print in the menu. A window with a print preview opens.

  3. If required, change the printer settings. Click on the Print button.

  4. Close the print preview window.

Tip: You can also print a phone list with multiple contacts.
