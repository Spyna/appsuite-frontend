5.5.12. E-Mails drucken

Um E-Mails zu drucken, haben Sie folgende Möglichkeiten:

  • den Inhalt einer E-Mail drucken

So drucken Sie den Inhalt einer E-Mail:

  1. Wählen Sie in der Seitenleiste eine E-Mail. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Drucken. Ein Fenster mit einer Druckvorschau wird geöffnet.

  2. Falls erforderlich, ändern Sie die Druckeinstellungen. Klicken Sie auf die Schaltfläche Drucken.

  3. Schließen Sie das Fenster mit der Druckvorschau.

Tipp: Sie können auch die Inhalte mehrerer E-Mails gemeinsam drucken.
