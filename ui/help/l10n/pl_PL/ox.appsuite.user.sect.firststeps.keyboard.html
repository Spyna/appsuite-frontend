3.2. Obsługa za pomocą klawiatury

Zamiast myszy, możesz używać następujących klawiszy:

  • Aby przełączyć między paskiem menu, drzewem folderów, paskiem bocznym a obszarem wyświetlania, użyj kombinacji klawiszy [Ctrl]+[F6].

  • Aby przenieść kursor w obrębie pól wejściowych, użyj klawiszy kursora lub klawiszy [Home] bądź [End].

  • Aby przejrzeć drzewo folderów lub pasek boczny, użyj klawiszy [Page up], [Page down], [Spacja] oraz [Shift]+[Spacja].

  • Aby otworzyć lub zamknąć folder w drzewie folderów, użyj klawisza [Spacja] lub [Enter].

  • Aby wybrać elementy w drzewie folderów, na pasku bocznym lub obszarze wyświetlania, użyj klawiszy kursora.

  • Aby wybrać funkcję, pole wejściowe lub pole wyboru, użyj klawisza [Tab] lub kombinacji [Shift]+[Tab]. Wybrany element zostanie wyróżniony.

    Za pomocą kombinacji [Shift]+[Tab] można wybierać elementy w odwrotnej kolejności.

  • Aby wykonać wybraną funkcję, naciśnij klawisz [Enter].

  • Aby zaznaczyć lub wyczyścić wybrane pole wyboru, naciśnij klawisz [Spacja].

  • Aby przełączyć pola wyboru, użyj klawiszy kursora.

  • Aby zamknąć wyskakujące okienko lub okno dialogowe, naciśnij klawisz [Esc].
