9.8. Using Cloud Storages

If you use third party cloud storages like Google Drive, Dropbox, Box or OneDrive, you can access your cloud storage from within the groupware. There are the following options:

  • First you have to set up and edit the third party cloud storage in the groupware.

  • Then you can use your cloud storage with the help of the folder tree.
