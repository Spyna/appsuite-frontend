7.6.7. 打印聯繫人

要列印聯絡人，您可以:

  • 列印聯絡人的電話資料

如何列印聯絡人的電話資料：

  1. 開啟 資料夾樹中的通訊錄。

  2. 在側邊欄選擇一個聯絡人。點選顯示區域中的 更多 in the display area. 點選選單中的 列印。

  3. 如果需要，更改打印機設置。

  4. 要使用目前選擇的印表機列印，點選 列印。要使用系統對話框列印，單擊左下角的 使用系統對話框列印...。

提示: 您可以一次列印 多個聯絡人。
