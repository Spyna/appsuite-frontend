Praktijkvoorbeelden
Vaak gestelde vragen
1. Algemene vragen
Vraag: Waar kan ik mijn persoonlijke gegevens en instellingen vinden?
Vraag: Hoe kan ik mijn wachtwoord wijzigen?
Vraag: Waarom zijn bepaalde knoppen of invoervelden niet te vinden?
Vraag: Hoe wordt ik op de hoogte gebracht van nieuwe e-mailberichten of afspraakuitnodigingen?
Vraag: Hoe kan ik actuele berichten uit sociale netwerken of nieuwspagina's lezen?
2. Vragen over e-mailberichten en contacten
Vraag: Hoe kan ik een e-mailbericht sturen naar meerdere adressen in één keer ?
Vraag: Wat is het doel van de invoervelden Kopie aan... en Blinde kopie aan...?
Vraag: Ik stuur regelmatig e-mailberichten met de zelfde of vergelijkbare inhoud. Hoe kan ik dit sneller doen?
Vraag: Kan ik bijvoorbeeld e-mailberichten versturen naar mijn Google Mail account?
Vraag: Hoe kan ik automatisch e-mailberichten doorsturen naar mijn vervanger?
Vraag: Hoe kan ik de contacten uit mijn sociale netwerken gebruiken?
3. Vragen over taken, agenda's en afspraken
Vraag: Wanneer moet ik een taak maken en wanneer een afspraak?
Vraag: Hoe kan ik een taak maken van een e-mailbericht?
Vraag: Hoe kan ik de afspraken van iemand anders beheren als vertegenwoordiger?
Vraag: Hoe gebruik ik de beschikbaarheidsstatussen zoals Vrij, Bezet, Afwezig enz.?
Vraag: Hoe gebruik ik de herhalingsinstellingen in de agenda?
Vraag: Hoe vindt u vrije momenten bij het maken van afspraken?
4. Vragen over gegevensorganisatie en teamwork
Vraag: Hoe kan ik bepaalde contacten delen met mijn externe partners?
Vraag: Hoe kan ik bepaalde documenten delen met mijn externe partners?
Vraag: Ik wil een map met andere gebruikers delen. Welke rechten moet ik instellen?
