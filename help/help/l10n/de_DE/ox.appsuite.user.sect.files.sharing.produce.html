Freigaben

9.6.1. Daten freigeben

Sie haben die folgenden Möglichkeiten:

  • Ein Objekt oder mehrere Objekte wie Dateien oder Ordner zum Lesen oder Bearbeiten freigeben

  • Interne Benutzer oder externe Partner mit Hilfe einer E-Mail zu der Freigabe einladen.

    Für externe Partner, die kein Groupware Konto besitzen, wird automatisch ein Gastkonto angelegt. Die Einladungs-E-Mail enthält ein vorgegebenes Kennwort und Hinweise für den Zugriff auf das Gastkonto. Nach der erstmaligen Anmeldung auf dem Gastkonto muss der externe Partner das vorgegebene Kennwort ändern.

    Hinweis: Die Einladung-E-Mail wird ausschließlich im HTML-Format gesendet.

  • Alternativ können Sie für die Freigabe einen Link generieren lassen. Diesen Link können Sie per E-Mail versenden oder in sozialen Netzwerken veröffentlichen.

    Vorsicht: Jeder, der diesen Link besitzt, kann ohne Passwort auf die Freigabe zugreifen.

So geben Sie Objekte zum Lesen oder Bearbeiten frei:

  1. In der App Drive wählen Sie einen Ordner.

    Wählen Sie im Anzeigebereich ein Objekt oder mehrere Objekte.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Teilen . Das Fenster zum Freigeben von Objekten öffnet sich.

    Alternativ können Sie eine einzelne Datei auch im Viewer freigeben.

  3. Um zu bestimmen, für wen Sie die Daten freigeben wollen, verwenden Sie eine dieser Methoden:

    • Um andere Benutzer oder externe Partner mit Hilfe ihrer E-Mail-Adresse einzuladen, wählen Sie Andere einladen. Geben Sie die E-Mail-Adressen der Benutzer oder der externen Partner ein. Optional geben Sie eine Nachricht für die Empfänger ein.

    • Um einen Link zu erhalten, den Sie zum Beispiel per E-Mail, Messenger oder in sozialen Netzwerken weitergeben können, wählen Sie Einen Link abrufen.

      Vorsicht: Jeder, der diesen Link besitzt, kann ohne Passwort auf die Freigabe zugreifen.

  4. Standardmäßig werden die Daten auf unbestimmte Dauer zum Lesen freigegeben. Um ein andere Art der Freigabe festzulegen, klicken Sie auf Erweiterte Optionen. Aktivieren Sie eine oder mehrere der folgenden Optionen.

    • Wenn die Freigabe nach einer bestimmten Zeit erlöschen soll, aktivieren Sie Läuft ab in. Wählen Sie einen Zeitraum.

    • Wenn die Empfänger der Freigabe die Daten auch bearbeiten sollen, aktivieren Sie Empfänger können bearbeiten.

  5. Klicken Sie auf Teilen.
