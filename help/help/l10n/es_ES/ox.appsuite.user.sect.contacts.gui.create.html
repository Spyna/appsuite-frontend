6.1.8. La ventana de edición de contactos

Esta ventana se usa cuando se está creando un nuevo contacto o editando uno ya existente.

Para abrir la ventana, realice una de las siguientes acciones:

  • Pulse en Nuevo en la barra de herramientas. Pulse en Añadir contacto.

  • Seleccione un contacto. Pulse en Editar en la barra de herramientas.

Contenido

  • foto de contacto, nombre, empresa

    Pulse en la imagen de contacto para editarla o cargar una nueva imagen de contacto.

  • Datos de contacto

    • De manera predeterminada, se muestran esos datos:

      • nombre, apellidos, día de nacimiento

      • descripción del trabajo

      • Correo, teléfono

  • Añadir adjuntos. Añade ficheros.

  • Añadir campos. Añade campos para datos de contacto adicionales.
