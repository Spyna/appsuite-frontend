Managing appointments

Some of the techniques for organizing appointments require having set up own calendar folders. Information about creating folders can be found in Folders.

The following possibilities exist:

  • Editing appointments

  • Editing appointments with drag and drop

  • Changing the appointment status

  • Moving appointments to another folder

  • Deleting appointments

Editing appointments

You can edit all data entered when creating an appointment at a later time.

How to edit an appointment:

Prerequisite: You have the permission to modify objects in the folder containing the appointment.

  1. Depending on the view selected, use one of the following methods:

    • Click on an appointment in the Day, Work week, Week or Month view. In the pop-up click on Edit.

    • In the List view, select an appointment in the sidebar. Click on Edit in the display area.

    The appointment's data are displayed in a new page.

  2. Edit the data.

  3. Click on Save.
