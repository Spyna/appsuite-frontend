5.9.2. Configurer et modifier les comptes de courrier externes

Les possibilités suivantes s'offrent à vous :

  • configurer des comptes de courrier externes

  • modifier les comptes de courrier externes

  • supprimer des comptes de courrier externes

Pour configurer un compte de courrier électronique externe :

  1. Cliquez sur Ajouter un compte dans l'arborescence des dossiers. Dans la fenêtre qui s'ouvre, choisissez Compte de courrier.

  2. Cliquez sur une icône. La procédure dépend ensuite du fournisseur de compte de courrier.

    • Pour certains fournisseurs, une nouvelle fenêtre de navigateur s'ouvre. Connectez-vous avec vos identifiants pour autoriser l'accès au compte de courrier.

    • Si le fournisseur vous demande d'autoriser l'accès aux données, donnez l'autorisation correspondante.

    • Pour certains fournisseurs, saisissez vos identifiants dans la fenêtre Ajouter un compte de courrier. Cliquez ensuite sur Ajouter.

    Vous pouvez aussi saisir manuellement les données requises en cliquant surManuellement.

  3. Si les courriels de ce compte doivent également apparaître dans le dossierMessagerie unifiée, activez l'option Utiliser la messagerie unifiée pour ce compte.

Une entrée correspondant au compte de courrier apparaît alors dans l'arborescence des dossiers.

Pour modifier un compte de courrier :

  1. Cliquez sur l'icône Menu système à droite de la barre de menus. Dans le menu, choisissez Réglages.

  2. Dans la barre latérale, cliquez sur Comptes.

  3. Dans la zone d'affichage, cliquez sur Modifier à côté d'un compte. Les réglages apparaissent dans une fenêtre.

    Si vous utilisez plusieurs services liés au même fournisseur, par exemple courrier et stockage cloud, choisissez le service que vous souhaitez modifier.

  4. Modifiez les régalges. Remarques :

    • Si vous saisissez un nom dans le champ Votre nom des Réglages du compte, ce nom écrase le nom d'expéditeur pré-existant.

    • Les options de Serveur entrant et Serveur sortant ne peuvent être modifiées pour votre compte principal.

  5. Cliquez sur Enregistrer en bas de la fenêtre.

Pour supprimer un compte de courrier électronique externe :

  1. Cliquez sur l'icône Menu système à droite de la barre de menus. Dans le menu, choisissez Réglages.

  2. Dans la barre latérale, cliquez sur Comptes.

  3. Cliquez sur l'icône Supprimer.
