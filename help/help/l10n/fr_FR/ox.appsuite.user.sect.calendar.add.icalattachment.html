7.3.2. Créer des rendez-vous à partir de pièces jointes au format iCal

Vous pouvez créer un rendez-vous à partir d'un fichier iCal joint à un courriel.Un fichier iCal se reconnaît à son extension « .ics ».

Pour créer un rendez-vous à partir d'une pièce jointe iCal :

  1. Dans l'application Courriel, choisissez un courriel avec une pièce jointe iCal.

  2. Cliquez sur le nom de la pièce jointe dans la vue détaillée. Dans le menu, choisissez Ajouter à l'agenda.
