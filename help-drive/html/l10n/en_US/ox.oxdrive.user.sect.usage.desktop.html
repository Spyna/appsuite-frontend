2.1. On a Workstation

How to use the OX Drive client functions on a workstation:

  1. Open the OX Drive icon's context menu.

    Depending on the system, there are different ways of opening the context menu.

    • In MS Windows right-click the Files icon in the task bar on the right side of the info area. You might have to display the icon first by clicking the Show hidden icons icon.

  2. Select a function from the context menu.

The following functions are available.

  • Pause synchronization

  • OX Drive Open folder

  • Recently changed files

  • Preferences

  • OX Drive Exit

Pause synchronization Stops the synchronization. To proceed with the synchronization, click on Resume synchronization.

OX Drive Open folder Opens the local folder in your system's file browser.

Recently changed files Displays a list of recently changed files.

Preferences Opens the dialog window preferences. The following settings are available.

  • General tab

    • Quota. Shows information on the available space.

      To get more space click on Buy Quota.

    • Password settings. When having changed your password on the groupware server, you can enter the new password here.

      To do so, click on Refresh. Enter the new password.

    • Language Defines the user interface language.

  • Advanced tab

    • Autostart. Defines whether the OX Drive client is automatically launched at the system start.

    • Windows Explorer Integration. If this function is installed, icons in the Windows Explorer indicate the synchronization status of folders and files.

    • Unlink. Finishes the synchronization and logs you out. Launches the OX Drive Setup Wizard then. Here you can again define your credentials, the local folder and the remote folder.

  • Notifications tab. Shows possible errors that occurred during the synchronization.

  • About tab. Shows information about the OX Drive client version and the groupware server used.

OX Drive Exit Exits the OX Drive client.
