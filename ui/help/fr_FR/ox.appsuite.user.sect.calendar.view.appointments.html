8.2.3. Comment les rendez-vous sont-ils affichés ?

Dans les vues en calendrier, les rendez-vous sont colorés en fonction de la disponibilité :

  • Provisoire : jaune

  • Disponible : vert

  • Réservé : bleu

  • Absent : rouge
