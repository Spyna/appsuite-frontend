9.2.1. Visualizzare i file nella vista Elenco

Come visualizzare un file nella vista Elenco:

  1. Fare clic sull'icona Vista nella barra degli strumenti. Fare clic su Elenco.

  2. Aprire una cartella nell'albero delle cartelle.

  3. Per trovare rapidamente un determinato file, utilizzare la funzione di ricerca.

  4. Fare clic su un file nella barra laterale. I dati del file sono mostrati nell'area di visualizzazione. Per alcuni tipi di file viene mostrata un'anteprima.

  5. Per visualizzare un altro file, utilizzare uno dei seguenti metodi:

    • Fare clic su un altro file nella barra laterale.

    • Utilizzare le frecce direzionali per sfogliare l'elenco.
