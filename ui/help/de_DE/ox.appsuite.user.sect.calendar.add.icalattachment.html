8.3.2. Termin aus iCal-Anlage anlegen

Sie können einen Termin aus einer iCal-Anlage einer E-Mail anlegen. Eine iCal-Anlage hat standardmäßig die Namenserweiterung ics.

So legen Sie einen Termin aus der iCal-Anlage einer E-Mail an:

  1. In der App Mail zeigen Sie eine E-Mail an, die eine iCal-Anlage enthält.

  2. Klicken Sie im Anzeigebereich auf den Namen der Anlage. Klicken Sie im Menü auf Zum Kalender hinzufügen.
