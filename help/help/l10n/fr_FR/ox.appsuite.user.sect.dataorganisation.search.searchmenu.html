12.1.2. Quelles options de recherche existent ?

Vous disposez des options suivantes :

  • Chercher des objets à partir d'un terme à rechercher. Exemple :

    • Vous recherchez tous les courriels dont le sujet comporte le terme "Société".

    • Vous recherchez tous les rendez-vous dont la description comporte le terme "Réunion".

  • Chercher des objets à partir d'une propriété de l'objet. Exemple :

    • Vous recherchez tous les courriels envoyés durant la semaine passée.

    • Vous recherchez tous les rendez-vous que vous avez acceptés.

Dès que vous saisissez des caractères dans le champ Chercher, le contenu du menu de recherche est adapté.Si vous cliquez sur l'un de ses éléments, cela démarre la recherche. Suivant l'application, le contenu du menu de recherche varie :

Le menu de recherche de Courriel Présente les éléments suivants :

  • Si vous ne saisissez aucun terme à rechercher, les expéditeurs et destinataires les plus fréquents sont présentés sous Expéditeur/Destinataire

  • Si vous saisissez un terme à rechercher :

    • Courriels. Recherche parmi l'expéditeur et les destinataires ainsi que dans le sujet des courriels.

    • Sujet. Recherche dans le sujet des courriels.

    • Texte du courriel. Recherche dans le texte des courriels.

    • Expéditeur/Destinataire. Affiche les expéditeurs ou destinataires correspondant au terme à rechercher.

  • Heure. Recherche les courriels envoyés ou reçus durant un intervalle de temps donné.

Le menu de recherche de Carnet d'adresses Présente les éléments suivants :

  • Si vous ne saisissez aucun terme à rechercher, les contacts les plus fréquemment utilisés sont présentés sous Contact

  • Si vous saisissez un terme à rechercher :

    • Si vous cliquez sur le terme de recherche, la recherche s'effectue sur les noms, adresses et numéros de téléphone.

    • Noms. Recherche parmi les noms et prénoms.

    • Adresse électronique. Recherche parmi les adresses de courrier électronique des contacts.

    • Numéros de téléphone. Recherche parmi les numéros de téléphone.

    • Adresses. Recherche parmi les adresses.

  • Type. Rechercher soit parmi les contacts, soit parmi les listes de diffusion.

  • Type de dossier. Recherche dans un dossier public, privé ou partagé.

Le menu de recherche de Agenda Présente les éléments suivants :

  • Si vous ne saisissez aucun terme à rechercher, les participants les plus fréquemment utilisés sont présentés sous Participants

  • Si vous saisissez un terme à rechercher :

    • Si vous cliquez sur le terme de recherche, la recherche s'effectue sur le sujet et la description.

    • Sujet. Recherche dans le sujet.

    • Description. Recherche dans la description.

    • Lieu. Recherche dans le lieu.

    • Nom de pièce jointe. Recherche parmi les noms de fichiers des pièces jointes.

  • Mon état. Recherche parmi les rendez-vous ayant un état donné.

  • Date. Recherche parmi les rendez-vous soit passés, soit à venir.

  • Type. Recherche parmi les rendez-vous soit uniques, soit récurrents.

  • Type de dossier. Recherche dans un dossier public, privé ou partagé.

Le menu de recherche de Tâches Présente les éléments suivants :

  • Si vous ne saisissez aucun terme à rechercher, les participants les plus fréquemment utilisés sont présentés sous Participants à la tâche

  • Si vous saisissez un terme à rechercher :

    • Si vous cliquez sur le terme de recherche, la recherche s'effectue sur le sujet.

    • Sujet. Recherche dans le sujet.

    • Description. Recherche dans la description.

    • Nom de pièce jointe. Recherche parmi les noms de fichiers des pièces jointes.

    • Participants à la tâche. Recherche parmi les noms des participants.

  • État de la tâche. Recherche parmi les tâches ayant un état donné.

  • Type. Recherche parmi les tâches soit uniques, soit répétitives.

  • Type de dossier. Recherche dans un dossier public, privé ou partagé.

Le menu de recherche de Fichiers Présente les éléments suivants :

  • Si vous saisissez un terme à rechercher :

    • Si vous cliquez sur le terme de recherche, la recherche s'effectue sur le nom de fichier.

    • Nom de fichiers. Recherche parmi les noms de fichiers.

    • Description de fichier. Recherche dans la description des fichiers.

    • Contenu du fichier. Recherche parmi le contenu des fichiers.

  • Type de fichier. Recherche les fichiers d'un type donné.

  • Type de dossier. Recherche dans un dossier public, privé ou partagé.

  • Taille de fichier. Recherche les fichiers d'une taille donnée.

  • Type de dossier. Recherche dans un dossier public, privé ou partagé.
