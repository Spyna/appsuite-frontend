11.5. Visualización o finalización de sesiones

Puede iniciar sesión en su cuenta del software colaborativo con diferentes dispositivos en varios clientes simultáneamente, por ejemplo, con navegadores de escritorio, clientes de correo electrónico, tabletas, teléfonos inteligentes. Con cada inicio de sesión, se crea una sesión activa en el servidor del software colaborativo. En ocasiones sucesivas, puede tener sentido listar las sesiones activas y finalizarlas si es necesario:

  • Querrá comprobar si está conectado en un determinado dispositivo o cliente.

  • Querrá desconectarse de un determinado dispositivo.

  • Querrá evitar alcanzar el número máximo de sesiones permitidas ya que después no podrá iniciar sesión desde dispositivos adicionales.

Tiene las siguientes posibilidades:

  • mostrar todas las sesiones activas

  • finalizar sesiones individuales

  • finalizar todas las sesiones excepto la actual

Cómo editar las sesiones activas:

  1. Pulse el icono Configuración de la parte derecha de la barra de menús.

  2. En la barra lateral, pulse en Seguridad. Pulse en Clientes activos.

    Todas las sesiones activas se muestran en el área de visualización.

  3. Para finalizar sesiones, puede hacer lo siguiente:

    • Para finalizar una sesión individual, pulse el icono Cerrar sesión junto al cliente.

    • Para finalizar todas las sesiones excepto la activa, pulse en Cerrar sesión en todos los clientes.
