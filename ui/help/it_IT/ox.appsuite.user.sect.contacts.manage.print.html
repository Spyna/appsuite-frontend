7.6.7. Stampare i contatti

Per stampare i contatti, è possibile:

  • stampare i dati telefonici di un contatto

Come stampare i dati telefonici di un contatto:

  1. Aprire una rubrica nell'albero delle cartelle.

  2. Selezionare un contatto nella barra laterale. Fare clic su Altro nell'area di visualizzazione. Fare clic su Stampa dal menu. Si aprirà una finestra con un'anteprima della stampa.

  3. Se necessario, modificare le impostazioni della stampante. Fare clic sul pulsante Stampa.

  4. Chiudere la finestra dell'anteprima di stampa.

Suggerimento: è possibile stampare un elenco telefonico con " >più contatti.
