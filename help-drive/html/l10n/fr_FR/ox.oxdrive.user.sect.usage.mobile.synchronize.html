2.2.1. Choisir les données à synchroniser

Pour choisir les données à synchroniser sur un appareil mobile :

  1. Dans l'app OX Drive, touchez l'icône de menu.

  2. Choisissez Fichiers.

  3. Choisissez Tous. Assurez-vous que le dossier dont vous souhaitez synchroniser le contenu est visible.

  4. À côté du nom du dossier, faites glisser de la droite vers le milieu. Le menu du dossier apparaît.

  5. Touchez l'icône Favoris. Fermez le menu dossier.
