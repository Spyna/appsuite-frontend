Travailler avec les Applications

Pour lancer une application :

  1. Pour afficher un certain type d'applications, cliquez sur une des vues dans la barre latérale.

  2. Cliquez sur Démarrer dans la zone d'affichage sous l'application.

Résultat : la page Apps se ferme et l'application est lancée.

Pour ajouter une application à vos applications favorites :

  1. Cliquez dans la vue Installées dans la barre latérale.

  2. Cliquez sur Marquer comme favorite dans la zone d'affichage sous l'application.

Résultat : l'application est ajouté à la vue des applications Favorites. Un astérisque sera ajouté à côté du nom de l'application.

Pour ajouter une application à vos favoris :

  1. Cliquez sur la vue Installées dans la barre latérale.

  2. Cliquez sur un des choix dans la zone d'affichage sous l'application.

Résultat : l'application mise à jour est installée.
