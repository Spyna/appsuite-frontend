6.3. Viewing or Saving Contact Attachments

The file names of contact attachments are displayed in the display area below the contact name. The following functions are available:

  • displaying a preview of the attachment

  • opening the attachment in the browser

  • downloading the attachment

  • saving the attachment to Drive

Note: Depending on the attachment's file format, the available function might differ.

How to use the contact attachment functions:

  1. Select a contact with an attachment.

  2. Click on an attachment's name in the detail view. A menu with several functions opens.

  3. Click on the function wanted. Depending on the function, further actions might be available.

Tip: If a contact contains several attachments, an action can be executed for all attachments at once. To do so, click on All attachments. Click on the function wanted.
