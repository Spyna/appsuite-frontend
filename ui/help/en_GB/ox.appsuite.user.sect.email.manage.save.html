6.5.10. Saving email messages

You can save an email as a text file. The text file will have the file extension EML.

How to save an email:

  1. Select an email in the side bar.

  2. Click on More in the display area. Select Save as file from the menu.

  3. Follow the instructions for saving the EML file.
