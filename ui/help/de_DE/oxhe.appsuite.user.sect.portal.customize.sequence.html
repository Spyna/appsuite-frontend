Reihenfolge der Kacheln ändern

Sie können festlegen, in welcher Reihenfolge die Kacheln angezeigt werden.

So ändern Sie die Reihenfolge der Kacheln:

  1. Klicken Sie rechts unterhalb der Menüleiste auf Diese Seite personalisieren.

  2. Ziehen Sie eine Kachel an eine andere Position.
