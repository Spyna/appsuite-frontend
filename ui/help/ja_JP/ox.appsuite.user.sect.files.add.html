9.4. ファイルの作成

ファイルは次の要素から構成されます。一部の要素は省略できます。

  • 説明

  • ローカルに保存されたファイルの 1 つ以上のバージョン

  • コメント

新規ファイルを作成する

  1. フォルダ ツリーで、フォルダを 開きます。

    注: オブジェクトを作成する権限のあるフォルダを開いてください。

  2. ツール バーで、追加 アイコン をクリックします。新規ファイルをアップロード をクリックします。

  3. 新規ファイルを追加 ウィンドウに説明を入力します。

  4. ローカルに保存されたファイルをアップロードするには、参照 をクリックし、目的のファイルを選択します。

  5. コメント フィールドには、追加情報を入力できます。

  6. 保存 をクリックします。

ヒント: ファイルをデスクトップから ファイル アプリケーション ウィンドウ上にドラッグし、このウィンドウの上部にドロップしても新規のファイルを作成できます。
