Configuration requise

Pour pouvoir travailler correctement avec OX App Suite, la machine sur laquelle vous travaillez doit satisfaire aux exigences suivantes.

Résolution et dimensions de l'écran La résolution de l'affichage minimale est 1024 x 768.

Navigateur 

  • Google Chrome, dernière version

Réglages du navigateur 

  • Les cookies doivent être activés

  • JavaScript doit être activé

  • Les fenêtres qui apparaissent spontanément (pop-ups) doivent être autorisées
