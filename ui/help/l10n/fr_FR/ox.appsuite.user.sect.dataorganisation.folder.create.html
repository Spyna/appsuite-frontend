12.2.3. Créer des dossiers

Vous pouvez créer un nombre arbitraire de sous-dossiers dans un dossier privé. Pour créer des sous-dossiers dans un dossier partagé ou public, vous devez avoir les droits requis. Selon l'app, vous pouvez créer des dossiers ou des sous-dossiers.

  • Dans l'app Courriel, vous pouvez créer des dossiers et des sous-dossiers.

  • Dans l'app Lecteur, vous pouvez uniquement créer des sous-dossiers, pas de dossiers.

  • Dans les apps Carnet d'adresses, Agenda et Tâches, vous pouvez créer des dossiers, mais pas de sous-dossiers.

Pour créer un nouveau dossier :

  1. Suivant l'app, la procédure est différente :

    • Dans l'app Courriel, cliquez sur Nouveau dossier sous l'arborescence des dossiers.

    • Dans les apps Carnet d'adresses, Agenda et Tâches, cliquez sur Ajouter un dossier privé ou sur Ajouter un dossier public dans l'arborescence des dossiers.

      Remarque : dans l'app Lecteur, vous ne pouvez pas créer de dossiers, que des sous-dossiers.

  2. Saisissez un nom dans la fenêtre Nouveau dossier. Cliquez sur Ajouter un dossier.

Pour créer un nouveau sous-dossier :

  1. Dans les apps Courriel et Lecteur, dans l'arborescence des dossiers, sélectionnez le dossier dans lequel vous voulez créer un nouveau sous-dossier.

    Remarque : choisissez un dossier dans lequel vous avez le droit de créer des sous-dossiers.

  2. Cliquez sur l'icône Actions sur le dossier près du nom du dossier. Cliquez sur Nouveau sous-dossier.

  3. Saisissez un nom dans la fenêtre Nouveau dossier. Cliquez sur Ajouter un dossier.
