9.6. Freigaben

Mit Hilfe von Freigaben können Sie mit anderen Benutzern oder mit externen Partnern zusammenarbeiten. Sie haben die folgenden Möglichkeiten.

  • Einzelne Dateien oder Ordner für andere Benutzer oder für externe Partner zum Lesen oder Bearbeiten freigeben.

  • Auf einzelne Dateien oder Ordner zugreifen, die andere Benutzer für Sie freigeben.

  • Verschaffen Sie sich einen Überblick über Ihre Freigaben oder ändern Sie einzelne Freigaben, indem Sie diese auf einer Übersichtsseite verwalten.
