9.9. Réglages de Drive

Pour utiliser les réglages de Drive :

  1. Cliquez sur l'icône Menu système à droite de la barre de menu. Cliquez sur Réglages dans le menu.

  2. Si l'intitulé Drive n'apparaît pas dans la barre latérale, activez les Réglages avancés en bas de la barre latérale.

    Cliquez sur Drive dans la barre latérale.

  3. Modifiez les réglages.

Les réglages suivants sont disponibles.

  • Afficher les fichiers et les dossiers cachés

Afficher les fichiers et les dossiers cachés Indique si les fichiers et dossiers cachés doivent être affichés ou non. Les clients Drive locaux peuvent créer ce type de fichiers et dossiers.
