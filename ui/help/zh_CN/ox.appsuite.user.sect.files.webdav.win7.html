9.6.4. 在 Windows 7 设置 WebDAV

为了在 Windows 7 设置 WebDAV 访问，使用以下选项之一：

  • 使用 OX 更新器。在 OX 更新器用户文档可以找到信息。

  • 第一，为网路文件夹安装微软软件更新。然后，设置WebDav 访问。

如何在 Windows 7 设置 WebDAV 访问：

  1. 在文件管理器的导航区域，选择条目主机。

  2. 在图标条里单击链接盘。会打开连接盘窗口。

  3. 单击链接一个您可以用来保存文档的网站。会打开添加网路位置窗口。单击下一步。

  4. 选择选择自定义网络地址 。单击下一步。

  5. 在互联网或网路地址输入字段里输入以下地址： webdav://<地址>/servlet/webdav.infostore 其中<地址>必须要用组件服务器的 IP 地址或 URL 代替。单击下一步。

  6. 输入您的用户名和密码。使用您组件服务器的登录数据。单击 确定。

  7. 在下一个对话页上，您可以向网络资源指定一个名称。单击下一步。

  8. 单击结束。

结果在主机下面，您能访问您的 文件 应用程序文件。
