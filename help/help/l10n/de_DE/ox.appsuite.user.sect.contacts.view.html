6.2. Kontakte anzeigen

Ihr persönliches Adressbuch finden Sie im Ordnerbaum unter dem Namen Kontakte. Welches Adressbuch standardmäßig geöffnet wird, können Sie in den Adressbuch-Einstellungen bestimmen.

So zeigen Sie einen Kontakt an:

  1. Öffnen Sie im Ordnerbaum ein Adressbuch.

  2. Um Kontakte ab einem bestimmten Anfangsbuchstaben anzuzeigen, klicken Sie in der Navigationsleiste auf einen Buchstaben.

  3. Klicken Sie in der Liste auf einen Kontakt. Die Daten des Kontakts werden in der Detailansicht angezeigt.

    Sie können den Kontakt auf einer separaten Seite öffnen, indem Sie den Kontakt in der Liste doppelklicken.

  4. Um einen anderen Kontakt anzuzeigen, verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Liste auf einen anderen Kontakt.

    • Blättern Sie mit Hilfe der Cursortasten durch die Liste.
