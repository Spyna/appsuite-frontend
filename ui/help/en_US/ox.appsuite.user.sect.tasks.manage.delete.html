11.5.7. Deleting tasks

You can delete one task or multiple tasks together.

How to delete a task:

Warning: When deleting a task, this task is irrevocably lost.

  1. Select a task in the sidebar.

  2. Click on Delete in the display area.

  3. Confirm that you want to delete the task.

Result: The task is deleted.
