E-Mails organisieren

5.5.1. Mit Kategorien arbeiten

Mit Hilfe von Kategorien können Sie bestimmen, dass eingehende E-Mails im Ordner Posteingang je nach Absender getrennt abgelegt werden. Sie haben die folgenden Möglichkeiten.

  • Kategorien aktivieren oder deaktivieren

  • Eingehende E-Mails einer Kategorie zuweisen

  • Die Darstellung von Kategorien bearbeiten

So aktivieren oder deaktivieren Sie die Verwendung von Kategorien:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Öffnen Sie in der Seitenleiste den Eintrag E-Mail.

  3. Aktivieren oder deaktivieren Sie im Anzeigebereich unterhalb von Posteingang Tabs das Kontrollfeld Tabs für Posteingang anzeigen.

So weisen Sie eingehende E-Mails im Posteingang einer Kategorie zu:

  1. Öffnen Sie im Ordnerbaum den Ordner Posteingang.

  2. Wählen Sie in der Kategorienleiste die Kategorie Allgemein. Die E-Mails, die Sie bisher keiner Kategorie zugewiesen haben, werden in der Liste angezeigt.

    Wenn Keine Kategorienleiste angezeigt wird, aktivieren Sie die Kategorienleiste in den E-Mail Einstellungen.

  3. Ziehen Sie eine E-Mail aus der Liste auf einen Tab in der Kategorienleiste.

    Je nach Einstellung der Posteingang Tabs werden Sie nach dem Loslassen der E-Mail gefragt, ob Sie nur diese eine Nachricht oder alle vorhandenen und künftigen Nachrichten dieses Absenders der Kategorie zuweisen wollen.

So bearbeiten Sie die Darstellung von Kategorien:

Voraussetzung: Die Anzeige von Kategorien ist aktiviert.

  1. Klicken Sie in der Kategorienleiste rechts auf das Symbol Bearbeiten . Das Fenster Edit tabs öffnet sich.

  2. Um eine Kategorien ein- oder auszuschalten, aktivieren oder deaktivieren Sie das zugehörige Kontrollfeld.

    Um eine Kategorien umzubenennen, bearbeiten Sie den zugehörigen Text.

    Hinweis: Einige Kategorien können nicht bearbeitet werden.

  3. Klicken Sie auf Speichern.
