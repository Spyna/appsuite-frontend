13.2.1. Exporter des données depuis un dossier

Vous pouvez exporter les objets suivants :

  • Les contacts d'un dossier de contacts privé ou public.

  • Les rendez-vous d'un dossier d'agenda privé ou public.

  • Les tâches d'un dossier de tâches privé ou public.

Pour exporter les objets d'un dossier :

  1. Dans l'arborescence des dossiers, Ouvrez un dossier privé ou public.

  2. Cliquez sur l'icône Modifier le dossier sous l'arborescence des dossiers. Choisissez Exporter.

  3. Choisissez un format de données. Cliquez sur Exporter.
