Edición de varias tareas de una vez

Puede aplicar las siguientes funciones para múltiples tareas a la vez:

  • Mover tareas a otra carpeta

  • Etiquetado de tareas como completadas o no completadas

  • Eliminar tareas

Cómo ejecutar una función para varias tareas a la vez:

  1. Presione el icono Seleccionar Se mostrará una casilla de verificación en la barra lateral junto a cada tarea.

  2. Marque la casilla de verificación para al menos 2 tareas.

  3. Presione en una función del área de visualización.
