2.1. Using OX Drive Clients on a Workstation

How to use the OX Drive client functions on a workstation:

  1. Open the OX Drive icon's context menu.

    Depending on the system, there are different ways of opening the context menu.

    • In MS Windows right-click the Files icon in the task bar on the right side of the info area. You might have to display the icon first by clicking the Show hidden icons icon.

  2. Select a function from the context menu.

The following functions are available.

  • Pause synchronisation

  • Open OX Drive Folder

  • Recently changed files

  • Preferences

  • Exit OX Drive

Pause synchronisation Stops the synchronisation. To proceed with the synchronisation, click on Resume.

Open OX Drive Folder Opens the local folder in your system's file browser.

Recently changed files Displays a list of recently changed files.

Preferences Opens the preferences dialog window. The following settings are available.

  • General tab

    • Quota. Shows information on the available space.

      To get more space click on Buy Quota.

    • Password settings. When you have changed your password on the groupware server, you can enter the new password here.

      To do so, click on Refresh. Enter the new password.

    • Language

  • Advanced tab

    • Autostart. Defines whether the OX Drive client is automatically launched at system start.

    • Windows Explorer Integration. If this function is installed, icons in the Windows Explorer indicate the synchronisation status of folders and files.

    • Unlink. Finishes the synchronisation and logs you out. Then launches the OX Drive Setup Wizard. Here you can define your credentials, the local folder and the remote folder again.

  • Notifications tab. Shows possible errors that occurred during the synchronisation.

  • About tab. Shows information about the OX Drive client version and the groupware server used.

Exit OX Drive Exits the OX Drive client.
