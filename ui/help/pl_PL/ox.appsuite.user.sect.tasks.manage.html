11.5. Zarządzanie zadaniami

Niektóre metody porządkowania zadań wymagają wcześniejszego przygotowania folderów zadań. Informacje o tworzeniu folderów można znaleźć tutaj: Foldery.

Dostępne są następujące możliwości:

  • Edycja zadań

  • Oznaczanie zadań jako wykonanych

  • Zmiana terminu zadania

  • Przenoszenie zadań

  • Zmiana potwierdzenia zadań

  • Drukowanie zadań

  • Usuwanie zadań

  • Edycja wielu zadań jednocześnie
