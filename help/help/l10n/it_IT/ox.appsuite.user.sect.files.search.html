9.6. Cercare i file

Per cercare i file, è possibile utilizzare i seguenti criteri di ricerca:

  • termini di ricerca per nomi dei file, descrizioni dei file, contenuto dei file

  • Cartelle

  • tipo di file

Come cercare i file:

  1. Digitare un termine di ricerca nel campo di inserimento della barra di ricerca. Si aprirà il menu di ricerca.

  2. Utilizzare uno dei seguenti metodi:

    • Per cercare nel nome del file, nella descrizione del file o nel contenuto del file, fare clic sul termine di ricerca nel menu di ricerca. È possibile anche premere Invio o fare clic sull'icona Ricercanella barra di ricerca.

      Per cercare solo tra i nomi, fare clic su nei nomi dei file nel menu di ricerca.

      Allo stesso modo, è possibile limitare la ricerca alla descrizione del file o al contenuto del file.

    Risultato: i risultati della ricerca sono visualizzati nell'elenco mostrato nell'area di visualizzazione. I criteri di ricerca utilizzati sono visualizzati sotto la finestra di ricerca.

  3. È possibile raffinare ulteriormente il risultato della ricerca:

    • Per visualizzare tutti i criteri di ricerca, abilitare Mostra filtri avanzati.

      Per modificare un criterio, fare clic su accanto al criterio di ricerca. Selezionare una voce dal menu. È possibile utilizzare i seguenti criteri di ricerca:

      • È possibile utilizzare il menu a tendina Cartella per cercare in una cartella specifica.

      • Il menu a tendina Tipo di file consente di limitare la ricerca a determinati tipi di file.

    • Per migliorare la propria ricerca nell'elenco degli oggetti trovati, avviare una nuova ricerca.

    • Per rimuovere un criterio di ricerca, fare clic sull'icona . Se si rimuovono tutti i criteri di ricerca, la ricerca sarà terminata.

  4. Per terminare la ricerca, fare clic su Chiudi la ricerca.
