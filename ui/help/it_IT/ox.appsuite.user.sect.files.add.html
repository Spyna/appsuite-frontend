Creare i file

Un file può consistere di:

  • una descrizione

  • una o più versioni di un file salvato localmente

  • commenti

Come creare un nuovo file:

  1. Aprire una cartella nell'albero delle cartelle.

    Nota: aprire una cartella per la quale si dispone dei permessi appropriati per creare oggetti.

  2. Fare clic sull'icona Aggiungi nella barra degli strumenti. Fare clic su Carica un nuovo file.

  3. Digitare una descrizione nella finestra Aggiungi un nuovo file

  4. Per caricare un file salvato localmente, fare clic su Sfoglia. Selezionare un file.

  5. È possibile inserire informazioni aggiuntive nel campo Commenti.

  6. Fare clic su Salva.

Suggerimento: è possibile creare un nuovo file anche trascinando un file dal proprio desktop alla finestra dell'applicazione File e rilasciandolo nella parte superiore.
