E-Mail

5.3. E-Mail-Anhang ansehen oder speichern

Die Dateinamen von E-Mail-Anhängen werden in der Detailansicht angezeigt. Die folgenden Funktionen sind verfügbar:

  • Anhänge anzeigen oder herunterladen

  • Je nach Konfiguration der Groupware können Sie Anhänge in der App Drive anzeigen.

So verwenden Sie die Funktionen für E-Mail-Anhänge:

  1. Wählen Sie eine E-Mail, die einen Anhang oder mehrere Anhänge enthält.

  2. Klicken Sie neben dem Symbol Anhang auf eine der Schaltflächen.

    • Um den Anhang im Viewer anzuzeigen, klicken Sie auf Ansicht

      Je nach Dateiformat des Anhangs ist diese Funktion nicht verfügbar.

    • Um den Anhang in einem neuen Tab zu öffnen, klicken Sie auf In Browser öffnen.

    • Um den Anhang herunterladen, klicken Sie auf Download.

    • Um den Anhang in Drive zu speichern , klicken Sie auf In Drive speichern.

    Wenn mehrere Anhänge vorhanden sind, werden die Funktionen auf alle Anhänge angewendet.

  3. Wenn Sie eine Funktion für einen bestimmten Anhang ausführen wollen, klicken Sie auf das Symbol Anhang . Unterhalb des Symbols erscheinen die Namen des Anhangs. Klicken Sie auf den Namen eines Anhangs. Wählen Sie im Menü eine Funktion.

    Wenn Sie eine Vorschau eines Anhangs sehen wollen, klicken Sie rechts auf das Symbol Kacheln . Die Anhänge werden als Kacheln angezeigt. Klicken Sie auf eine Kachel.

    Je nach Konfiguration der Groupware können Sie die Anhänge auf einer eigenen Seite anzeigen. Klicken Sie in der Werkzeugleiste auf Ansicht. Klicken Sie auf All attachments.

So zeigen Sie Anhänge in der App Drive an:

Hinweis: Je nach Konfiguration der Groupware ist diese Funktion nicht verfügbar.

  1. Wählen Sie eine E-Mail, die einen Anhang oder mehrere Anhänge enthält.

    Klicken Sie unterhalb der Ordner auf View all Attachments. In der App Drive enthält der Ordner My mail attachments die Anhänge, die Sie gesendet oder empfangen haben.

  2. Bei Bedarf verwenden Sie die Funktionen von Drive.
