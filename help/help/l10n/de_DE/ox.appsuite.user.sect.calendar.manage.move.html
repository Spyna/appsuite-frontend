7.7.6. Termine in einen anderen Kalender verschieben

Sie können Termine in einen anderen Kalender verschieben.

Voraussetzung: Sie haben im Zielkalender die Berechtigung zum Anlegen von Objekten.

So verschieben Sie Termine in einen anderen Kalender:

  1. Je nach Ansicht verwenden Sie eine der folgenden Methoden:

    • In einer Kalenderansicht klicken Sie auf einen Termin. Klicken Sie im Popup auf das Symbol Weitere Aktionen .

    • In der Listenansicht wählen Sie einen Termin oder mehrere Termine. Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen .

    Klicken Sie auf Verschieben.

  2. Wählen Sie im Fenster Verschieben einen Kalender. Bei Bedarf legen Sie einen neuen Kalender an, indem Sie auf Ordner anlegen klicken.

  3. Klicken Sie auf Verschieben.
