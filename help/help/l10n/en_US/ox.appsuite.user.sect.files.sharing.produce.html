9.6.1. Sharing data

There are the following options:

  • share one or several objects like files or folders with read or edit permissions

  • Invite internal users or external partners per E-Mail to a share.

    For external partners without a groupware account, a guest account will be created automatically. The invitation E-Mail contains a preset password and notes on the guest account access. After the initial login to the guest account, the external partner has to change the preset password.

    Note: The invitation E-Mail is exclusively sent in html format.

  • You can also generate a link for the share. You can send this link per E-Mail or publish it in social networks.

    Caution: Everyone who receives this link can access the share without a password.

How to share objects with read or edit access:

  1. Select a folder in the Drive app.

    Select one or several objects in the display area.

  2. Click the Share icon in the toolbar. The window for sharing objects will be opened.

    You can also share a single file in the Viewer.

  3. In order to define the people with whom you want to share the data, use one of the following options:

    • To invite other users or external partners by using your E-Mail address, select Invite people. Enter the E-Mail addresses of the users or external partners. You can enter a message for the recipients.

    • To get a link that e.g. can be send per E-Mail, Messenger or in social networks, select Get a link.

      Caution: Everyone who receives this link can access the share without a password.

  4. Per default, the data are shared with read access for an unlimited time. To set other sharing options, click on Advanced options. Enable one or multiple of the following options.

    • To set a time limit for the access to the shared data, enable Expires in. Select a time range.

    • To give edit access to the recipients of the shared data, enable Recipients can edit.

  5. Click on Share.
