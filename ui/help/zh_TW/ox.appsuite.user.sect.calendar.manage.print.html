8.5.5. 打印約會

要列印日曆與約會，您可以:

  • 打印含約會的日曆表

  • 打印約會數據

如何打印包含約會的日曆表：

  1. 點選選單列中的 視圖 圖示 。點選下列一個項目：日、工作周、周、月

  2. 開啟 行事曆資料夾於資料夾樹。

  3. 指定要顯示的約會。

    • 要顯示所有私人日曆文件夾中您的所有約會，點選 顯示全部。

    • 要僅顯示您目前選擇的行事曆中的約會，取消選取 顯示全部。

  4. 點下工具列中的 列印 圖示 。

  5. 要使用目前選擇的印表機列印，點選 列印。要使用系統對話框列印，單擊左下角的 使用系統對話框列印...。

如何列印約會資料:

  1. 視所選的視圖而定，使用下列一個方式：

    • 在 日、工作周、週 或 月 視圖中點選一個約會。點選彈出視窗中的 列印。

    • 在 清單 視圖中選擇側邊欄中的約會。點選顯示區域中的 列印。

  2. 如果需要，更改打印機設置。

  3. 要使用目前選擇的印表機列印，點選 列印。要使用系統對話框列印，單擊左下角的 使用系統對話框列印...。
