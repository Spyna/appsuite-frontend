9.2. Viewing Files

By default, the content of your personal files folder is displayed. The folder is named My files.

How to display a file:

  1. Click the View icon in the toolbar. Select an entry.

    You can also use the icons in the function bar to select a view.

  2. Open a folder in the foldr tree.

  3. In order to quickly find a certain file, use the search function.

  4. Click on a file in the display area. The data are displayed in the popup.
