9.6.1. 在 Linux 设置 WebDAV

如何在 Linux 设置 WebDAV 访问：

  1. 打开 KDE Konqueror 或类似的浏览器。

  2. 在地址条中输入以下地址： webdav://<address>/servlet/webdav.infostore 用组件服务器的 IP 或 URL 替换 <address>。

  3. 输入您的用户名称和密码以访问组件服务器。

结果： 文件 应用程序在浏览器里显示。
