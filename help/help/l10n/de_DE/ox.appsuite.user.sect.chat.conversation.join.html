10.2.3. Einem Chat beitreten

Je nachdem, ob Sie einem Gruppenchat oder einem öffentlichen Kanal beitreten wollen, haben Sie folgende Möglichkeiten:

  • Ein Gruppenchat ist nur für die Mitglieder der Gruppe sichtbar. Sie können einem Gruppenchat nicht von sich aus beitreten.

  • Im Gegensatz dazu sind öffentliche Kanäle für alle Benutzer sichtbar. Sie können einem öffentlichen Kanal einfach beitreten.

So beteiligen Sie sich an einem Gruppenchat:

  • Bitten Sie ein Mitglied der Gruppe, Sie zu dem Gruppenchat hinzuzufügen.

So treten Sie einem öffentlichen Kanal bei:

  1. Klicken Sie in der Chat-Liste unten auf Alle Kanäle. Die öffentlichen Kanäle werden im Chat-Feld angezeigt.

  2. Verwenden Sie eine der folgenden Methoden:

    • Um einem Kanal direkt beizutreten, klicken Sie neben einem Kanal auf Beitreten.

    • Um die bisherigen Nachrichten eines Kanals und die Mitglieder des Kanals anzuzeigen, klicken Sie auf den Kanal.

      Um dem Kanal beizutreten, klicken Sie unterhalb des Chat-Fensters auf Beitreten.
