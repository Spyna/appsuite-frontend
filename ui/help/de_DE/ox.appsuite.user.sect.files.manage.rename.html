Dateinamen bearbeiten

Sie können den Dateinamen einer Datei bearbeiten.

So bearbeiten Sie den Dateinamen:

Voraussetzung: Sie haben für den Ordner, der die Datei enthält, die Berechtigung zum Bearbeiten von Objekten.

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Symbole klicken Sie auf eine Datei. Klicken Sie im Popup auf Weitere. Klicken Sie im Menü auf Umbenennen.

    • In der Ansicht Liste wählen Sie in der Seitenleiste eine Datei. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Umbenennen.

  2. Bearbeiten Sie den Dateinamen. Achten Sie auf die Namenserweiterung.

  3. Klicken Sie auf Umbenennen.
