9.4. Créer des fichiers

Un fichier consiste en :

  • une description

  • une ou plusieurs versions d'un fichier enregistré localement

  • des commentaires

Pour créer un nouveau fichier :

  1. Ouvrez un dossier dans l'arborescence des dossiers.

    Remarque : ouvrez un dossier pour lequel vous avez les droits permettant de créer des objets.

  2. Cliquez sur l'icône Ajouter dans la barre d'outils. Cliquez sur Télécharger un nouveau fichier.

    Conseil : vous pouvez aussi créer un nouveau fichier en cliquant sur Télécharger un nouveau fichier dans la barre de fonctions en haut de la zone d'affichage.

  3. Saisissez un titre dans la fenêtre Ajouter un nouveau fichier.

  4. Pour télécharger un ficher enregistré localement, cliquez sur Choisir un fichier. Choisissez un fichier.

  5. Vous pouvez saisir des informations supplémentaires dans le champ Description.

  6. Cliquez sur Enregistrer.

Conseil : vous pouvez aussi créer un nouveau fichier en faisant glisser un fichier depuis votre bureau jusqu'à la fenêtre de l'app Fichiers et en le déposant dans la partie supérieure.
