9.5.3. Toont de link naar een bestand

U kan de link tonen naar een bestand welke is opgeslagen in de Bestanden app.

U kan de link naar het bestand als volgt opvragen:

  • Klik op een bestand in het hoofdgebied. In de popup klikt u op Meer. Klik op Toon als interne link in het menu.
