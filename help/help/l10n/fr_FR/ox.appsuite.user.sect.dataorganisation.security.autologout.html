Sécurité

10.5.1. Déconnexion automatique

Vous pouvez définir si vous êtes automatiquement déconnecté si vous n'avez pas travaillé dans le collecticiel pour une durée donnée.

Pour définir si vous êtes automatiquement déconnecté :

  1. Cliquez sur l'icône Réglages à droite de la barre de menu. Cliquez sur l'élément Réglages.

  2. Dans la barre latérale, cliquez sur Sécurité.

  3. Choisissez une option dans le menu déroulant Déconnexion automatique dans la zone d'affichage.
