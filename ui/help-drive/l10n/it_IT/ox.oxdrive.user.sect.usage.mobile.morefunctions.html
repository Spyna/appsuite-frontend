2.2.3. Utilizzare altre funzioni

Sui dispositivi mobili, il client di OX Drive contiene le seguenti funzioni.

  • Seleziona una o più cartelle. Spostare o eliminare le cartelle selezionate.

  • Creare una nuova cartella.

  • Cercare file o cartelle.

  • Ordinare i file o le cartelle in base a determinati criteri.

  • Filtrare i file in una cartella. I criteri di filtraggio sono: tutti i file, i file sincronizzati, i file non sincronizzati.

Come utilizzare le funzioni di OX Drive sui dispositivi mobili:

  1. Nell'applicazione OX Drive, toccare l'icona del menu.

  2. Selezionare File.

  3. Selezionare una funzione dalla barra delle funzioni.

    In base al dispositivo mobile, la barra delle funzioni può essere in posizioni diverse: su un dispositivo Android sul bordo superiore, sui dispositivi iOS sul bordo inferiore dello schermo.

    In base al dispositivo mobile, le funzioni nella barra delle funzioni sono disposte diversamente.
