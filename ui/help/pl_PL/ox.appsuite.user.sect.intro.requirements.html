Wymagania systemowe

Aby można było sprawnie pracować z oprogramowaniem OX App Suite, komputer lokalny musi spełniać poniższe wymagania systemowe.

Rozdzielczość/rozmiar ekranu Minimalna rozdzielczość wynosi 1024 x 768.

Przeglądarka 

  • Google Chrome w najnowszej wersji

Ustawienia przeglądarki 

  • Musi być włączona obsługa plików cookie

  • Musi być włączona obsługa skryptów JavaScript

  • Musi być dozwolone wyświetlanie wyskakujących okienek
