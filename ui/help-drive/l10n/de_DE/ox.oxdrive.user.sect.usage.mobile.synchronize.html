2.2.1. Daten zum Synchronisieren auswählen

So wählen Sie auf einem Mobilgerät die Daten zum Synchronisieren:

  1. In der App OX Drive tippen Sie auf das Menü-Symbol.

  2. Wählen Sie Dateien.

  3. Wählen Sie Alle. Stellen Sie sicher, dass der Ordner zu sehen ist, dessen Inhalt Sie synchronisieren wollen.

  4. Streichen Sie neben dem Ordnernamen vom rechten Rand zur Mitte. Das Ordnermenü öffnet sich.

  5. Tippen Sie auf das Symbol Favoriten. Schließen Sie das Ordnermenü.
