Wo finde ich freigegebene Ordner anderer Benutzer?

Das folgende Beispiel zeigt, wo Sie im Ordnerbaum die freigegebenen Ordner anderer Benutzer finden. Folgende Situation ist vorgegeben:

  • Der Benutzer Timo Meier gibt Ihnen folgende persönliche Ordner frei:

    • seinen Kalenderordner

    • seinen Dokumente-Ordner

    • seinen persönlichen E-Mail-Ordner Team

  • Der Benutzer Tom Green erzeugt im Ordner Öffentliche Ordner den neuen Ordner VacationCalendar. Er gewährt allen Benutzern die Berechtigung, in diesem Ordner Objekte anzulegen.

  • Der Benutzer Jean Dupont gibt in seinem persönlichen Dokumente-Ordner den Ordner MeetingMinutes frei. Sein persönlicher Dokumente-Ordner ist nicht freigegeben.

Sie sehen folgende Ordner:
