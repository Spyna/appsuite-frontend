E-Mail

5.3. E-Mail-Anhang ansehen oder speichern

Die Dateinamen von E-Mail-Anhängen werden in der Detailansicht angezeigt. Die folgenden Funktionen sind verfügbar:

  • den Anhang im Viewer anzeigen

  • den Anhang in einem neuen Tab öffnen

  • den Anhang herunterladen

  • den Anhang speichern in Drive

Hinweis: Je nach Dateiformat des Anhangs stehen möglicherweise nicht alle Funktionen zur Verfügung.

So verwenden Sie die Funktionen für E-Mail-Anhänge:

  1. Wählen Sie eine E-Mail, die einen Anhang oder mehrere Anhänge enthält.

  2. Klicken Sie in der Detailansicht auf eine der Schaltflächen Ansicht, In Browser öffnen, Download, In Drive speichern.

    Wenn mehrere Anhänge vorhanden sind, wird die Funktion auf alle Anhänge angewendet.

  3. Wenn Sie eine Funktion für einen bestimmten Anhang ausführen wollen, klicken Sie auf das Symbol Anhang . Unterhalb des Symbols erscheinen die Namen des Anhangs. Klicken Sie auf den Namen eines Anhangs. Wählen Sie im Menü eine Funktion.

    Wenn Sie eine Vorschau eines Anhangs sehen wollen, klicken Sie rechts auf das Symbol Kacheln . Die Anhänge werden als Kacheln angezeigt. Klicken Sie auf eine Kachel.

    Wenn Sie die Anhänge auf einer eigenen Seite anzeigen wollen, klicken Sie in der Werkzeugleiste auf Ansicht. Klicken Sie auf All attachments.

So zeigen Sie Anhänge von gesendeten oder empfangenen E-Mails an:

  1. Wählen Sie eine E-Mail, die einen Anhang oder mehrere Anhänge enthält.

    Klicken Sie unterhalb der Ordner auf View all Attachments. In der App Drive zeigt der Ordner My attachments die Anhänge.

  2. Bei Bedarf verwenden Sie die Funktionen von Drive.
