10.2.1. Konversationen anlegen

Sie haben folgende Möglichkeiten:

  • Eine private Konversation mit einem einzelnen Kontakt anlegen

  • Eine Gruppenkonversation mit bestimmten Teilnehmern anlegen

  • Einen öffentlichen Kanal anlegen, dem andere Kontakte beitreten können

So legen Sie eine Konversation an:

  1. Klicken Sie neben Ihrem Anzeigenamen auf das Symbol Start new conversation . Ein Menü öffnet sich.

  2. Je nach Art der gewünschten Konversation verwenden Sie die folgenden Methoden:

    • Um eine Konversation mit einem einzelnen Kontakt anzulegen, wählen Sie Private chat. Das Fenster Start new conversation öffnet sich.

      Wählen Sie einen Kontakt.

      Klicken Sie auf Start conversation.

    • Um eine Konversation mit mehreren Teilnehmern anzulegen, wählen Sie Group chat. Das Fenster Create group chat öffnet sich.

      Bestimmen Sie den Gruppennamen. Optional bestimmen Sie ein Bild, indem Sie das vorhandene Symbol anklicken.

      Fügen Sie Kontakte hinzu.

      Klicken Sie auf Create Chat.

    • Um einen öffentlichen Kanal anzulegen, dem andere Kontakte beitreten können, wählen Sie Public channel. Das Fenster Create new channel öffnet sich.

      Bestimmen Sie den Kanalnamen. Optional bestimmen Sie ein Bild, indem Sie das vorhandene Symbol anklicken.

      Klicken Sie auf Create Chat.

Die neue Konversation wird in der Liste angezeigt.

Tipp: Sie können eine neue Konversation auch anlegen, indem Sie unmittelbar nach dem Öffnen des Chat-Fensters die Symbole im Konversationsfeld verwenden. Sobald Sie in der Liste eine Konversation gewählt haben, werden diese Symbole nicht mehr angezeigt.
