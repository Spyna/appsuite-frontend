第 11 章 以 Guard 加密数据

卫士 is a groupware security component that allows to encrypt E-Mails and files.

  • 加密您和其他用户可外部伙伴的电子邮件通信。

  • 加密您的文件。与其他用户分享加密数据。

  • 使用保安选项以定义加密等级。

  • 加密数据是密码保护的。视配置而定，您能够在忘记密码时重设。
