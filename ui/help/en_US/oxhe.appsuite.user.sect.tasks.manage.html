Managing Tasks

Some of the techniques for organizing tasks require having set up own tasks folders. Information about creating folders can be found in Folders.

The following possibilities exist:

  • Editing tasks

  • Marking tasks as done

  • Moving tasks

  • Deleting tasks

Editing tasks

You can edit a task's data at a later point.

How to edit a task:

Prerequisite: You have the permission to create objects in the folder containing the task.

  1. Select a task in the sidebar.

  2. Click on Edit in the display area. The task's data are displayed.

  3. Edit the data.

  4. Click on Save.
