13.1.3. Searching for objects

There are the following options: You can define the app that should be used for the search:

  • You can search in the selected app.

  • You can open the search window to search within any wanted app.

How to search for objects:

  1. Click the Search icon in the menu bar. The search window opens.

  2. In the search window, select the app that should be used for the search.

  3. Use one of the following methods:

    • In order to use a search term, enter it in the Search input field. The search menu will be displayed.

    • In order to search for objects with a specific property, click in the Search input field. The search menu will be displayed.

  4. Select an entry from the search menu.

    Result: A list of found objects will be displayed in the search window.

  5. In order to enhance your search in the list of found objects, start a further search.

    To reset the result, close the search criterion's button.

How to search for objects in the selected app:

  1. In order to search for E-Mails, go to the folder tree and open the folder that you want to browse.

    In the other apps all folders are browsed by default.

  2. Use one of the following methods:

    • In order to use a search term, enter it in the Search input field in the menu bar. The search menu will be displayed.

    • In order to search for objects with a specific property, click in the Search input field in the menu bar. The search menu will be displayed.

  3. Select an entry from the search menu.

    Result: A list of found objects will be displayed in the search window.

  4. In order to enhance your search in the list of found objects, start a further search.

    To reset the result, close the search criterion's button.
