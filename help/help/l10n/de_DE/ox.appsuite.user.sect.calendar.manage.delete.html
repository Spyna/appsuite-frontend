7.7.9. Termine löschen

Sie können Termine löschen.

So löschen Sie Termine:

Achtung: Wenn Sie einen Termin löschen, ist dieser Termin endgültig gelöscht.

Je nach Konfiguration des Servers können Sie in Ihren privaten Kalendern Termine nur löschen, wenn Sie Organisator des Termins sind.

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In einer Kalenderansicht klicken Sie auf einen Termin. Klicken Sie im Popup auf Löschen.

    • In der Listenansicht wählen Sie einen Termin oder mehrere Termine. Klicken Sie in der Werkzeugleiste auf Löschen.

  2. Bestätigen Sie, dass Sie die Termine löschen möchten.

Ergebnis: Die Termine werden gelöscht.
