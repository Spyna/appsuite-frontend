9.5.1. Envoyer les fichiers comme lien

Vous pouvez envoyer un lien vers un fichier comme pièce jointe d'un courriel à un utilisateur. Vous pouvez envoyer des liens vers plusieurs fichiers à la fois en tant que pièces jointes d'un courriel. Référez-vous également aux indications de Section 9.7, « Fichiers et équipes ».

Pour envoyer un lien vers un fichier comme pièce jointe à un courriel :

  1. Selon la vue choisie, utilisez une des méthodes suivantes :

    • Dans la vue Icônes, cliquez sur un fichier. Dans la fenêtre qui s'ouvre, cliquez sur Plus.... Dans le menu, cliquez sur Envoyer comme lien.

    • Dans la vue Liste, choisissez un fichier dans la barre latérale. Dans la zone d'affichage, cliquez sur Plus.... Dans le menu, cliquez sur Envoyer comme lien.

  2. Dans la page Rédiger un nouveau courriel, remplissez les détails nécessaires pour envoyer le courriel.
