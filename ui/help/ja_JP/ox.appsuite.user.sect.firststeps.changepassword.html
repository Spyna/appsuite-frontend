3.4. パスワードの変更

パスワードを変更する

  1. メニュー バーで、ポータル をクリックします。

  2. ユーザ データ ポータル タイルにある マイ パスワード をクリックします。

  3. 現在のパスワードを入力して、次に、新しいパスワードを 2 回入力します。

  4. パスワードを変更 をクリックします。

注: このタイルは表示されていないことがあります。表示されていない場合、タイルを 追加 できます。
