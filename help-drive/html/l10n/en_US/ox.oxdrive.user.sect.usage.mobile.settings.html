2.2.4. Settings on a mobile device

How to use the OX Drive client settings on a mobile device:

  1. In the OX Drive app, tap the menu icon.

  2. Select Settings.

The following settings are available.

  • Sign out

  • General

  • Photostream

  • Synchronized files

  • Battery

  • Bandwidth

  • Support

  • About

Sign out Signs you out. Stops the synchronization.

General Shows various settings and information.

  • Sync on application start. Sets whether the synchronization is automatically started after the app start.

  • Sync changes automatically. Sets whether the synchronization is automatically started after a change.

  • Limit cache size to. Specifies the space volume that should be reserved for synchronized data.

  • Server URL. Shows the address of the server that is used for the synchronization.

Photostream Sets whether photos taken with the mobile device are transferred to the server.

  • Import photos automatically. Enables or disables Photostream.

  • New photos. Limits the import to photos that were taken after Photostream was enabled.

  • All photos. Transfers all photos on the mobile device and all photos taken from now on.

Synchronized files Specifies the files to be synchronized.

  • All. Synchronizes all files and folders that are located below My files on the groupware server.

  • Favorites. Synchronizes only files and folders that are marked as Favorites on the mobile device.

Battery Specifies the battery status that must be met to allow the synchronization.

  • Sync only while charging. Only synchronizes while loading the batteries.

  • Pause sync while below. Sets whether the synchronization pauses if the battery capacity falls below a certain level.

Bandwidth Determines which type of connection is to be used for synchronizing.

Support Displays this user guide. Warning: If clicking on Reset app, all synchronized data within the app are locally deleted. The data on the server will be kept. In order to use the data on the mobile device again, you have to re-synchronize.

About Shows version notes on OX Drive.
