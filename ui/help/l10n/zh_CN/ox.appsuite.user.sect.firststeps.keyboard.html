3.2. 键盘输入

作为滑鼠的替代您可以使用以下按键：

  • 使用按键组合 [Ctrl]+[F6]，在菜单条、文件夹树、边栏、显示区域之间来回切换。

  • 要输入字段里移动鼠标，使用以下按键：方向键、[Home]、[End]。

  • 要浏览文件夹树或边栏，请使用以下按键：[Page up]、[Page down]、[Space bar]、[Shift]+[Space bar]。

  • 要打开或关闭文件夹树里的一个文件夹，请使用 [Space bar] 或 [Enter]。

  • To select elements in the folder tree, in the sidebar or in the display area, use the cursor keys.

  • 为了选择一个功能、一个输入字段、或一个选择框，请使用以下按键：[Tab]、[Shift]+[Tab]。选择的元素将会高亮。

    您可以使用 [Shift]+[Tab] 反向选择元素。

  • 要执行选择功能，按下 [Enter]。

  • 要启用或停用一个选择框，请按下 [Space bar]。

  • 要切换选择框，请使用方向键。

  • 要关闭一个弹出窗口或取消一个对话窗口，请按 [Esc]。
