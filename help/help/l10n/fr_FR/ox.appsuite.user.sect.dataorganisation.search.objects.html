12.1.3. Rechercher des objets

Pour chercher un objet :

  1. Cliquez sur l'icône Recherche oui/non dans la barre de menu. La fenêtre de recherche s'ouvre.

  2. Dans la barre d'outils, sélectionnez l'application à utiliser pour la recherche.

  3. Cliquez sur le bouton Dossier. Choisissez le dossier que vous souhaitez utiliser pour la recherche.

  4. Utilisez l'une des méthodes suivantes :

    • Pour utiliser un terme à rechercher, saisissez-le dans le champ Chercher. Cliquez sur l'un des éléments du menu de recherche.

    • Pour rechercher des objets ayant une propriété donnée, cliquez dans le champ de saisie Chercher. Cliquez sur l'un des éléments du menu de recherche.

    Résultat : la liste des objets trouvés est affichée dans la fenêtre de recherche.

  5. Vous pouvez affiner davantage le résultat de la recherche :

    • Pour rechercher dans un autre dossier, cliquez sur Dossier.

    • Pour affiner votre recherche sur la liste des objets trouvés, démarrez une recherche supplémentaire.

    • Pour réinitialiser les résultats, fermez le bouton correspondant au critère de recherche.
