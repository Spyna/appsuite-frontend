System Requirements

In order to successfully work with the OX App Suite , your local machine has to meet the following system requirements.

Resolution/screen size. The minimum display resolution is 1024 x 768.

Browser. 

  • Google Chrome, latest version

Browser settings. 

  • Cookies must be activated

  • JavaScript must be activated

  • Pop-up windows must be allowed
