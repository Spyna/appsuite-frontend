10.4.3. Starting a chat with appointment participants

You can communicate with appointment participants by starting a chat with the participants.

How to start a chat with an appointment's participants:

  1. Depending on the selected view, you have the following options in the Calendar:

    • In a calendar view, click on an appointment with multiple participants. In the pop-up, click the More actions icon . Click on Start chat with participants.

    • In the list view, double-click on an appointment with multiple participants. The appointment will be displayed in a window. Click the More actions icon . The Create group chat window opens.

  2. Enter a name for the chat group. You can add or remove participants if required.

    Click on Create chat.
