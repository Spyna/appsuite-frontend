Ordner umbenennen

Sie können Unterordner Ihrer persönlichen Ordner umbenennen. Für andere Ordner benötigen Sie die entsprechenden Berechtigungen.

So benennen Sie einen Ordner um:

  1. Wählen Sie im Ordnerbaum den Ordner, den Sie umbenennen wollen.

  2. Klicken Sie unterhalb des Ordnerbaums auf die Schaltfläche Bearbeiten . Klicken Sie auf Umbenennen.

  3. Bearbeiten Sie den Namen oder geben Sie einen neuen Namen ein. Klicken Sie auf Umbenennen.
