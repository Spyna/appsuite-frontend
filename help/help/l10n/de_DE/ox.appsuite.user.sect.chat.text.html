10.3. Nachrichten senden oder lesen

Sie haben folgende Möglichkeiten:

  • Nachrichten und Dateien in einer Gruppe oder an einen einzelnen Kontakt senden

  • Nachrichten einer Konversationen suchen und lesen

So senden Sie im Chat-Fenster eine Nachricht:

  1. Verwenden Sie eine der folgenden Alternativen:

    • Um eine neue Konversation zu beginnen, klicken Sie auf das Symbol Start new conversation .

      Wählen Sie im Menü die gewünschte Form der Konversation.

    • Um eine vorhandene Konversation weiterzuführen, wählen Sie in der Liste eine Konversation. Die bisher ausgetauschten Nachrichten dieser Konversation werden angezeigt.

  2. Tippen Sie eine Nachricht in das Nachrichtenfeld.

    Um die Nachricht zu senden, drücken Sie [Eingabe]. Ein Symbol rechts neben der Nachricht zeigt den Lesestatus ihrer Nachricht:

    • Ein grauer Haken: Die Nachricht ist auf dem Server angekommen.

    • Zwei graue Haken: Die Nachricht wurde vom Server an alle Empfänger übertragen.

    • Zwei blaue Haken: Die Nachricht wurde von allen Empfängern gelesen.

Optionen:

  • Um innerhalb einer Nachricht einen Zeilenumbruch zu erzeugen, drücken Sie [Umschalt]+[Eingabe].

  • Wenn Sie die Nachricht in einer Gruppe senden, können Sie in der Nachricht einen Kontakt gezielt ansprechen. Dazu tippen Sie [@], dahinter den Namen der Person, dahinter die Nachricht. Der angesprochene Kontakt sieht in der Liste eine rote Markierung am Namen der Gruppe.

  • Um eine Datei zu senden, klicken Sie neben dem Eingabefeld auf das Symbol Anhang. Wählen Sie eine Datei.

  • Um alle bisher gesendeten oder empfangenen Dateien zu sehen, klicken Sie in der Liste auf All files.

So lesen Sie im Chat-Fenster eine Nachricht:

  1. Wählen Sie in der Liste eine Konversation.

    Konversationen mit ungelesene Nachrichten werden folgendermaßen dargestellt:

    • Der Namen des Kontakts oder der Gruppe wird in Fettschrift angezeigt.

    • Eine Zahl nennt die Anzahl ungelesener Nachrichten.

  2. Lesen Sie die Nachrichten der Konversation. Um ältere Nachrichten dieser Konversation zu sehen, blättern Sie mit Hilfe der Symbole am rechten Rand nach oben.

Optionen:

  • Um nach Konversationen zu suchen, geben Sie in das Eingabefeld Search or start a new chat einen Suchbegriff ein.

Benutzeroberfläche:

  • Das Chat-Fenster

Verwandte Themen:

  • Konversationen anlegen

    Mit Konversationen arbeiten

  • Anwesenheitsstatus festlegen
