10.2. 建立文件

您可以建立包含格式的文件、表格與圖片。要建立新文件，請進行以下操作：

  • 在 文字 或 試算表 應用程式中建立文件。您可以使用範本。

  • 在 雲端硬碟 應用程式中建立文件。

如何建立新文件:

  1. 要不使用範本建立新文件，點擊 文字 應用程式中工具列的 新增文字文件。

    要不使用範本建立新試算表，點擊 試算表 應用程式中工具列的 新增試算表。

    要使用範本，點擊 新增自範本 中的範本。

    文件將在 未命名 頁面顯示。

  2. 點選文件列右上角的 未命名。輸入文件的名稱。

  3. 建立與編輯文件內容。要編輯文件，使用跟其他熟悉的 Office 應用程式一樣的技巧。

    注意 文字 與 試算表 功能的詳情。

  4. 要完成，點選文件列右邊的 關閉文件 圖示 。

如何在 雲端硬碟 應用程式中建立新文件:

  1. 執行 雲端硬碟 應用程式。

  2. 在資料夾樹中 開啟 資料夾。

    註: 開啟您有權限建立物件的資料夾。

  3. 點下工具列中的 新增 圖示。點選 新增純文字文件 或 新增試算表。

  4. 點選文件列右上角的 未命名。輸入文件的名稱。

  5. 建立與編輯文件內容。要編輯文件，使用跟其他熟悉的 Office 應用程式一樣的技巧。

    注意 文字 與 試算表 功能的詳情。

  6. 要完成，點選文件列右邊的 關閉文件 圖示 。
