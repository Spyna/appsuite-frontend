8.4.3. Adding participants to a task

How to add participants on the Create task page:

  • Enter the participants' E-Mail addresses, a group's name or a distribution list's name in the input field below Participants. Tips:

    • While entering the E-Mail address, several matching suggestions are displayed. To accept a suggestion, use one of the following methods:

      • Use the scrollbar to browse the list. Click on a suggestion.

      • Use the cursor keys to select a suggestion. Press Enter.

    • To select contacts from a list, open the Select contacts window. To do so, click the Address book icon on the right side of the input field.

    In order to remove a participant, click the icon next to the name.
