13.2. Daten exportieren

Sie können bestimmte Daten in eine Datei exportieren, um diese Daten in anderen Anwendungen zu nutzen. Sie können die folgenden Daten exportieren:

  • Kontakte in den folgenden Formaten:

    • CSV

    • vCard

    • hCard

  • Termine in den folgenden Formaten:

    • iCalendar

  • Aufgaben in den folgenden Formaten:

    • iCalenar

Diese Formate sind standardisiert und können von vielen anderen Anwendungen importiert werden. Sie haben die folgenden Möglichkeiten:

  • Alle Daten eines Ordners exportieren.
