5.5.15. Modifier tous les messages d'un fil de discussion

Vous pouvez lancer les actions suivantes pour tous les courriels d'un fil de discussion d'un seul coup :

  • transmettre des courriels

  • déplacer ou copier des courriels dans un autre dossier

  • marquer des courriels comme lus ou non lus

  • catégoriser des courriels avec des étiquettes

  • supprimer des courriels

Pour lancer une même action pour tous les message d'un fil de discussion :

  1. Sélectionnez le message courant dans le fil de discussion dans la barre latérale. Il apparaît au sommet du fil de discussion.

  2. Cliquez sur Fil de discussion entier en haut du message dans la zone d'affichage. Choisissez une action dans le menu.
