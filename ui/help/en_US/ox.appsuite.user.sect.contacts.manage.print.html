Printing contacts

In order to print contacts you can:

  • print a list of contacts

How to print a list of contacts:

  1. Open an address book in the folder tree.

  2. Click on Print in the display area.

  3. If required, change the printer settings. Click on Print Using System Dialog... at the bottom left side.
