E-Mails drucken

Um E-Mails zu drucken, haben Sie folgende Möglichkeiten:

  • den Inhalt einer E-Mail drucken

So drucken Sie den Inhalt einer E-Mail:

  1. Wählen Sie in der Seitenleiste eine E-Mail. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Drucken

  2. Falls erforderlich, ändern Sie die Druckereinstellungen. Klicken Sie links unten auf Über das Systemdialogfeld drucken....
