12.1.1. Die Bestandteile des Suchfensters

Das Suchfenster enthält die folgenden Bestandteile.

  • Eine Werkzeugleiste mit App-Schaltflächen. Anklicken einer App-Schaltfläche bestimmt, in welcher App gesucht werden soll.

  • Eingabefeld Suchen. Sobald Sie einen Suchbegriff eingeben, werden die Einträge im Suchmenü entsprechend angepasst.

  • Die Schaltfläche Ordner. Sie zeigt den Ordner, in dem gesucht wird. Um in einem anderen Ordner zu suchen, klicken Sie auf die Schaltfläche.

  • Ein Suchmenü mit App-spezifischen Einträgen. Anklicken eines Eintrags startet eine Suche.

  • Symbol Suchen . Startet eine Suche.

    Sobald eine Suche ausgeführt wurde, zeigt das Suchfenster weitere Elemente:

    • Eine oder mehrere Schaltflächen mit den Suchkriterien, die Sie für die Suche verwendet haben. Einige Schaltflächen enthalten ein Auswahlmenü, mit dem Sie die Suchkriterien ändern können.

      Um ein Suchergebnis zurückzusetzen, schließen Sie die Schaltfläche eines Suchkriteriums.

    • Die Liste der gefundenen Objekte. Klicken Sie auf ein Objekt, um seine Daten im Popup anzuzeigen.

      Sie können in der Liste suchen, indem Sie eine neue Suche starten.
