6.7.9. Kontakte drucken

Sie können die Daten eines Kontaks oder mehrerer Kontakte drucken. Sie können zwischen verschiedenen Drucklayouts wählen.

So drucken Sie die Daten von Kontakten:

  1. Öffnen Sie im Ordnerbaum ein Adressbuch. Wählen Sie in der Liste einen Kontakt oder mehrere Kontakte.

  2. Klicken Sie in der Werkzeugleiste auf Aktionen. Klicken Sie im Menü auf Drucken. Das Fenster Drucklayout wählen wird geöffnet.

  3. Wählen Sie ein Drucklayout. Klicken Sie auf Drucken.

    Vervollständigen Sie die Schritte zum Starten des Druckvorgangs.

  4. Schließen Sie das Fenster mit der Druckvorschau.
