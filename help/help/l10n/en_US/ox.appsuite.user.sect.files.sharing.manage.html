9.6.3. Managing shared data

In order to organize own shares, you can do the following:

  • change the permission type for existing shares, e.g. read or edit permission

  • add other users or external partners to existing shares

  • remove other users or external partners from shares

  • withdraw existing shares

How to manage your shares:

  1. In the Drive app, open Shared files in the folder tree.

    Click on the Shared by me folder. The folders and files shared by you are displayed in the detail view.

  2. Use the following methods:

    • In order to change the permission type, click on Options next to a shared folder or a shared file. Select an entry.

    • In order to withdraw the share, click the Remove icon next to a shared folder or file.
