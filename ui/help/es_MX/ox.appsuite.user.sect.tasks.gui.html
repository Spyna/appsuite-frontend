11.1. Componentes de Tareas

Puede encontrar una introducción a la interfaz de usuario en El interfaz de usuario.

Barra de herramientas Contiene los siguientes iconos:

  • Icono Crear tarea nueva . Crea una nueva tarea.

  • Icono Cambiar carpeta . Abre o cierra el árbol de carpetas.

  • Icono Cambiar búsqueda . Abre un campo de texto para buscar tareas con un asunto concreto.

Árbol de carpetas Muestra las carpetas de tareas. Permite navegar por la estructura de carpetas. Para abrir el árbol de carpetas, presione el icono Cambiar carpeta en la barra de herramientas. Las siguientes funciones están disponibles:

  • Si presiona en una carpeta, se muestran las tareas.

  • Los iconos Menú añadir carpeta y Acciones de carpeta situados bajo las carpetas, ofrecen funciones para organizar datos e intercambiar datos.

Barra lateral Muestra la lista de tareas de la carpeta seleccionada. Se muestran el asunto, estado y fecha de vencimiento, y el progreso. Las siguientes funciones están disponibles:

  • Si presiona sobre una tarea, se muestran sus datos en el área de visualización.

  • Si presiona en el icono Seleccionar que hay debajo de la lista, se mostrará una casilla de verificación junto a cada tarea. De este modo podrá seleccionar varias tareas.

  • El texto que hay bajo la lista muestra el nombre de la carpeta seleccionada. Si presiona sobre el nombre, se abrirá el árbol de carpetas. El número corresponde a la cantidad de objetos que hay en la carpeta seleccionada.

  • Si presiona el icono Ordenar que hay bajo la lista, se abrirá un menú que permite ordenar las tareas. También puede definir en este menú si se muestran las tareas finalizadas.

Área de visualización Muestra los datos de la tarea y las funciones disponibles:

  • Botones: Editar , Borrar , Completada , Cambiar fecha de vencimiento

    Si se presiona en Más, se abre un menú con las siguientes funciones: Mover , Cambiar estado de confirmación , Imprimir .

  • Asunto

  • Fecha de vencimiento, si se ha indicado

  • Estado y progreso

  • Icono de importancia

  • Si la tarea tiene adjuntos, los nombres de los adjuntos se muestran.

  • Descripción de la tarea, si está disponible

  • Si la tarea es repetitiva, se mostrarán los parámetros de repetición.

  • Fecha de inicio de tarea, si está disponible
