9.1.4. Le barra di navigazione di File

Il percorso di navigazione è posizionato sotto la barra degli strumenti.

Funzioni

  • Il percorso di navigazione. Mostra il percorso alla cartella aperta. Per aprire una cartella superiore, fare clic su una voce di percorso.

  • Il pulsante Seleziona. Contiene funzioni per selezionare o filtrare oggetti nell'area di visualizzazione:

    • selezionare tutti i file e le cartelle, selezionare tutti i file, cancellare la selezione

    • visualizzare solo determinati tipi di file, visualizzare tutti i tipi di file

  • Il pulsante Ordina per. Ordina i file nell'area di visualizzazione in base a vari criteri.
