2.2.3. Utiliser d'autres fonctionnalités

Sur les appareils mobiles, le client OX Drive offre les fonctionnalités suivantes.

  • Sélectionner un ou plusieurs dossiers. Déplacer ou supprimer les dossiers sélectionnés.

  • Créer un nouveau dossier.

  • Chercher des fichiers ou des dossiers.

  • Trier les fichiers ou les dossiers selon certains critères.

  • Filtrer des fichiers dans un dossier. Les critères de filtrage sont : tous les fichiers, fichiers synchronisés, fichiers non synchronisés.

Pour utiliser les fonctionnalités de OX Drive sur un appareil mobile :

  1. Dans l'app OX Drive, touchez l'icône de menu.

  2. Choisissez Fichiers.

  3. Choisissez une fonction dans la barre de fonctionnalités.

    Selon l'appareil mobile, la barre de fonctionnalités peut figurer à différents endroits : sur les appareils sous Android, en haut de l'écran, sur les appareils sous iOS, en bas.

    Selon l'appareil mobile, les fonctionnalités sont arrangées différemment dans la barre de fonctionnalités.
