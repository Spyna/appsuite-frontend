5.6.4. Enregistrer tous les destinataires d'un message en tant que liste de diffusion.

Si un message s'adresse à plusieurs destinataires, vous pouvez les enregistrer tous dans une nouvelle liste de diffusion.

Pour enregistrer les destinataires d'un courrier électronique en tant que liste de diffusion :

  1. Choisissez un courriel dans la barre latérale.

  2. Cliquez sur Tous les destinataires dans la zone d'affichage.

    Remarque : ce bouton ne s'affiche que si le message comporte plusieurs destinataires.

  3. Choisissez Enregistrer comme liste de diffusion dans le menu.

  4. Complétez les champs pour créer une liste de diffusion.
