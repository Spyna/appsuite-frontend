Organiser les données

11.4. Abonnements

Les abonnements vous permettent d'utiliser les données de vos réseaux sociaux dans le collecticiel. Les options suivantes existent :

  • Importer des contacts en provenance de réseaux sociaux en vous abonnant à ces données.

  • Gardez une vue d'ensemble sur les données auxquelles vous êtes abonné en les gérant sur une page d'aperçu commune.
