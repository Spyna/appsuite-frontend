Managing E-Mail messages

Some of the techniques for organizing E-Mails require having set up own E-Mail folders. Information about creating folders can be found in Folders.

The following possibilities exist:

  • Moving E-Mails

  • Copying E-Mails

  • Marking E-Mails as read or unread

  • Categorizing E-Mails with labels

  • Showing the E-Mail source

  • Enabling E-Mail reminder

  • Saving E-Mails

  • Importing E-Mails

  • Working with E-Mail drafts

  • Deleting E-Mail messages

  • Editing multiple E-Mails at once

  • Editing all E-Mails of an E-Mail thread
