Ficheros

9.8. Ficheros y equipos

Puede compartir sus ficheros con colaboradores internos. Dependiendo de los requerimientos, existen diferentes métodos.

  • En Ficheros públicos y Ficheros compartidos se pueden encontrar carpetas de ficheros compartidas por otros usuarios.

  • Para hacer que los ficheros adicionales estén disponibles para usuarios internos o colaboradores externos, proceda como se indica a continuación:

    • Cree una nueva carpeta personal o pública y copie o mueva los ficheros requeridos a esta carpeta.

    • Comparta esta carpeta.

    También puede compartir una carpeta existente.
