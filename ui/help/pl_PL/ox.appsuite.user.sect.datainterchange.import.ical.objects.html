13.1.1.1. Które obiekty iCal można importować?

Opis wszystkich obiektów iCal można znaleźć w dokumencie RFC2445.

Poniższa tabela przedstawia, które obiekty iCal są importowane, a które nie. Informacje w kolumnach „Spotkania” i „Zadania” mają następujące znaczenie:

  • „X” oznacza, że obiekt jest importowany. Wartość obiektu jest ustalona w dokumencie RFC2445.

  • Liczba (na przykład „255”) oznacza, że obiekt jest importowany, a podana liczba wskazuje maksymalną liczbę znaków.

  • Wpis „bez ograniczeń” oznacza, że obiekt jest importowany bez maksymalnej liczby znaków.

  • Wpis „-” oznacza, że obiekt nie jest importowany.
