5.5.12. Importare i messaggi di posta

È possibile importare un messaggio di posta disponibile nel formato EML.

Come importare un messaggio di posta:

  1. Aprire la cartella di posta nella quale si desidera importare il messaggio.

  2. Trascinare il file EML da un navigatore di file o dal desktop alla finestra dell'applicazione Posta elettronica. Rilasciare il file EML nell'elenco dell'area di visualizzazione.
