Exportation des données

12.2.1. Exporter des données depuis un dossier

Vous pouvez exporter les objets suivants :

  • Les contacts et listes de distributions d'un dossier de contacts privé ou public.

  • Les rendez-vous d'un dossier d'agenda privé ou public.

  • Les tâches d'un dossier de tâches privé ou public.

Pour exporter les objets d'un dossier :

  1. Dans l'arborescence des dossiers, sélectionnez un dossier privé ou public.

  2. Cliquez sur l'icône Actions sur le dossier à côté du nom du dossier. Choisissez Exporter.

  3. Choisissez un type de format.

    • Il vous est possible d'exporter, en même temps que les contacts, des listes de distribution. Pour cela, activez Inclure les listes de distribution.

    • Remarque : certaines applications ne peuvent pas importer des contacts avec des listes de distributions. Dans ce type d'application, l'importation de contacts avec des listes de distributions peut engendrer des erreurs.

    Cliquez sur Exporter.
