Objekte drucken

Wenn Sie Objekte drucken, können Sie verschiedene Druckeinstellungen verwenden, zum Beispiel Seitenformat, Druckränder oder Auswahl eines installierten Druckers. Das Fenster Drucken zeigt auf der linken Seite die Druckeinstellungen, auf der rechten Seite eine Druckvorschau.

So verwenden Sie die Druckeinstellungen:

Voraussetzung: Das Fenster Drucken ist geöffnet.

  1. Um einen installierten Drucker auszuwählen, klicken Sie in Ziel auf die Schaltfläche Ändern.

  2. Um festzulegen, welche Seiten gedruckt werden sollen, ändern Sie die Einstellungen in Seiten.

  3. Bestimmen Sie in Kopien die Anzahl der Ausdrucke. Bestimmen Sie in Layout die Seitenorientierung. Legen Sie in Farbe fest, ob farbige Dokumente in Farbe oder in Graustufen gedruckt werden.

  4. Bestimmen Sie in Ränder die Größe der Seitenränder. Um benutzerdefinierte Werte einzustellen, führen Sie folgende Schritte aus:

    1. Wählen Sie in Ränder die Einstellung Benutzerdefiniert.

    2. Ziehen Sie in der Druckvorschau die Ränder auf die gewünschte Position. Alternativ klicken Sie auf einen Wert, um eine bestimmte Zahl einzugeben.

  5. Bestimmen Sie in Optionen, ob Kopf-und Fußzeilen gedruckt werden sollen.
