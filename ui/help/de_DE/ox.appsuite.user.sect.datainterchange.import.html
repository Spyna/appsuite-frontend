13.1. Daten importieren

Mit Hilfe der Importfunktion können Sie Daten importieren, die mit Hilfe von anderen Anwendungen erstellt wurden. Sie haben die folgenden Möglichkeiten:

  • Termine und Aufgaben im iCal-Format importieren

  • Kontakte im vCard-Format importieren

  • Kontakte im CSV-Format importieren
