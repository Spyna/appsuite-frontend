7.6.3. Google Kalender abonnieren

Sie können Ihren externen Google-Kalender abonnieren. Dabei gilt:

  • Änderungen, die Sie in der Groupware in dem abonnierten Kalender vornehmen, werden in den externen Google-Kalender nicht übertragen.

  • Änderungen, die in Ihrem externen Google-Kalender stattfinden, können Sie in die Groupware übernehmen, indem Sie den abonnierten Kalender aktualisieren.

So abonnieren Sie Ihren Google-Kalender:

  1. Klicken Sie innerhalb der Ordneransicht auf Neuen Kalender hinzufügen. Ein Menü öffnet sich.

  2. Klicken Sie im Menü auf Google-Kalender. Ein neues Browserfenster öffnet sich. Erlauben Sie den Zugriff auf Ihren Google-Kalender.

    Die Termine werden in einen neuen Kalender importiert. Sobald der Import beendet ist, wird das Fenster Kalender abonnieren geschlossen.

  3. Um die abonnierten Daten zu aktualisieren, klicken Sie neben dem Kalender auf das Symbol Aktionen . Klicken Sie im Menü auf Diesen Kalender aktualisieren.
