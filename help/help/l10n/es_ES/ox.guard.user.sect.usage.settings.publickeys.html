12.6.3. Administración de claves públicas

Existen las siguientes opciones:

  • mostrar claves públicas

  • subir claves públicas

  • eliminar una clave pública propia

Cómo mostrar claves públicas:

  1. Marque la casilla de verificación Mostrar ajustes avanzados bajo Avanzado.

    Pulse en Claves públicas bajo Claves. Se abrirá la ventana Claves públicas.

    Se muestran las claves públicas compartidas por usted u otros usuarios. Si se muestra la clave pública de un usuario en esta lista, puede suponer que este usuario puede descifrar los correos electrónicos que envía cifrados a este usuario.

  2. Para mostrar detalles de una clave, pulse en el icono Detalles que hay junto a una clave. Se abrirá la ventana Detalles de claves públicas.

    Puede compartir sus propias claves públicas marcando Compartir clave. Otros usuarios pueden ver y descargar sus claves compartidas.

Cómo subir una clave pública de un colaborador externo en la ventana Claves públicas:

  • Pulse el icono Añadir . Seleccione un fichero que contenga una clave pública.

Cómo eliminar la propia clave pública en la ventana Claves públicas:

  • Pulse el icono Borrar que hay junto a una clave pública.

    Confirme que desea eliminar la clave pública.
