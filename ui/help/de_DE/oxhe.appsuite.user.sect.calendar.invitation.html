Termineinladung beantworten

Wenn ein Benutzer Sie als Teilnehmer zu einem Termin hinzufügt, informiert Sie die Notification area über diesen Termin. Sie können Ihre Teilnahme an dem Termin zusagen, vorläufig zusagen oder ablehnen.

So beantworten Sie eine Termineinladung:

  1. Klicken Sie in der Menüleiste auf das Symbol Unread badge. Die Notification area wird angezeigt.

  2. Klicken Sie unter Invitations auf Accept/Decline.

  3. Geben Sie im Fenster Change confirmation status eine Anmerkung ein. Klicken Sie auf eine der Schaltflächen Ablehnen, Vorläufig, Akzeptieren.

Sie können den Status Ihrer Terminbestätigung später wieder ändern.
