8.1.3. The Tasks folder tree

Displays the tasks folders.

Content

  • My tasks. Contains your personal tasks.

  • Public tasks. Contains tasks shared with all users.

  • Shared tasks. Contains tasks shared with you by other users.

Depending on the groupware configuration, the content can differ from this information.

Functions

  • If clicking on a folder, its tasks are displayed.

  • The Actions icon next to the selected folder offers functions for organising data and for exchanging data.

  • The Add new folder button allows creating a new private or public folder.
