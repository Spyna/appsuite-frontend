12.1.3. Objekte suchen

So suchen Sie nach Objekten:

  1. Klicken Sie in der Menüleiste auf das Symbol Suchen . Das Suchfenster öffnet sich.

  2. Wählen Sie in der Werkzeugleiste die App, in der Sie suchen wollen.

  3. Klicken Sie auf die Schaltfläche Ordner. Wählen Sie den Ordner, in dem Sie suchen wollen.

  4. Verwenden Sie eine dieser Methoden:

    • Wenn Sie einen Suchbegriff verwenden möchten, geben Sie in das Eingabefeld Suchen den Suchbegriff ein. Klicken Sie im Suchmenü auf einen Eintrag.

    • Wenn Sie nach Objekten mit einem bestimmten Objektmerkmal suchen möchten, klicken Sie in das Eingabefeld Suchen. Wählen Sie im Suchmenü einen Eintrag.

    Ergebnis: Eine Liste der gefundenen Objekte wird im Suchfenster angezeigt.

  5. Bei Bedarf können Sie das Suchergebnis weiter anpassen:

    • Um in einem anderen Ordner zu suchen, klicken Sie auf Ordner.

    • Um in der Liste der gefundenen Objekte weiter zu suchen, starten Sie eine weitere Suche.

    • Um das Ergebnis zurück zu setzen, schließen Sie die Schaltfläche des Suchkriteriums.
