7.6.5. ファイルから予定表をインポートする

他のアプリケーションで作成した予定表をインポートできます。インポートしたい予定表は、iCalファイルとして利用できなければなりません。

ファイルから予定表をインポートする：

  1. フォルダツリーで、新規予定表を追加をクリックします。メニューを表示します。

  2. メニューからファイルのアップロードを選択します。ファイルからインポートウィンドウが表示されます。

    目的のファイルを選択します。インポート をクリックします。
