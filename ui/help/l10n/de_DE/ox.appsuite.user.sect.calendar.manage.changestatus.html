7.5.3. Terminbestätigung ändern

Sie können Ihre Terminbestätigung nachträglich ändern. Je nach Konfiguration des Servers können Sie in Ihren privaten Kalendern eine Terminbestätigung nur ändern, wenn Sie Teilnehmer des Termins sind.

So ändern Sie Ihre Terminbestätigung:

  1. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Tag, Arbeitswoche, Woche oder Monat klicken Sie auf einen Termin. Klicken Sie im Popup auf Status ändern.

    • In der Ansicht Liste wählen Sie in der Seitenleiste einen Termin. Klicken Sie im Anzeigebereich auf Status ändern.

  2. Geben Sie im Fenster Bestätigungsstatus ändern eine Anmerkung ein. Klicken Sie auf eine der Schaltflächen Ablehnen, Vorläufig, Akzeptieren.
