10.4.3. Mit Terminteilnehmern einen Chat starten

Sie können mit Teilnehmern eines Termins kommunizieren, indem Sie einen Chat mit den Teilnehmern starten.

So starten Sie einen Chat mit den Teilnehmern eines Termins:

  1. In der App Kalender verwenden Sie je nach Ansicht eine der folgenden Methoden:

    • In einer Kalenderansicht klicken Sie auf einen Termin, der mehrere Teilnehmer hat. Klicken Sie im Popup auf das Symbol Weitere Aktionen . Klicken Sie auf Chat mit Teilnehmern starten.

    • In der Listenansicht doppelklicken Sie auf einen Termin, der mehrere Teilnehmer hat. Der Termin wird in einem Fenster angezeigt. Klicken Sie auf das Symbol Weitere Aktionen . Das Fenster Gruppenchat erstellen öffnet sich.

  2. Bestimmen Sie den Namen der Chatgruppe. Bei Bedarf können Sie Teilnehmer hinzufügen oder entfernen.

    Klicken Sie auf Chat erstellen.

Benutzeroberfläche:

  • Das Kalender-Popup

Verwandte Themen:

  • Nachrichten lesen, senden, beantworten

Übergeordnetes Thema: Mit Nachrichten arbeiten
