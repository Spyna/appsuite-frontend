使用 WebDAV 存取檔案

9.7.1. 在 Linux 下存取 WebDAV

如何在 Linux 下設定 WebDAV 存取：

  1. 打開 KDE Konqueror 或類似的瀏覽器。

  2. 在瀏覽器中輸入以下地址： webdav://<address>/servlet/webdav.infostore 其中，<address>必須要用組件伺服器的 IP 或網址代替。

  3. 輸入用於登錄組件伺服器的用戶名和密碼。

結果： Result: The Drive 應用程式檔案顯示在瀏覽器中。
