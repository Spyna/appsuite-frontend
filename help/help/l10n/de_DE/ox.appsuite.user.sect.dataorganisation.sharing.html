Datenorganisation

11.3. Freigaben

Mit Hilfe von Freigaben können Sie mit anderen Benutzern oder mit externen Partnern zusammenarbeiten, indem Sie Groupware Daten wie E-Mails, Kontakte, Termine und Aufgaben zum Lesen oder Bearbeiten zur Verfügung stellen. Sie haben die folgenden Möglichkeiten.

  • Daten für andere Benutzer oder für externe Partner zum Lesen oder Bearbeiten freigeben.

  • Auf Daten zugreifen, die andere Benutzer für Sie freigeben.

  • Verwalten Sie Ihre Freigaben, indem Sie neue Benutzer hinzufügen, Benutzer entfernen oder Berechtigungen ändern.

Hinweise:

  • Mit den nachfolgend beschriebenen Methoden können Sie E-Mails, Kontakte, Termine oder Aufgaben freigeben. E-Mails können Sie nur für interne Benutzer freigeben, nicht für externe Partner. Um Dateien und Ordner in der App Drive freizugeben, verwenden Sie die Methoden, die in 9.6: Freigaben beschrieben sind.

  • Als Ergebnis einer Freigabe erhalten interne Benutzer oder externer Partner bestimmte Berechtigungen an den freigegebenen Daten. Informationen über Berechtigungen finden Sie in 11.2: Berechtigungen.
