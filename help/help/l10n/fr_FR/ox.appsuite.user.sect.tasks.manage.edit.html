8.7.1. Modifier des tâches

Vous pouvez modifier ultérieurement les détails d'une tâche.

Pour modifier une tâche :

Pré-requis : vous devez avoir le droit de créer des objets dans le dossier contenant la tâche.

  1. Choisissez une tâche dans la liste.

  2. Cliquez sur Modifier dans la barre d'outils. Les détails de la tâche s'affichent.

  3. Modifiez les données.

    Vous trouverez une description des champs de saisie dans La fenêtre d'édition de tâche.

  4. Cliquez sur Enregistrer.
