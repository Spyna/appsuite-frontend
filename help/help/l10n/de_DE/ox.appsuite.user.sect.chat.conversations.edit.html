10.2.2. Konversationen bearbeiten

Gruppenkonversationen und öffentliche Kanäle können von allen Teilnehmern bearbeitet werden. Sie haben folgende Möglichkeiten:

  • Das Bild ändern

  • Den Namen ändern

  • Bei einer Gruppenkonversation: Teilnehmer hinzufügen oder löschen

So bearbeiten Sie eine Gruppenkonversation oder einen öffentlichen Kanal:

  1. Wählen Sie in der Liste eine Gruppenkonversation oder einen öffentlichen Kanal.

  2. Klicken Sie rechts oben auf das Symbol More actions .

    Wählen Sie Edit chat. Ein Bearbeitungsfenster öffnet sich.

  3. Bei Bedarf können Sie Folgendes ändern:

    • Ändern Sie das Bild, indem Sie das Bild anklicken.

    • Ändern Sie den Namen.

    • Bei einer Gruppenkonversation: Fügen Sie Teilnehmer hinzu oder löschen Sie Teilnehmer.

  4. Klicken Sie auf Edit Chat.
