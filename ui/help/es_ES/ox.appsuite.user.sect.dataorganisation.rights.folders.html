Permisos para carpetas ya existentes

Usted tiene los siguientes permisos para carpetas ya existentes:

  • Usted es el propietario de sus carpetas personales.

  • Cada usuario es un invitado en la carpeta Pública.

  • Cada usuario es un invitado en la carpeta Libreta de direcciones global.

  • Cada usuario es un invitado en el Almacén de información y en la carpeta Almacén de usuario.

  • Cada usuario es un invitado en la carpeta Almacén de información público.
