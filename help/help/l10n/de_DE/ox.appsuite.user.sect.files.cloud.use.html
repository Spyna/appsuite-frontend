9.8.2. Cloud Speicher verwenden

Sie haben die folgenden Möglichkeiten:

  • Sie können mit Hilfe des Ordnerbaums auf die Inhalte Ihrer Cloud Speicher zugreifen. Je nach Inhalt Ihres Cloud Speichers können Sie die Funktionen der Werkzeugleiste verwenden, zum Beispiel um Fotos anzusehen oder Daten zu kopieren.

  • Je nach Inhalt können Sie die Funktionen der Werkzeugleiste verwenden, zum Beispiel um Fotos anzuzeigen oder Dateien zu kopieren.
