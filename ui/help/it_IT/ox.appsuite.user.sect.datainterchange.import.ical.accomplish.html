13.1.1.3. Eseguire un'importazione iCal

Notare ciò che segue durante l'importazione di dati iCal.

  • Assicurarsi che il file da importare contenga dati iCal corretti.

Come importare appuntamenti o attività nel formato iCal:

  1. Avviare l'applicazione Calendario o Attività .

  2. Selezionare la cartella per l'importazione di appuntamenti e attività nell'albero delle cartelle.

  3. Fare clic sull'icona Modifica cartella sotto l'albero delle cartelle. Fare clic su Importa.

  4. Fare clic su Seleziona file nella finestra Importa in. Selezionare un file in formato iCal.

  5. Fare clic su Importa.

Risultato: gli appuntamenti o le attività sono aggiunti alla cartella.
