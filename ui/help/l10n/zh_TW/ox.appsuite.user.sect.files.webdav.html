9.6. 使用 WebDAV 存取檔案

使用 WebDAV，您可以如存取本機磁碟中的檔案一樣存取 雲端硬碟 應用程式的檔案。與從組件 GUI 訪問相比，WebDAV 有優點也有缺點：

  • 優點是使用檔案瀏覽器如 Windows 檔案總管快速直接訪問。您無需登錄到組件 GUI。

  • 缺點是您只能看到當前文檔版本。不會顯示文檔的歷史和其他信息，例如註釋。

警告： 用 WebDav 刪除檔案時，將刪除所有版本的檔案，而不僅是當前版本。

在使用檔案瀏覽器，如 Windows 檔案總管存取 雲端硬碟 應用程式檔案前，您必須設定 WebDAV 存取。視系統軟體而定，操作方式有所不同:

  • 在 Linux 下存取 WebDAV

  • 在 Windows XP 下存取 WebDAV

  • 在 Windows Vista 下存取 WebDAV

  • 在 Windows 7 下存取 WebDAV
