8.6.6. Impresión de tareas

Para imprimir tareas puede hacer lo siguiente:

  • imprimir los datos de una tarea

Cómo imprimir los datos de una tarea:

  1. Seleccione una tarea de la lista.

  2. Pulse el icono Acciones de la barra de herramientas. Pulse en Imprimir en el menú. Se abrirá una ventana con una vista previa de impresión.

  3. Si es necesario, cambie los ajustes de la impresora. Pulse el botón Imprimir.

  4. Cierre la ventana de vista previa de impresión.

Consejo: También puede imprimir los datos de múltiples tareas al mismo tiempo.
