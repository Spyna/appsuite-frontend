7.5.7. Edición de varias citas de una vez

Puede ejecutar las siguientes funciones para varias citas a la vez:

  • Mover citas a otra carpeta

  • Impresión de varias citas

Cómo ejecutar una función para varias citas a la vez:

  1. En la vista Lista, pulse en el icono Seleccionar situado debajo la barra lateral. Se mostrará una casilla de verificación junto a cada cita de la barra lateral.

  2. Marque las casillas de verificación de al menos dos citas.

    También puede usar las funciones de selección múltiple de su sistema.

    Para seleccionar todas las citas de la carpeta actualmente seleccionada, marque la casilla de verificación Seleccionar todo situada en la parte inferior izquierda de la barra lateral.

  3. Pulse en una función del área de visualización.
