实现 iCal 导入

导入 iCal 数据是请注意以下。

  • 确保要导入的文件包含正确的 iCal 数据。

如何导入 iCal 格式的约会或任务：

  1. 执行 日历 oder 任务 应用程序。

  2. 在文件夹树里选取导入约会或任务文件夹。

  3. 在文件夹树下面单击编辑文件夹图标。单击导入。

  4. 单击选取文件（在导入到窗口里）。选择一个 iCal 格式的文件。

  5. 单击导入。

结果：约会和任务已加入到文件夹。
