Working with Messages

10.3.1. Sending, reading or replying to messages

In the Chat window, there are the following options:

  • send messages and files to a group or single contact

  • search for and read a chat's messages

  • Reply to an earlier message. The original message will be quoted in your reply.

How to send a message:

  1. Use one of the following methods:

    • In order to start a new chat, click the New icon .

      Select a conversation type in the menu.

    • To proceed with an existing chat, select a chat from the list. The chat's messages exchanged so far will be displayed.

  2. Enter a message in the input field.

    To send the message, press [Enter] or click the Send icon .

    For private and group chats, an icon next to the message indicates your message's read status:

    • A gray check mark: The message has been arrived on the server.

    • Two gray check marks: The message has been transferred by the server to all recipients.

    • Two blue check marks: The message has been read by all recipients.

Options:

  • To create a line break within a message, press [Shift]+[Enter].

  • If you send the message to a group, you can address a contact specifically. To do so, enter [@] followed by the person's name and the message. The contact addressed will see a red marker at the group name in the list.

  • In order to send a file, click the Upload file icon next to the input field. Select one or several files.

    You can also drag one file or several files from a file browser or the desktop to the chat field.

  • To view all previous sent or received files, click on All files in the list.

How to read a message:

  1. Select a conversation from the list.

    Chats with unread messages will be displayed as follows:

    • The name of the contact or group will be displayed in bold letters.

    • A figure indicates the number of unread messages.

  2. Read the conversation's messages. To view earlier messages in this conversation, use the icons on the right border to scroll up.

Options:

  • To search for chats, enter a search term in the Search or start a new chat input field.

How to reply to an earlier message:

  1. In the chat field, go to a message received earlier.

    Click the Message actions iconnext to the message. Select Reply.

    The message and its sender will be copied to the message field.

  2. Enter your reply in the message field.

    To send the reply, press [Enter] or click the Send icon .
