11.1.7. Ordner verschieben

Sie können Unterordner Ihrer persönlichen Ordner verschieben. Für andere Ordner benötigen Sie die entsprechenden Berechtigungen.

So verschieben Sie einen Ordner:

  1. Wählen Sie im Ordnerbaum den Ordner, den Sie verschieben wollen.

  2. Klicken Sie neben dem Ordnernamen auf das Symbol Ordnerspezifische Aktionen . Klicken Sie auf Verschieben.

  3. Wählen Sie im Fenster Ordner verschieben einen Ordner. Bei Bedarf legen Sie einen neuen Ordner an, indem Sie auf Create folder klicken.

  4. Klicken Sie auf Verschieben.
