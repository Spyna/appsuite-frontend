6.4.2. Kontakt aus vCard hinzufügen

Sie können einen Kontakt aus einer vCard-Anlage einer E-Mail hinzufügen. Eine vCard-Anlage hat standardmäßig die Namenserweiterung vcf.

So fügen Sie einen Kontakt aus der vCard-Anlage einer E-Mail an:

  1. In der App E-Mail zeigen Sie eine E-Mail an, die eine vCard-Anlage enthält.

  2. Klicken Sie im Anzeigebereich auf den Namen der Anlage. Klicken Sie im Menü auf Zu Adressbuch hinzufügen.
