5.6.2. E-Mail-Ordner abonnieren

Um freigegebene E-Mail-Ordner anderer Benutzer zu sehen, müssen Sie diese Ordner abonnieren.

So abonnieren Sie freigegebene E-Mail-Ordner:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Systemmenü . Klicken Sie im Menü auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf E-Mail.

  3. Klicken Sie im Anzeigebereich unten auf Abonnement ändern.

  4. Im Fenster IMAP-Ordner abonnieren aktivieren Sie die Kontrollfelder der Ordner, die Sie abonnieren wollen. Klicken Sie auf Speichern.
