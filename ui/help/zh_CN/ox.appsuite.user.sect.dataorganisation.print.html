12.2. 打印对象

在打印对象是有不同的打印设定可用，例如：纸张大小、边框、或选择已安装打印机。打印窗口在左边显示打印机设定，和打印预览在右边显示。

如何使用打印机设置：

前提：打印窗口为显示。

  1. 要选择一个已安装打印机，单击更改按钮（在目标里）。

  2. 要定义打印的页，更改页的设定。

  3. 在复制份数里，定义打印的数量。在布局里，定义页的定向。在颜色里，您可以设置彩色文档是以彩色还是灰阶打印。

  4. 在边框里，定义边框大小。要自定义数值，请进行以下动作：

    1. 选择自定义。设置（在边框里）。

    2. 在打印预览里，拖动边框到需要的位置。您也可以单击一个数值后输入需要的值。

  5. 在选项里，定义要否打印页头和页尾。
