8.7. Managing Tasks

Some of the techniques for organising tasks require having set up your own tasks folders. Information on creating folders can be found in Folders.

The following options are available:

  • Editing tasks

  • Marking tasks as done

  • Changing a task's due date

  • Moving tasks to another folder

  • Exporting tasks

  • Changing tasks confirmations

  • Printing tasks

  • Deleting tasks
