9.1. Los componentes de Ficheros

Se puede encontrar una introducción a la interfaz de usuario en El interfaz de usuario.

Barra de herramientas Contiene los siguientes iconos:

  • Icono Añadir . Contiene las siguientes funciones:

    • Añadir archivo nuevo. Crea un nuevo archivo.

    • Nuevo documento de texto. Crea un nuevo documento de texto.

  • Icono Cambiar carpeta . Abre y cierra el árbol de carpetas.

  • Icono Cambiar búsqueda . Abre un campo para buscar ficheros con un nombre determinado.

  • Icono Vista . Permite seleccionar una de las siguientes vistas para el área de visión:

    • Iconos

    • Lista

Árbol de carpetas Muestra las carpetas. Permite la navegación por la estructura de carpetas. Para abrir el árbol de carpetas, presione el icono Cambiar carpeta en la barra de herramientas. Están disponibles las siguientes funciones:

  • Si ha pulsado en una carpeta, se muestran sus ficheros.

  • El icono Acciones específicas de carpeta que hay junto a la carpeta seleccionada nos ofrece funciones para organizar datos y para intercambiar datos.

Área de visualización en la vista Iconos Se muestra la siguiente información:

  • Ruta de la carpeta seleccionada. Para abrir otra carpeta, presione en la ruta.

  • El botón Subir nuevo fichero. Abre la ventana para subir un nuevo fichero.

  • El botón Compartir esta carpeta. Crea una publicación para la carpeta seleccionada.

  • Si la carpeta contiene imágenes, archivos de audio o de vídeo, se muestran funciones para reproducir archivos multimedia.

  • Un icono por cada archivo de la carpeta seleccionada y, si está disponible, una vista previa del archivo.

Si se presiona en un archivo, sus datos se muestran en la Ventana emergente, que muestra la misma información que el área de visualización en la vista Lista.

Barra lateral en la vista Lista Muestra una lista de los ficheros de la carpeta seleccionada actualmente. Están disponibles las siguientes funciones:

  • Si se pulsa en un archivo, se muestran sus datos en el área de visualización.

  • Si se pulsa el icono Seleccionar que hay debajo de la lista, se muestra una casilla de verificación junto a cada archivo. Se pueden seleccionar múltiples ficheros para editarlos todos al mismo tiempo.

  • El texto que aparece debajo de la lista contiene el nombre de la carpeta seleccionada. El número corresponde a la cantidad de objetos que hay en la carpeta seleccionada.

Área de visualización en la vista Lista Muestra los datos del archivo:

  • Botones: Abrir , Descargar , Borrar .

    Si se pulsa en un archivo editable de la barra lateral, se muestra el botón Editar.

    Si se presiona en Más, se abre un menú que muestra las siguientes funciones: Enviar como enlace , Enviar por correo , Mostrar enlace , Renombrar , Editar descripción , Bloquear , Mover , Copiar y Añadir al portal .

    Nota: Algunos botones no se muestran si no se tienen los permisos adecuados.

    Si se pulsa en un documento de la barra lateral, se muestra el botón Editar como nuevo en el menú, bajo Más.

  • Nombre, nombre del archivo

  • Una vista previa del archivo, si está disponible

  • Comentarios del archivo

  • Ruta de navegación a la carpeta seleccionada. Para abrir otra carpeta, presione en la ruta.

  • Botones para subir una nueva versión.

  • Una lista de versiones si el archivo tiene varias versiones. La versión actual aparece arriba. Para cada versión se muestra la siguiente información:

    • el nombre de archivo de la versión

    • comentarios de la versión, si están disponibles

    • nombre del usuario que subió la versión

    • fecha y hora de la carga de la versión
