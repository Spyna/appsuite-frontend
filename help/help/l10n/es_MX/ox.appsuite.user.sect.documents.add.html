10.2. Creación de documentos

Puede crear documentos que contengan texto formateado, tablas e imágenes. Para crear un nuevo documento, puede hacer lo siguiente:

  • Crear un documento en cualquiera de las aplicaciones Texto o Hoja de cálculo. Puede utilizar una plantilla.

  • Crear un documento en la aplicación Ficheros.

Cómo crear un nuevo documento:

  1. Para crear un nuevo documento de texto sin usar plantillas, presione en Nuevo documento de texto en la barra de herramientas de la aplicación Texto.

    Para crear una nueva hoja de cálculo sin usar plantillas, presione en Nueva hoja de cálculo en la barra de herramientas de la aplicación Hoja de cálculo.

    Para usar una plantilla, presione en ella en Nuevo desde plantilla.

    El documento se muestra en la página sin nombre page.

  2. Presione en Sin nombre en la parte superior derecha de la barra de documento. Introduzca un nombre para el documento.

  3. Crear y editar el contenido de un documento. Para editar un documento, utilice las técnicas habituales de las ya conocidas aplicaciones de Office.

    Tenga en cuenta los detalles de las funciones de Texto y Hoja de cálculo.

  4. Para terminar, presione el icono Cerrar documentode la parte derecha de la barra de documento.

Cómo crear un nuevo documento en la aplicación Ficheros:

  1. Lance la aplicación Ficheros.

  2. Abra una carpeta en el árbol de carpetas.

    Nota: Abra una carpeta en la que tenga los permisos adecuados para crear objetos.

  3. Presione el icono Nuevo de la barra de herramientas. Presione en Nuevo documento de texto o en Nueva hoja de cálculo.

  4. Presione en Sin nombre en la parte superior derecha de la barra de documento. Introduzca un nombre para el documento.

  5. Crear y editar el contenido de un documento. Para editar un documento, utilice las técnicas habituales de las ya conocidas aplicaciones de Office.

    Tenga en cuenta los detalles de las funciones de Texto y Hoja de cálculo.

  6. Para terminar, presione el icono Cerrar documentode la parte derecha de la barra de documento.
