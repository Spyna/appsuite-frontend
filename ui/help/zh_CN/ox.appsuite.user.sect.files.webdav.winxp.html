9.6.2. 在 视窗 XP 下设置 WebDAV

如何在视窗 XP 下设置 WebDAV 访问：

  1. 单击我的网路位置。在我的电脑窗口里单击我的网路位置 或双击视窗桌面里的 我的网路位置 图标。

  2. 前往网络任务，打开网上邻居对话窗口，然后单击添加一个网上邻居 。一个用于添加网络资源的向导启动。单击下一步。

  3. 在下一个对话窗口中选择选择另一个网络位置 。

  4. 输入以下地址： https://<adresse>/servlet/webdav.infostore 其中 <address> 需要用组件服务器的 IP 地址或 URL 取代。

  5. 输入组件服务器的用户名称和密码。

  6. 在下一个对话窗口中，您可以向网络资源指定一个名称。

    备注：如果您已经将您的编码设置为 ISO，请不要在文件夹或文档名称中使用任何特殊字符。组件服务器使用 UTF-8 编码。

  7. 在下一个对话窗口中单击完成。

结果：您可以使用网路位置，以访问 文件 应用程序文件。
