8.6. Taakmappen Toevoegen

U kan extra persoonlijke taakmappen aanmaken onder Mijn taken

De manier waarop een nieuwe persoonlijke taakmap aangemaakt wordt:

  1. In het mappenoverzicht klikt u op Nieuw map toevoegen. Een scherm wordt geopend.

  2. Geef een naam in. Indien de nieuwe map en publieke map moet zijn, schakel dan Toevoegen als publieke map in. Klik op Toevoegen.
