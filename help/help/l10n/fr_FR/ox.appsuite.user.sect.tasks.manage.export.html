8.7.5. Exportation de tâches

Vous pouvez exporter des tâches au format iCalendar. Vous pouvez par exemple utiliser ce format pour échanger vos tâches avec d'autres applications d'agendas.

Pour exporter des tâches :

  1. Choisissez une tâche ou plusieurs tâches dans la liste.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils. Dans le menu, cliquez sur Exporter. La fenêtre Exporter la sélection s'ouvre.

  3. Cliquez sur Exporter. Complétez les champs.
