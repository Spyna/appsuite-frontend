7.2.2. Afficher plusieurs fuseaux horaires

En plus du fuseau horaire par défaut, vous pouvez afficher des fuseaux horaires que vous avez configurés comme favoris. Le fuseau horaire par défaut peut être modifié dans les réglages de base.

Pour afficher plusieurs fuseaux horaires dans une vue en calendrier :

  1. Cliquez sur Vue dans la barre d'outils. Sélectionnez l'une des entrées suivantes : Jour, Semaine ouvrée ou Semaine.

  2. Cliquez sur le bouton Fuseau horaire situé au dessus de la page d'agenda, sur la gauche. Choisissez un fuseau horaire dans le menu sous Favoris.

    Pour ajouter un fuseau horaire à la liste des favoris ou l'en retirer, cliquez sur Gérer les fuseaux horaires. Vous êtes redirigé vers les réglages de l'agenda.
