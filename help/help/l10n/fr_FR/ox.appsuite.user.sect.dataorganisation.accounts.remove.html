11.4.3. Supprimer des comptes

Si vous ne souhaitez plus accéder à un compte donné depuis le collecticiel, vous pouvez supprimer ce compte. Vous pouvez supprimer tous les comptes sauf votre compte de courrier principal.

Pour supprimer un compte :

  1. Cliquez sur l'icône menu Système à droite de la barre de menus. Dans le menu, cliquez sur Réglages.

  2. Dans la barre latérale, cliquez sur Comptes.

  3. Cliquez sur l'icône Supprimer
