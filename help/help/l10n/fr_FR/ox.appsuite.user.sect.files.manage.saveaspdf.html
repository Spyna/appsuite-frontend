9.5.6. Envoyer les fichiers comme PDF

Vous pouvez enregistrer des documents et fichiers texte au format PDF. Le fichier PDF sera enregistré dans le dossier contenant le document ou le fichier texte.

Comment enregistrer un fichier au format PDF :

  1. Sélectionnez un fichier dans la zone d'affichage. Cliquez sur l'icône Actions dans la barre d'outils. Sélectionnez Enregistrer en PDF depuis le menu.

  2. Dans la fenêtre Enregistrer en PDF, vous pouvez saisir le nom du fichier PDF. Cliquez sur Enregistrer.
