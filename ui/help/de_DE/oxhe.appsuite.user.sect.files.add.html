Dateien anlegen

Eine Datei enthält die folgenden Bestandteile:

  • Name

  • eine oder mehrere Versionen

  • Anmerkungen

Um eine neue Datei anzulegen, müssen Sie im Fenster Neue Datei hinzufügen zumindest einen Namen eingeben oder eine lokal gespeicherte Datei hochladen. Alle weiteren Angaben sind optional.

So legen Sie eine neue Datei an:

  1. Öffnen Sie im Ordnerbaum einen Ordner.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Anlegen von Objekten haben.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Neu . Klicken Sie auf neue Datei hochladen.

  3. Um eine lokal gespeicherte Datei hochzuladen, klicken Sie auf Datei auswählen. Wählen Sie eine Datei aus.

  4. Um weitere Informationen einzugeben, klicken Sie auf Weitere anzeigen. Geben Sie die Informationen ein.

  5. Klicken Sie auf Speichern.

Tipp: Sie können eine neue Datei auch anlegen, indem Sie eine Datei vom Desktop Ihres Betriebssystems in das Fenster der App Dateien ziehen und dort in der oberen Hälfte ablegen.
