3.1.7. The editing window

will be used when creating a new item or an existing item.

Content

  • The title bar includes the following elements:

    • window title

    • icons for setting the window position:

      • Minimized icon . Shows the editing window as icon in the task bar.

      • Centered . The editing window will be centered.

      • Aligned icon . Shows the editing window at the bottom right.

  • Depending on the app, app specific buttons or icons might be available.

  • Buttons for closing or canceling the editing window. Depending on the groupware's configuration, those functions are at the top or at the bottom.

An editing window has the following properties:

  • An editing window can not be moved freely.

  • If an editing window is opened, you can use further functions within the groupware.

  • You can open multiple editing windows. Only one editing window can be active though. The other editing windows are shown as icons in the task bar.
