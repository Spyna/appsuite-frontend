10.5. Chat settings

How to use the chat settings:

  1. Click the Settings icon Click on Settings.

  2. Click on Chat in the list.

Depending on the configuration, the following settings are available:

Enabling notification sounds

Defines whether a sound will be played for incoming messages. If notification sounds are enabled, additional settings are available:

  • the sound to be played

  • when the sound should be played

Showing chat desktop notifications

Defines whether you will receive a desktop notification for incoming messages. This setting will only come into effect if you also have enabled desktop notifications in the basic settings, see Receiving Notifications
