11.2. Editing Text Files

How to edit a text file:

  1. Launch the Files app.

  2. Open a folder containing a text file in the folder tree.

    Note: Open a folder for which you have the permissions to edit objects.

  3. Depending on the view selected, use one of the following methods:

    • Click on a text file in the Icons view. Click on Edit in the pop-up.

    • In the List view, select a text file in the sidebar. Click on Edit in the display area.

    The text file will be opened for editing.

  4. In order to edit the text file, use common techniques from familiar applications.

  5. In order to save the text file, click on Save. A new Version will be created.

    In order to finish, click on Close.
