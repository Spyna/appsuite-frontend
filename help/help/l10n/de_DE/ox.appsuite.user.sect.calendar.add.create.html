7.4.1. Neuen Termin anlegen

Um einen neuen Termin anzulegen, haben Sie folgende Möglichkeiten:

  • einen Termin anlegen, indem Sie Betreff, Zeit und Ort angeben

  • weitere Funktionen verwenden: Zeitzone ändern, Serientermin erstellen, andere Teilnehmer oder Ressourcen hinzufügen, Verfügbarkeit festlegen, Anlagen hinzufügen.

So legen Sie einen neuen Termin an:

  1. Öffnen Sie im Ordnerbaum einen Kalenderordner.

    Hinweis: Öffnen Sie einen Kalenderordner, in dem Sie die Berechtigung zum Anlegen von Terminen haben.

  2. Klicken Sie in der Werkzeugleiste auf Neu.

  3. Geben Sie einen Betreff ein. Bei Bedarf geben Sie den Ort und eine Beschreibung ein.

  4. Legen Sie den Beginn und das Ende des Termins fest. Bei ganztägigen Terminen aktivieren Sie Ganztägig.

  5. Wenn Sie eine Terminerinnerung wünschen, wählen Sie in Erinnerung eine Einstellung.

  6. Bei Bedarf können Sie weitere Funktionen verwenden: Zeitzone ändern, Serientermin erstellen, andere Teilnehmer oder Ressourcen hinzufügen, Verfügbarkeit festlegen, Farbe zuweisen, Anlagen hinzufügen.

  7. Klicken Sie auf Anlegen.

Tipp: Sie können alternativ auch eine der folgenden Methoden verwenden:

  • Wählen Sie eine der Kalenderansichten Tag, Arbeitswoche, Woche, Monat. Um einen bestimmten Zeitraum anzuzeigen, klicken Sie oberhalb des Kalenderblatts auf das Datum. Im Kalenderblatt doppelklicken Sie auf eine freie Stelle, oder ziehen Sie einen Bereich vom Beginn zum Ende des neuen Termins.

  • Um einen ganztägigen Termin zu erstellen, können Sie alternativ eine der folgenden Methoden verwenden:

    • Wählen Sie eine der Kalenderansichten Tag, Arbeitswoche, Woche. Doppelklicken Sie oberhalb des Kalenderblatts auf eine freie Stelle.

    • Wählen Sie eine der Kalenderansichten Arbeitswoche, Woche. Klicken Sie oberhalb des Kalenderblatts auf einen Tag.

Ähnliche Aktionen

  • Termin aus iCal-Anlage anlegen, Termine aus Datei importieren, Termine aus anderen Kalendern abonnieren

So verwenden Sie beim Anlegen eines Termins weitere Funktionen:

  1. Wenn Sie die Zeiten des Termins in einer anderen Zeitzone angeben wollen, klicken Sie neben der Uhrzeit für den Beginn oder das Ende auf eine Zeitzonen-Schaltfläche. Das Dialogfenster Zeitzone ändern wird geöffnet. Verwenden Sie die folgenden Methoden:

    • Um die Zeitzone für den Beginn und das Ende des Termins zu bestimmen, wählen Sie in Start date timezone eine Zeitzone.

    • Um für das Ende des Termins eine andere Zeitzone zu bestimmen, wählen Sie in End date timezone eine Zeitzone.

      Hinweis: Die Zeiten für Terminbeginn und Terminende werden in der Zeitzone des Terminbeginns gespeichert, sobald Sie das Dialogfenster beenden.

    • Das Kontrollfeld Convert the entered start and end dates to match the modified timezones bestimmt, ob die eingegebenen Zeiten in die geänderte Zeitzone umgerechnet werden.

  2. Wenn Sie einen Serientermin erstellen möchten, aktivieren Sie Wiederholen. Die aktuellen Wiederholungsparameter werden angezeigt.

    Um einen Wiederholungsparameter einzustellen, klicken Sie ihn an. Beispiele zur Anwendung finden Sie bei den Fragen und Antworten zum Kalender.

    Um die Wiederholungsparameter auszublenden, klicken Sie auf das Symbol Schließen . Um sie wieder einzublenden, klicken Sie auf Bearbeiten.

  3. In Anzeigen als bestimmen Sie die Anzeige der Verfügbarkeit. Beispiele zur Anwendung finden Sie bei den Fragen und Antworten zum Kalender.

  4. Wenn der Betreff des Termins für andere Benutzer nicht sichtbar sein soll, aktivieren Sie Privat. Private Termine werden durch das Symbol Privat gekennzeichnet.

  5. Wenn Sie dem Termin eine eigene Farbe geben möchten, klicken Sie auf ein Farbfeld. Wenn Sie keine Farbe wählen oder wenn Sie das erste Farbfeld anklicken, erhält der Termin die Farbe des Kalenderordners. Die Standardfarbe eines Kalenderordners ist Hellblau.

    Hinweis: Das Farbwahlfeld wird nur angezeigt, wenn Sie in der Werkzeugleiste im Menü Ansicht die Einstellung Benutzerdefinierte Farben wählen.

  6. Wenn Sie andere Teilnehmer oder Ressourcen hinzufügen möchten, geben Sie die E-Mail-Adresse des Teilnehmers, den Namen einer Gruppe oder einer Verteilerliste oder den Namen der Ressource unterhalb von Teilnehmer in das Eingabefeld ein. Klicken Sie auf das Symbol .

    Tipp: Während der Eingabe werden passende Vorschläge angezeigt. Um einen Vorschlag zu übernehmen, verwenden Sie eine der folgenden Methoden:

    • Blättern Sie mit Hilfe des Rollbalkens durch die Liste. Klicken Sie auf einen Vorschlag.

    • Wählen Sie einen Vorschlag mit den Cursortasten. Drücken Sie die Eingabetaste.

    Um Teilnehmer oder Ressourcen zu entfernen, klicken Sie neben dem Namen auf das Symbol .

    Um alle Teilnehmer über den neuen Termin zu informieren, aktivieren Sie Alle Teilnehmer per E-Mail benachrichtigen.

    Tipp: Um verfügbare Zeiten der Teilnehmer und Ressourcen zu finden, können Sie die Planungsansicht verwenden.

  7. Wenn Sie dem Termin eine Anlage hinzufügen wollen, klicken Sie unterhalb von Anlagen auf Anlagen hinzufügen. Wählen Sie eine Datei oder mehrere Dateien.

    Um eine Anlage zu entfernen, klicken Sie auf das Symbol Löschen .

    Tipp: Sie können eine Anlage auch hinzufügen, indem Sie ein Dokument aus einem Datei-Browser oder vom Desktop in das Termin-Fenster ziehen und ablegen.

Verwandte Themen

  • Gruppen verwalten, Ressourcen verwalten, Verteilerliste anlegen
