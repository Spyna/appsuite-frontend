Inviting all recipients of an E-Mail to an appointment

If there are several recipients in an E-Mail, all recipients can be invited to a new appointment.

How to invite all recipients of an E-Mail to an appointment:

  1. Select an E-Mail from the sidebar.

  2. Click on All recipients in the display area.

    Note: This button is only displayed if the E-Mail has multiple recipients.

  3. Select Invite to appointment from the menu.

  4. Complete the data for Creating an appointment.
