10.3. Modifier des documents

Lorsque vous modifiez des documents, vous pouvez :

  • modifier le texte, mettre en forme les caractères, mettre en forme les paragraphes

  • modifier les tableaux, mettre en forme les cellules, modifier les formules

  • télécharger ou imprimer le document sélectionné, ou l'envoyer par courriel

  • rechercher et remplacer du texte

Vous pouvez modifier un document soit dans l'une de ces apps : Texte ou Feuille de calcul, soit dans l'app Fichiers.

Pour modifier un document :

  1. Dans l'app Texte, cliquez sur Ouvrir un document de texte dans la barre d'outils.

    Dans l'app Feuille de calcul, cliquez sur Ouvrir unefeuille de calcul dans la barre d'outils.

  2. Choisissez un document dans la fenêtre Ouvrir un document. Cliquez surOuvrir. Le document sera ouvert en mode modification.

  3. Pour modifier le document, utilisez les techniques usuelles des applications Office familières.

    Pour mettre en forme des caractères ou des paragraphes, utilisez les outils de la barre de documents.

    Remarque : toutes les modifications sont immédiatement enregistrées.

  4. Pour télécharger ou imprimer le document sélectionné, ou pour l'envoyer dans un courriel, utilisez les icônes sur la gauche de la barre d'outils.

  5. Pour chercher et remplacer des éléments de texte, cliquez sur l'icône Recherche oui/non sur la gauche de la barre d'outils. Les éléments nécessaires à la recherche et au remplacement apparaîtront.

  6. Pour arrêter les modifications et fermer le document, cliquez sur l'icône Fermer le document à droite de la barre de document.

Pour modifier un document dans l'app Fichiers :

  1. Lancez l'app Fichiers.

  2. Ouvrez un dossier contenant des documents dans l'arborescence des dossiers.

    Remarque : ouvrez un dossier dans lequel vous avez le droit de créer des objets.

  3. Cliquez sur un document dans la zone d'affichage. Cliquez sur Modifier dans la fenêtre qui apparaît alors. Le document s'ouvrira en mode modification.

    Utilisez les techniques décrites dans les instructions pour modifier un document.
