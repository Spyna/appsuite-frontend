2.2.1. Selecting data for synchronisation

How to select the data to be synchronised on a mobile device:

  1. In the OX Drive app, tap the menu icon.

  2. Select All. Open the folder with the contents you want to synchronise.

  3. Next to the folder name, swipe from the right side to the middle. The folder menu opens.

  4. Tap the Favourites icon. Close the folder menu.
