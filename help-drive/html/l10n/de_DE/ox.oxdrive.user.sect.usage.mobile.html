2.2. Auf einem Mobilgerät

Auf einem Mobilgerät sind diese Funktionen verfügbar:

  • Daten zum Synchronisieren auswählen

  • Die Inhalte des Fotostream Ordner ansehen

  • Weitere Funktionen verwenden, zum Beispiel Ordner anlegen, suchen, sortieren

  • Einstellungen anpassen
