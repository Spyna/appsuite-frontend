Berechtigungen

Berechtigungen legen fest, was ein Benutzer mit einem bestimmten Ordner und mit seinen Inhalten tun darf. Dazu haben Benutzer bestimmte Berechtigungen an den Ordnern. Je nach Ordnerart sind verschiedenen Berechtigungen vordefiniert:

  • Für Ihre persönlichen Ordner besitzen Sie das Administrator-Recht. Dies bedeutet, dass Sie zum Beispiel Unterordner anlegen oder löschen können. Andere Benutzer haben keine Berechtigungen an Ihren persönlichen Ordnern.

  • Im Ordner Öffentliche Ordner haben Sie die Berechtigung, Unterordner anzulegen und diese für andere Benutzer freizugeben.

  • Wenn Sie möchten, dass ein anderer Benutzer Informationen in einem Ihrer privaten Ordner verwenden darf, müssen Sie diesem Benutzer bestimmte Berechtigungen an diesem Ordner erteilen. Dieser Vorgang wird auch als Freigeben des Ordners bezeichnet.

Bevor Sie Freigaben einrichten, beachten Sie folgende Hinweise:

  • Sie können nicht einzelne Objekte freigeben, sondern nur komplette Ordner.

  • Überlegen Sie, welche Berechtigung ein anderer Benutzer benötigt. Wenn ein Benutzer Ordnerinhalte nur lesen soll, benötigt er kein Schreibrecht für diesen Ordner.Sie können nicht einzelne Objekte freigeben, sondern nur komplette Ordner.
