File

9.1. I componenti di File

Un'introduzione all'interfaccia utente è disponibile in L'interfaccia utente.

Albero delle cartelle Visualizza le cartelle. Consente la navigazione della struttura delle cartelle. Per aprire l'albero delle cartelle, fare clic sul pulsante Vista nella barra degli strumenti. Abilitare Vista della cartella. Sono disponibili le seguenti funzioni:

  • Facendo clic su una cartella, i file contenuti sono visualizzati.

  • L'icona Azioni delle cartelle accanto alla cartella selezionata offre le funzioni per organizzare i dati e per scambiare i dati.

Barra degli strumenti Contiene i seguenti pulsanti:

  • il pulsante Nuovo. Contiene le seguenti funzioni:

    • Carica un nuovo file. Creare un nuovo file.

    • Aggiungi nota. Crea un semplice file di testo.

    • Nuovo documento di testo. Crea un nuovo documento di testo.

    • Nuovo foglio elettronico Crea un nuovo foglio elettronico.

  • In caso di apertura di una cartella personale, il pulsante Condividi cartella viene visualizzato. Facendo clic su questo pulsante, sarà creata una pubblicazione di questa cartella.

  • Se la cartella contiene immagini, file audio o video, saranno visualizzate icone per riprodurre i file multimediali.

  • Se un file è stato selezionato, l'icona Scarica sarà visualizzata.

  • Se un file è stato selezionato, l'icona EliminaAltro sarà visualizzata.

  • Se un file è stato selezionato, il pulsante Altro sarà visualizzato. Facendo clic sull'icona, si aprirà un menu con le voci seguenti: Invia tramite posta, Invia come collegamento interno, Mostra collegamento interno, Aggiungi al portale, Sposta, Copia, , Blocca.

    Nota: se sono stati selezionati più file, alcune funzioni non sono disponibili.

  • Il pulsante Vista. Selezionare una delle seguenti viste per la vista dettagliata:

    • Elenco

    • Icone

    • Riquadro

Nota: alcuni pulsanti sono visualizzati solo se la cartella selezionata contiene file.

Area di visualizzazione nella vista Elenco Per ogni file viene visualizzata una riga con il seguente contenuto.

  • Una casella di selezione per marcare uno o più file.

  • Un'icona per ogni file nella cartella selezionata. Un'anteprima dei file è visualizzata, se disponibile.

  • Nome e dimensione del file

  • Data o ora dell'ultima modifica

Facendo clic su un file, si aprirà una finestra a comparsa. Essa mostra ulteriori dati e funzioni.

Area di visualizzazione nella vista Icone Il percorso di navigazione alla cartella selezionata

  • Un'icona per ogni file nella cartella selezionata. Un'anteprima viene visualizzata, se disponibile.

  • Il nome del file

Facendo clic su un file, si aprirà una finestra a comparsa. Essa mostra altri dati e funzioni.

Area di visualizzazione nella vista Riquadro Visualizza un riquadro per ogni file nella cartella selezionata. Se disponibile, un'anteprima dei file è visualizzata. Facendo clic su un file, si aprirà una finestra a comparsa. Tale finestra mostra ulteriori dati e funzioni.

Finestra a comparsa Facendo clic su un file, si aprirà una finestra a comparsa, che visualizza le seguenti informazioni:

  • il pulsante Apri

    Facendo clic su un documento, il pulsante Visualizza viene visualizzato al posto del pulsante Apri.

  • Facendo clic su un file modificabile, il pulsante Modificaviene visualizzato.

  • I pulsanti Scarica, Elimina

  • Facendo clic sull'icona Altro, si apre un menu con le seguenti voci: Condividi questo file, Invia tramite posta, Invia come collegamento interno, Mostra collegamento interno, Aggiungi al portale, Rinomina, Modifica descrizione, Sposta, Copia, Blocca.

    Nota: alcuni pulsanti potrebbero non essere visualizzati se non si dispone dei permessi appropriati.

  • Facendo clic su un documento, il pulsante Modifica come nuovo viene visualizzato nel sottomenu Altro.

  • nome file

  • un'anteprima dei file, se disponibile

  • commenti dei file

  • Il percorso di navigazione alla cartella selezionata. Per aprire un'altra cartella, fare clic sul percorso.

  • I pulsanti per caricare una nuova versione.

  • Il pulsante Mostra cronologia delle versioni. Facendo clic su questo pulsante, un elenco delle versioni viene visualizzato. La versione corrente è elencata in alto. Per ogni versione sono visualizzate le seguenti informazioni:

    • il nome file della versione

    • commenti della versione, se disponibili

    • nome dell'utente che ha caricato la versione

    • data e ora del caricamento della versione
