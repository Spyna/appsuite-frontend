5.6.4. Saving all email recipients as a distribution list

If an email contains multiple recipients, you can save all the recipients as a new distribution list.

How to save the recipients of an email as a distribution list:

  1. Select an email from the side bar.

  2. Click on All recipients in the display area.

    Note: This button is only displayed if the email has multiple recipients.

  3. Select Save as distribution list from the menu.

  4. Complete the data for Creating a distribution list.
