9.5.1. Dateien als E-Mail-Anhang senden

Sie können die aktuellen Versionen von Dateien als E-Mail-Anhang senden.

So senden Sie Dateien als E-Mail-Anhang:

  1. Wählen Sie im Anzeigebereich eine Datei oder mehrere Dateien. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Per E-Mail versenden.

    Alternativ verwenden Sie im Viewer das Symbol Aktionen .

  2. Vervollständigen Sie auf der Seite Verfassen die Angaben zum Senden der E-Mail.
