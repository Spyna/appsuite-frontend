8.8.1. Marquer des tâches comme terminées

Vous pouvez marquer les tâches comme terminées.

Pré-requis : vous devez avoir le droit de créer des objets dans le dossier contenant la tâche.

Pour marquer une tâche comme terminée :

  1. Choisissez une tâche ou plusieurs tâches dans la liste.

  2. Cliquez sur Terminé dans la barre d'outils.

    Pour marquer les tâches sélectionnées comme non terminées, cliquez sur Non terminée.
