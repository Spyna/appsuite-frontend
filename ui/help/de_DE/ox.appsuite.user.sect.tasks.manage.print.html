Aufgaben drucken

Um Aufgaben zu drucken, haben Sie folgende Möglichkeiten:

  • die Daten einer Aufgabe drucken

So drucken Sie die Daten einer Aufgabe:

  1. Wählen Sie in der Seitenleiste eine Aufgabe.

  2. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Drucken.

  3. Falls erforderlich, ändern Sie die Druckereinstellungen. Klicken Sie links unten auf Über das Systemdialogfeld drucken....
