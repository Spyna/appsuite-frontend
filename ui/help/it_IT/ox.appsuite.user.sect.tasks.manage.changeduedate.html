Cambiare la data di scadenza di un'attività

È possibile cambiare la data e l'ora di scadenza di un'attività.

Come cambiare la data di scadenza di un'attività:

Prerequisito: è necessario disporre dei permessi per creare oggetti nella cartella che contiene l'attività.

  1. Selezionare un'attività nella barra laterale.

  2. Fare clic su Cambia la data di scadenza nell'area di visualizzazione. Selezionare una voce.
