2.2.1. Selecting data for synchronisation

How to select the data to be synchronised on a mobile device:

  1. In the OX Drive app, tap the menu icon.

  2. Select Files.

  3. Select All. Make sure the folder you want to synchronise is visible.

  4. Next to the folder name, swipe from the right side to the middle. The folder menu opens.

  5. Tap the Favourites icon. Close the folder menu.
