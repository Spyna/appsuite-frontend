12.2.3. Créer des dossiers

Vous pouvez créer un nombre arbitraire de sous-dossiers dans un dossier privé. Pour créer des sous-dossiers dans un dossier partagé ou public, vous devez avoir les droits requis.

Pour créer un nouveau dossier :

  1. Dans l'arborescence des dossiers, sélectionnez le dossier dans lequel vous voulez créer un nouveau sous-dossier. Si vous ne choisissez pas de dossier, le nouveau sous-dossier sera créé dans la racine de votre dossier privé.

    Remarque : choisissez un dossier dans lequel vous avez les droits permettant de créer des sous-dossiers.

  2. Cliquez sur l'icône Ajouter un dossier sous l'arborescence des dossiers. Choisissez Ajouter un sous-dossier.

  3. Saisissez un nom dans la fenêtre Ajouter un nouveau sous-dossier. Cliquez sur Ajouter un dossier.
