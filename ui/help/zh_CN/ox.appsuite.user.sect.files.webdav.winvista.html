9.6.3. Windows Vista 下设置 WebDAV

为了设置在 Windows Vista 下设置 WebDAV 访问，请使用以下选项之一：

  • 使用 OX 更新器。在 OX 更新器文档里可以找到信息。

  • 第一，为网页文件夹安装微软软体更新。然后，设置WebDav 访问。

如何为 WebDAV 安装 Microsoft 软件更新：

  1. 打开 Web 浏览器。

  2. 输入以下地址：http://support.microsoft.com/kb/907306

  3. 浏览到更多信息，然后单击为网页邮件文件夹包下载软体更新。

  4. 在下载页面里单击下载。将下载文件 Webfldrs-KB907306-ENU.exe。

  5. 双击该文件。安装将启动。

如何在 Windows Vista 下设置 WebDAV 访问：

  1. 在 Windows 资源管理器中打开计算机。

  2. 在资源管理器的右窗格中右击一个空白区域。从下拉菜单中选择添加一个网络位置 。添加网路位置对话窗口打开。单击下一步。

  3. 选择选择自定义网络位置。单击下一步。

  4. 在添加网路路径对话窗口里输入以下地址： https://<address>/servlet/webdav.infostore，需要以组件服务器的 IP 地址或 URL 取代<address>。

  5. 输入您的用户名和密码。使用您的组件服务器登录数据。

  6. 在下一个对话窗口中，您可以向资源指定一个名称。

  7. 在下一个对话窗口中单击完成 。

结果：在计算机下面，您可以访问您的 文件 应用程序文件。
