9.5.10. Lavorare con le versioni

Esistono le seguenti opzioni per lavorare con le versioni:

  • aprire o salvare una versione

  • caricare una nuova versione

  • impostare una versione particolare come versione corrente

  • eliminare una versione

Come aprire o salvare la versione corrente:

  • In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, per aprire la versione, fare clic su Apri. Per salvare la versione, fare clic su Scarica.

    • Nella vista Elenco, selezionare un file nella barra laterale. Per aprire la versione, fare clic su Apri nell'area di visualizzazione. Per salvare la versione, fare clic su Scarica nell'area di visualizzazione.

Suggerimento: se viene visualizzata l'anteprima di un documento, è possibile anche salvare la versione corrente trascinando l'anteprima sul desktop e rilasciandola in un punto qualsiasi.

Come caricare una nuova versione:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Seleziona file sotto Carica una nuova versione. Selezionare un file.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic su Seleziona file sotto Carica una nuova versione nell'area di visualizzazione. Selezionare un file.

  2. Digitare un commento della versione.

  3. Fare clic su Carica.

Suggerimento: è possibile caricare una nuova versione anche trascinando un file dal proprio desktop all'area di visualizzazione dei file e rilasciandolo nella parte inferiore.

Come caricare una nuova versione:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic sul nome di una versione sotto Versioni. Si aprirà un menu.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic sul nome di una versione nell'area di visualizzazione sotto Versioni. Si aprirà un menu.

  2. Digitare un commento della versione.

Come impostare una determinata versione di un file come la versione corrente:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Fare clic sul nome di una versione nella finestra a comparsa sotto Versioni. Si aprirà un menu.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic sul nome di una versione nell'area di visualizzazione sotto Versioni. Si aprirà un menu.

  2. Fai clic sulla voce di menu Scegli questa come versione corrente.

Come aprire o salvare una determinata versione:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Fare clic sul nome di una versione nella finestra a comparsa sotto Versioni. Si aprirà un menu.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic sul nome di una versione nell'area di visualizzazione sotto Versioni. Si aprirà un menu.

  2. Per aprire la versione, fare clic sulla voce di menu Apri. Per salvare la versione, fare clic sulla voce di menu Scarica.
