8.3.2. Créer des rendez-vous à partir de pièces jointes au format iCal

Vous pouvez créer un rendez-vous à partir d'une pièce jointe au format iCal. Une telle pièce jointe se reconnaît à son extension « .ics ».

Pour créer un rendez-vous à partir d'une pièce jointe iCal :

  1. Choisissez un courriel avec une pièce jointe iCal dans l'application Courriel.

  2. Cliquez sur le nom de la pièce jointe dans la zone d'affichage. Choisissez Ajouter au calendrier dans le menu.
