Datenorganisation

11.3. Abonnements

Mit Hilfe von Abonnements können Sie Daten aus sozialen Netzwerken innerhalb der Groupware verwenden. Sie haben folgende Möglichkeiten:

  • Importieren Sie Kontakte aus sozialen Netzwerken, indem Sie diese Daten abonnieren.

  • Behalten Sie den Überblick über Ihre abonnierten Daten, indem Sie diese auf einer gemeinsamen Übersichtsseite verwalten.
