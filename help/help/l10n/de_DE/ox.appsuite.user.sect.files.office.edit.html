9.5.2. Office-Dokumente bearbeiten

Wenn Sie Office-Dokumente bearbeiten, haben Sie folgende Möglichkeiten:

  • Text editieren, Zeichen formatieren, Absätze formatieren

  • Das geöffnete Office-Dokument herunterladen oder drucken

So bearbeiten Sie ein Office-Dokument:

  1. Öffnen Sie im Ordnerbaum einen Ordner.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Bearbeiten von Objekten haben.

  2. Je nach eingestellter Ansicht verwenden Sie eine der folgenden Methoden:

    • In der Ansicht Symbole klicken Sie auf ein Dokument. Klicken Sie im Popup auf Bearbeiten.

    • In der Ansicht Liste wählen Sie in der Seitenleiste ein Dokument. Klicken Sie im Anzeigebereich auf Bearbeiten.

    Das Office-Dokument wird zur Bearbeitung geöffnet. Die Seitenleiste zeigt Werkzeuge für die Bearbeitung von Texten und Tabellen.

  3. Um den Text zu editieren, verwenden Sie die gängigen Techniken aus bekannten Office-Anwendungen.

    Um Zeichen oder Absätze zu formatieren, nutzen Sie die Textwerkzeuge in der Seitenleiste.

    Hinweis: Alle Änderungen werden sofort im Office-Dokument gespeichert.

  4. Um das geöffnete Office-Dokument herunterzuladen oder zu drucken, klicken Sie rechts in der Office-Werkzeugleiste auf das Symbol Herunterladen oder Drucken .

  5. Um das Bearbeiten zu beenden, klicken Sie rechts in der Office-Werkzeugleiste auf das Symbol Schließen .

Hinweis: Wenn Sie ein Office-Dokument bearbeiten, wird jedes Mal eine neue Version des Office-Dokuments angelegt.

Tipp: Wie Sie Office-Dokumenten verwalten, erfahren Sie in Abschnitt 9.6, „Dateien organisieren“.
