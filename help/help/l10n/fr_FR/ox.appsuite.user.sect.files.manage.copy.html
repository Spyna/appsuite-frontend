9.4.7. Copier des fichiers

Vous pouvez copier des fichiers vers un autre dossier :

Pour copier des fichiers vers un autre dossier :

Pré-requis : vous devez avoir les droits permettant de créer des objets dans le dossier cible.

  1. Sélectionnez un ou plusieurs fichiers dans la zone d'affichage. Cliquez sur l'icône Actions dans la barre d'outils. Sélectionnez Copier depuis le menu.

  2. Choisissez un fichier dans la fenêtre Copier. Cliquez sur Copier.
