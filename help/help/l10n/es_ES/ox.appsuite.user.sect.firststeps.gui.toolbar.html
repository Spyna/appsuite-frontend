3.1.3. La barra de herramientas

La barra de herramientas se ubica debajo de la barra de menús.

Contenido

  • Botones para crear objetos nuevos, por ejemplo un nuevo correo electrónico o un contacto nuevo.

  • Botones o iconos para editar objetos.

  • Dependiendo de la aplicación, podrían estar disponibles otros botones específicos de la aplicación o iconos.

  • Icono Acciones . Contiene funciones específicas de la aplicación para organizar objetos.

  • Botón Vista. Contiene funciones para controlar la disposición del área de visualización y para abrir o cerrar el árbol de carpetas.
