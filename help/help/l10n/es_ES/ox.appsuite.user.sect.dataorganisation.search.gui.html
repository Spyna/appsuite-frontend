12.1.1. Los componentes de la ventana de búsqueda

La ventana de búsqueda incluye los siguientes componentes:

  • Una barra de herramientas con botones de aplicaciones. Al pulsar en un botón de aplicación se elige la aplicación que se debe usar para la búsqueda.

  • Campo Buscar. A medida que va introduciendo término de búsqueda, las entradas del menú de búsqueda se ajustan de acuerdo a él.

  • El botón Carpeta. Muestra la carpeta que se usó para la búsqueda. Para buscar en otra carpeta, pulse el botón.

  • Un menú de búsqueda con elementos específicos de aplicaciones. Si pulsa en un elemento, se inicia la búsqueda.

  • Icono Buscar . Comienza una búsqueda.

    Tan pronto como se completa la búsqueda, la ventana de búsqueda muestra elementos adicionales:

    • Uno o varios botones con los criterios de búsqueda a utilizar. Algunos botones contienen un desplegable donde se puede cambiar el criterio de búsqueda.

      Para reiniciar el resultado de una búsqueda, cierre el botón de un criterio de búsqueda.

    • La lista de objetos encontrados. Pulse en un objeto para mostrar sus datos en la ventana emergente,

      Puede buscar dentro de la lista comenzando una nueva búsqueda.
