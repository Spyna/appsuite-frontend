2.2.3. Using additional functions

On mobile devices, the OX Drive client contains the following funtions.

  • Selecting one or multiple folders. Moving or deleting the selected folders.

  • Creating a new folder.

  • Searching for files or folders.

  • Sorting files or folders according to certain criteria.

  • Filtering files in a folder. Filter criteria are: all files, synchronised files, unsynchronised files

How to use the OX Drive functions on mobile devices:

  1. In the OX Drive app, tap the menu icon.

  2. Select Files.

  3. Select a function from the function bar.

    Depending on the mobile device, the function bar can be in different locations: on Android mobile devices on the upper border, on iOS devices on the bottom screen border.

    Depending on the mobile device, the functions in the function bar are arranged differently.
