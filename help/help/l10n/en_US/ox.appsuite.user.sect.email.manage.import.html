5.6.11. Importing E-Mails

You can import an E-Mail that is available in the EML format.

How to import an E-Mail:

  1. Open the E-Mail folder to which you want to import the E-Mail.

  2. Drag the EML file from a file browser or from the desktop to the E-Mail app window. Drop the EML file in the list in the display area.
