5.6.11. Salvare i messaggi di posta

È possibile salvare un messaggio di posta come un file di testo. Il file di testo ha estensione EML. Se si selezionano più messaggi, un file eml sarà creato per ogni messaggio. Il file eml sarà poi salvato come archivio zip.

Come salvare un messaggio di posta:

  1. Selezionare uno o più messaggi di posta elettronica.

  2. Fare clic sull'icona Altre azioni nella barra degli strumenti. Selezionare Salva come file dal menu.

    È possibile utilizzare anche l'icona Altre azioninella vista dettagliata.

  3. Compilare i dettagli per salvare il file.
