The Calendar Components

7.1.1. The New appointment button

Creates a new appointment. The More actions icon opens a menu with additional functions: view scheduling view, new email, new contact, new task

The More actions icon will not be available if the folder view is closed. To show the folder view, click the Open folder view icon on the bottom left side.
