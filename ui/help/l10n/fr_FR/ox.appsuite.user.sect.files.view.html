9.2. Afficher des fichiers

Par défaut, le contenu de votre dossier de fichiers privé est affiché. Le dossier est nommé Mes fichiers.

Pour afficher un fichier :

  1. Cliquez sur l'icône Vue de la barre d'outils. Choisissez dans le menu.

    Vous pouvez également utiliser les icônes de la barre de fonctions pour choisir une vue.

  2. Ouvrez un dossier dans l'arborescence des dossiers.

  3. Pour trouver rapidement un fichier donné, utilisez la fonctionnalité de recherche.

  4. Cliquez sur un fichier dans la zone d'affichage. Les données s'affichent dans une fenêtre qui apparaît.
