9.5. Dateien organisieren

Einige der Techniken zum Verwalten von Dateien setzen voraus, dass Sie eigene Ordner eingerichtet haben. Informationen zum Anlegen von Ordnern finden Sie in Dateien oder Ordner anlegen.

Sie haben folgende Möglichkeiten:

  • Dateien als E-Mail-Anhang senden

  • Dateinamen bearbeiten

  • Beschreibung erstellen oder bearbeiten

  • Dateien oder Ordner verschieben

  • Dateien kopieren

  • Datei als PDF speichern

  • Datei zum Portal hinzufügen

  • Dateien löschen

  • Dateien sperren oder entsperren

  • Mit Versionen arbeiten
