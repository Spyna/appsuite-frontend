Attività

8.7. Gestire le attività

Alcune delle tecniche per organizzare le attività richiedono la configurazione delle proprie cartelle di attività. Informazioni sulla creazione delle cartelle sono disponibili in Aggiungere cartelle di attività.

Esistono le seguenti opzioni:

  • Modificare le attività

  • Marcare le attività come completate

  • Cambiare la data di scadenza di un'attività

  • Spostare le attività in un'altra cartella

  • Cambiare le conferme delle attività

  • Importare le attività

  • Esportare le attività

  • Stampare le attività

  • Eliminare le attività
