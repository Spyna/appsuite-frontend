Address Book

6.2. Displaying Contacts

Your personal address book can be found in the folder tree below Contacts. In the address book settings, you can define the address book that is to be opened as default.

How to display a contact:

  1. Open an address book in the folder tree.

  2. In order to display contacts with a certain initial letter, click a letter in the navigation bar.

  3. Click on a contact in the list. The contact's data is displayed in the detail view.

    You can open the contact in a separate window by double-clicking on the contact in the list.

  4. In order to display another contact, do one of the following:

    • Click on another contact in the list.

    • Use the cursor keys to browse the list.
