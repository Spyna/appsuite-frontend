9.5.5. Beschreibung bearbeiten

Sie können die Beschreibung einer Datei bearbeiten.

So bearbeiten Sie die Beschreibung einer Datei:

Voraussetzung: Sie haben für den Ordner, der die Datei enthält, die Berechtigung zum Bearbeiten von Objekten.

  1. Klicken Sie in der Detailansicht auf eine Datei. Klicken Sie im Popup auf Weitere. Klicken Sie im Menü auf Beschreibung bearbeiten.

  2. Bearbeiten Sie die Beschreibung.

  3. Klicken Sie auf Speichern.
